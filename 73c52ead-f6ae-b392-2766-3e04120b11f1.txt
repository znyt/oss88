　　苏莫离开了塔楼，走出了千渊殿。
<br/>　　他也不知道该往哪里走，绫裳在何地，他也不清楚，只能日后慢慢的打探了。
<br/>　　“梁师兄，难道我们就这么算了吗？”
<br/>　　千渊殿府邸之中，梁师兄和几名弟子，伫立在一处树荫下，望着苏莫走出千渊殿，其中一名弟子向梁师兄问道。
<br/>　　“是啊，连师兄被上官昊击成重伤，没有几个月很难恢复了！”
<br/>　　“上官昊欺人太甚了！”
<br/>　　其他弟子纷纷开口，个个面色难看。
<br/>　　“此事以后再说吧！”梁师兄叹了口气，上官昊去了殿主的住处，不知道又得到了什么好处。
<br/>　　此人得殿主看重，他若是再针对，怕是会对自己不利。
<br/>　　“这……！”
<br/>　　“就这样放过他吗？”
<br/>　　众人虽然也知道殿主器重上官昊，但是，还有有些不忿。
<br/>　　“不要再说了，此事就此作罢！”
<br/>　　梁师兄叹了口气，随即又道：”过段时间我要外出历练，你们不要招惹此人。”
<br/>　　众人闻言，无奈的点了点头，既然梁师兄如此，他们也不好再说什么。
<br/>　　……
<br/>　　苏莫离开千渊殿之后，很快便离开了千渊城。
<br/>　　他身形一展，腾空而起，向远处飞去。
<br/>　　“这种飞行速度，实在太慢了！”
<br/>　　半空之中，苏莫有些郁闷，神域的空间太过凝实，飞行的速度非常慢。
<br/>　　而且，大地之上，有一种奇异的牵引力量，很难飞的太高。
<br/>　　他能肯定，若是他只有虚神境二变的修为，在神域根本无法飞行。
<br/>　　好在，神域的地域不是很大，这对于苏莫来说，也无伤大雅。
<br/>　　接下来的一个月之中，苏莫身形辗转了数个城池，暗自打听消息。
<br/>　　他所要打听的消息，自然是绫裳的消息。
<br/>　　此事倒是不难，苏莫打听了几个城池，甚至拿出源石询问见多识广之人，便对于绫裳的底细，便略有了解了。
<br/>　　“居然是神王！”
<br/>　　走在一座城池的街道之上，苏莫低声自喃，根据他打听到的消息，绫裳居然是一名神王。
<br/>　　换言之，绫裳是比千渊殿主还要强大很多倍的强者。
<br/>　　绫裳神王，隶属于天池神朝终极剑域‘终极神王殿’，乃是一名中级神王，当初也算是在天池神朝大有名气之人。
<br/>　　据说，绫裳神王和终极神王殿的紫霄神王，当年在终极剑域，号称为绝代双骄，名动终极剑域数十万年。
<br/>　　只不过，绫裳神王后来遭遇意外，香消玉损。
<br/>　　“终极剑域！”苏莫低声自喃，根据他的了解，终极剑域距离大夏界域比较远，中间相隔了十几个界域。
<br/>　　不过，即便再远，苏莫也要去。
<br/>　　因为，他要救活芊寻月，当初绫裳说过，神域有让芊寻月死而复生之法。
<br/>　　“绫裳应该会回到终极剑域吧！”苏莫心中暗道，对方居然回到了神域，肯定会回终极神王殿。
<br/>　　苏莫估计，绫裳如今的修为，肯定早已再次成神。
<br/>　　毕竟，作为神王转世，重修的速度，不是任何天才所能比拟的
<br/>　　深深的吸了口气，苏莫立刻出城，向终极剑域而去。
<br/>　　大夏界域不小，苏莫飞行的速度极慢，足足飞了几个月，才飞出了大夏界域。
<br/>　　……
<br/>　　在苏莫离开大夏界域，不停的向终极剑域赶路之时，千渊殿迎来了不速之客。
<br/>　　这一日，一股庞大的气势，浩荡无疆，滚滚而来，瞬间降临在千渊殿上空，化为了一名白衣青年。
<br/>　　青年脸庞白皙，五官英挺，俊朗非凡，但是，其一双眼眸，却是蕴含一丝淡淡的邪异。
<br/>　　白衣青年的威压无比庞大，如同苍天大岳，威震环宇。
<br/>　　唰唰唰！！
<br/>　　霎时之间，所有千渊殿之人，都被惊动了，纷纷掠出了住处。
<br/>　　“白衣神王天尘宇！”
<br/>　　“他怎么来了？”
<br/>　　“好恐怖的气息！”
<br/>　　一众千渊殿之人，见到白衣青年，均是心中大震，目瞪口呆。
<br/>　　因为，这名白衣青年，乃是名震神域的白衣神王天尘宇。
<br/>　　神域有十大无上神王，这十大无上神王倒不是神王之中最强的十人，而是天赋最强的十人。
<br/>　　十大无上神王，均是百岁之内成神，千岁之内成就神王，威震整个神域，天资旷古烁今。
<br/>　　众人心惊不已，白衣神王天尘宇怎么来到了这里，难道是来找殿主的？
<br/>　　嗖！
<br/>　　塔楼之中，光芒一闪，殿主夏千渊飞了出来。
<br/>　　“白衣神王驾临我千渊殿，在下有失远迎，还望勿怪！”夏千渊向天穹之上的白衣青年微微抱拳，朗声说道。
<br/>　　同时，他也心中疑惑，因为他和天尘宇并不相识。
<br/>　　对方乃是轮回神朝天王宫的少宫主，身份尊贵，怎么会来千渊城这个小地方？
<br/>　　“回答我，此地接引殿，最近几年之内，可有飞升之人？”天尘宇的目光，落在了夏千渊的身上，冷漠而淡然。
<br/>　　“飞升之人？”夏千渊闻言一怔，完全没想到对方来此，居然是为了这种事。
<br/>　　“有是吗？在哪里？”千尘宇见到夏千渊的神情，顿时眸光精光四射，大声质问。
<br/>　　这两年来，他在天池神朝，找了五十多个界域，终于找到了！
<br/>　　“的确有，不过已经走了！”夏千渊没有隐瞒，毕竟，上官昊是下界飞升之人，此事知晓的人太多了。
<br/>　　“唰！”
<br/>　　白光一闪，天尘宇瞬间便来到了夏千渊的身前，他目光冷冽而霸道，直视夏千渊，连续问道：“此人又何特征？何时离去？去往何处？”
<br/>　　夏千渊眉头微皱，对方的语气，未免有些太过咄咄逼人了。
<br/>　　不过，他也并未隐瞒，道：“此人名为上官昊，一个月前离开了千渊城，至于去了哪里？无从得知。”
<br/>　　夏千渊能猜到，此人对于苏莫来者不善，所以，就算他知晓苏莫要去哪里，也不会告诉天尘宇。
<br/>　　“无从得知？”天尘宇闻言，略带邪异的眼眸，微微一眯。
<br/>　　他的目光，如同犀利的刀锋，如同绝世的神剑，狠狠的刺进了夏千渊的双眼之中。
<br/>　　噗！
<br/>　　夏千渊身形一震，立刻口中喷出一口鲜血，脸色略显苍白。
<br/>　　所有千渊殿之人，无不大惊失色，这天尘宇居然如此霸道。
<br/>　　“你在隐瞒？”
<br/>　　天尘宇面无表情，冷漠的道：“你应该知晓隐瞒我的后果，我若愤，此城湮灭，我若怒，大夏血流成河！”
<br/>　　众人闻言心中一颤，白衣神王天尘宇的威名，他们都知晓，对方的确是有这个实力。
<br/>　　“我并未隐瞒什么，阁下未免太过骄横了！”夏千渊阴沉着脸道，作为九品真神，他岂会轻易被恐吓！
<br/>　　轮回神朝天王宫虽强，但他们大夏神王殿也不弱，他还就不信了，对方敢无缘无故屠城。
<br/>　　“我问，你答，但有隐瞒，死！”
<br/>　　天尘宇再次开口了，声音依旧冷漠，不含丝毫的感情。
<br/>　　“你问吧？”夏千渊道，对方根本不可能从他这里得知什么。
<br/>　　毕竟，上官昊去了哪里？他自己都不知道。
<br/>　　“此飞升之人，是不是修炼混沌之力？”天尘宇问道。
<br/>　　“是！”夏千渊点头。
<br/>　　“此人离开之前，是否说过要去何地？”天尘宇再次问道。
<br/>　　“没有！”夏千渊摇头。
<br/>　　“此人离去的方向，是不是混沌神朝的方向？”天尘宇再次问道。
<br/>　　“不清楚！”夏千渊再次摇头。
<br/>　　问了三个问题之后，天尘宇沉默了下来，他知晓夏千渊不会骗他。
<br/>　　这倒不是他相信夏千渊，而是，那个苏莫，既然离开了千渊城，肯定不会告诉任何人自己的去处。
<br/>　　“帝一魂啊帝一魂，你又遇到了难缠的对手了！”
<br/>　　良久之后，天尘宇轻叹一声，身形拔天而起，急速离去。
<br/>　　接下来，他要在大夏界域搜寻一番。
