　　在浓眉和尚身后不远处，一道血气，就像是一条长长的飘带，在空中穿行，紧紧跟着他。
<br/>　　“现在救他，还来得及。”道风淡淡说道。
<br/>　　叶少阳叹了口气道：“我们不能改变历史。这什么冥河老祖，还是留给将来的人对付他吧。”
<br/>　　两人默默望着那浓眉和尚走下山沟，和追赶他的血气一起消失在视线之外，一起回到了大雄宝殿，来到药师佛的雕像后面，上下看去，被浓眉和尚修改过的那部分，与周围浑然一体，完全看不出一点拆解过的痕迹，光是看到这尊佛像，谁也不会想到在里面藏着一把剑。
<br/>　　叶少阳伸手轻轻摸了下之前浓眉和尚往上刷漆的地方，经过火烤已经干了，颜色上也与周围无异，不禁佩服浓眉和尚的手艺。
<br/>　　“他为什么一定把剑藏在这里？”叶少阳纳闷。
<br/>　　“你刚才没注意吗，他在裹上最后一层布之前，在里面刷了很多符文，想来是某种封印，能够压制住宝剑的气息，让人发现不了。”
<br/>　　“我是说……为什么要藏在佛寺里，哪怕埋在地下也必这地方好吧？”
<br/>　　“我猜，可能他这一路被追杀，并没有太多时间选择合适的地方，剑埋在地下，也有会剑气泄露的，大概是这佛像上更容易刻写符文来压制剑气。”
<br/>　　叶少阳心想大概也只有这个原因了。
<br/>　　其实就藏东西来说，虽然这和尚手段高超，藏得天衣无缝，但也不能保证佛像不会被别的原因毁掉，到那时候，太阿剑也就藏不住了。
<br/>　　不过，他反过来想，如道风所说，这浓眉和尚大概也是慌不择路，没有更好的选择了，再说只要能躲过冥河老祖，将来太阿剑会在什么时间被什么人得到，一切全看缘分了吧。本来佛家也是讲缘分的。
<br/>　　“咱走吧。回我们的世界去找。”
<br/>　　道风看了他一眼，“我得提醒你，这才是元朝，还有快一千年的时间，这佛像会不会保存一千年，实在是个问题，有很大的可能，这太阿剑已经被人取走了。”
<br/>　　他一只手摸着佛像后面藏剑的位置，“你确定，不要从这里拿走它？”
<br/>　　太阿剑就在近在咫尺的地方……
<br/>　　要说不心动，那是假话。
<br/>　　叶少阳沉默半晌，道：“还是走吧。”
<br/>　　如果从这里拿走太阿剑，这个世界就没它了，叶少阳也推测不出会产生什么后果，但他不敢冒险，他记得徐福的警告：就算在历史中改变一个小点，都有可能引起蝴蝶效应，造成无法挽回的结果。更不要说是太阿剑这么重要的法器。
<br/>　　再说，如果自己把它拿走，那自己的世界就有了两把太阿剑，会引起什么变故，叶少阳不清楚，但他不想冒险。
<br/>　　听他这么说，道风也没再说什么。叶少阳用山海印打开时间虚空，两人钻了进去，回到自己的世界，已经几个小时过去了，天已经亮了。
<br/>　　道风表示自己先去天台山寻找太阿剑，让叶少阳处理完事情立刻赶过去。然后叫来杨宫梓等人，让他们先去鬼域等待。
<br/>　　岳恒已经回来了，现在的他脱胎换骨，彻底觉醒了，整个人的气质也不一样了，如果不是有太多正事要干，叶少阳真想跟他比试一下。
<br/>　　道风带着南宫影一起走了，打算路上向他询问有关冥河老祖的事，毕竟他是阿修罗王子，跟冥河老祖一向是邻居，对他肯定有所了解。
<br/>　　送走他们两个，杨宫梓一干人也要走了，叶少阳跟他们道别，突然想到现在阴阳两界的通道都被封锁了，去阴间只能到南海岛屿上，问杨宫梓他们是不是要从海眼突围出去。
<br/>　　杨宫梓笑着表示他们自有办法能回到风之谷在鬼域里的秘密据点，但现在不好明说。
<br/>　　大家挨个道别，然后杨宫梓带他们穿越虚空离去。
<br/>　　叶少阳一转身看到站在门外的小马，诧异道：“你咋没走？”
<br/>　　“我不走了，去那边暂时也没啥事，我还是跟着你吧。”小马坐过来嘿嘿笑着。
<br/>　　叶少阳想拍他肩膀，想起他是鬼，拍不着。
<br/>　　这时候外面有人敲门，叶少阳下楼开门，见是龙虎山的弟子，送了早饭过来，说是请叶少阳先吃饭，然后请他去大殿，曲波找他有要事商量。
<br/>　　“小伙，你吃了没有？”小马突然问他。
<br/>　　“我……一直在忙，还没顾上。”小道士不知道他为什么这么问，如实回答。
<br/>　　“那正好！”
<br/>　　小马往前一步，附在他的身上，拍了拍肚子，龇牙说道：“我一看到这豆浆油条就流口水了，正好你没吃饭嘛兄弟，我帮你吃饱！”
<br/>　　转身望着一旁发呆的另一个送饭的小道士，让他再送两份早餐过来，小道士赶紧去了。
<br/>　　叶少阳无奈摇头。
<br/>　　最后小马一个人吃了三份早饭，要不是叶少阳阻拦，他还想继续吃。
<br/>　　喝完最后半碗米粥，小马舒坦地揉着肚子，打了几个饱嗝，彻底享受够了，才离开了小道士的身体。
<br/>　　小道士好歹也是入门级法师，知道发生了什么，一脸欲哭无泪的表情，双手捧着肚子。
<br/>　　“你吃饱了吗？”小马问。
<br/>　　小道士连连点头，“管够一天不用吃东西了。”说话都打着饱嗝，挺直身子，一副撑到了的样子。叶少阳换位体会了小道士的心情，估计有一万头草泥马奔过……
<br/>　　来到大殿，叶少阳发现所有所有龙虎山道士都在，大殿门外摆着几副连夜打造的棺材，里面停放着龙阳真人等几个昨晚死在战斗中的同门。
<br/>　　看到这几口棺材，叶少阳心情一下子就沉重起来。
<br/>　　曲波上前来，请求叶少阳帮忙主持超度几位的道场。
<br/>　　按照这种道场，应该由本门掌门主持，但慕寒跑了，活着的几位长老，都遭受打击，心绪悲伤，无法主持，叶少阳也就不再推辞，主持了典礼，然后亲自扶着龙阳真人的棺材，一路来到后山坟地，将几人安葬。
<br/>　　由于是为了守护山门而死，纵然生前身份低微、声名不显，但他们的名字，将会写在龙虎山的《历代英杰录》中，万古长存。
