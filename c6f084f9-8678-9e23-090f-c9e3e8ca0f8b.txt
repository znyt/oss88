　　毒性已除，但依依失血过多，还是晕厥了过去。
<br/>　　萧逸为她做了简单的包扎，止住流血的伤口，待会儿只需上些金创药，依依便可保无虞。
<br/>　　这时，萧逸看向了地面杨伏的尸体。
<br/>　　杨伏毕竟是紫云城有名的杀手，要是被人发现了，萧逸也说不清楚。一个凡境二重的武者将一个凡境九重的武者击杀掉，这是件难以解释的事。
<br/>　　“腾”的一声，萧逸掌中冒出一团炽热的火焰。
<br/>　　进阶为橙阶武魂后，控火兽的火焰威力强了十倍不止。
<br/>　　“橙阶武魂，威力果然比赤阶武魂强多了。”萧逸微微吃了一惊。
<br/>　　同时他心中也有了决定，自己身怀紫色武魂冰鸾剑的事绝对不能外传，否则，定将惹来杀身之祸。
<br/>　　紫色武魂有多珍贵自不必多说。
<br/>　　最重要的是，武魂，武者觉醒后便会跟随武者一生，不会再有变化。
<br/>　　赤阶武魂，便一辈子都是赤阶武魂。
<br/>　　橙阶武魂，便一辈子都是橙阶武魂。
<br/>　　这也是为什么炎龙大陆有着武魂决定武者未来的说法。
<br/>　　可现在，冰鸾剑竟然赋予了自己吸收别人武魂力量的能力，并让自己的武魂进阶。这是整个炎龙大陆绝无仅有的事。
<br/>　　一旦传出去，可想而知会造成多大的波澜。
<br/>　　而且，自己现在的实力太弱了。该保留的秘密和底牌，还是得严实地捂着。
<br/>　　起码，在有足够的自保之力前，冰鸾剑的存在都不能让别人知道。
<br/>　　“去。”萧逸低喝一声，将掌中火苗扔向杨伏的尸体。
<br/>　　霎那间，密集而高温的火焰包裹了杨伏的尸体。
<br/>　　十数分钟后，杨伏的尸体被燃烧殆尽。
<br/>　　单论火焰的威力，控火兽由于精通控火，倒是比之其他橙阶武魂要强上一筹。起码比火烧藤和火狼这两种武魂的火焰威力要强。
<br/>　　毕竟，火烧藤除了赋予控火能力外，还有缠绕的能力。而火狼则多赋予了速度和力量的增幅。
<br/>　　所以，单论火焰威力，倒是控火兽武魂强一些。
<br/>　　这时，杨伏被烧尽的尸体上，多了一个小布袋。
<br/>　　这种袋子，名为乾坤袋，乃是这个世界武者必不可少的辅助之一。内含乾坤空间，足有五个立方，可装放食物、武器等东西。
<br/>　　为武者提供极大的方便。
<br/>　　不过，乾坤袋的价格不便宜，起码数百两，凡境以下的武者一般买不起。
<br/>　　萧逸眼前一亮。
<br/>　　当杀手这行的，风险大，但报酬也极高。杨伏这种有名的杀手，身家自然丰厚。
<br/>　　而且杀手独来独往，肯定将全副身家都随身带着。
<br/>　　正当萧逸打算打开乾坤袋看看里面有什么东西时，门外却传来了一阵脚步声。
<br/>　　“有人来了。”萧逸快速将乾坤袋收入怀中。
<br/>　　“扣扣”
<br/>　　“萧逸少爷睡下了吗？三长老请你到议事厅。”
<br/>　　这是萧家下人的声音。
<br/>　　萧逸皱了皱眉头，暗道，“这么晚了，还没结束议事吗，请我去又是干嘛？”
<br/>　　萧逸心头冒出一股不好的预感。
<br/>　　轻轻地为依依盖好被子，萧逸走出了房门，往家族议事厅而去。
<br/>　　.....
<br/>　　议事厅，是萧家的权利中心，几乎所有大事，都是在这里由各大长老与执事共同商议并作出决定的。
<br/>　　当萧逸来到这里时，发现偌大个议事厅，坐满了族人。
<br/>　　九大长老中，来了七个；各大执事，几乎来了一大半；还有几个家族子弟。
<br/>　　萧逸出现的瞬间，几乎所有族人都对他投向了异样的目光。
<br/>　　“逸儿，过来这边。”三长老萧重沉声说着，摆了摆手。
<br/>　　三长老是代理家主，坐于议事厅首位。
<br/>　　萧逸点点头，坐到了他旁边的位置上。
<br/>　　“三长老，这么晚了，叫我来所为何事？”萧逸问道。
<br/>　　三长老沉默了一下，脸色有些复杂，道，“叫你来，是有些事要说。”
<br/>　　萧逸发现，厅内所有人里，除了三长老神色复杂，面露愧疚外....
<br/>　　其余所有人都是一副幸灾乐祸的样子看着他。
<br/>　　“三长老有话直说就是。”萧逸道。
<br/>　　“哎。”三长老叹了口气，要说的话却始终卡在喉咙里，说不出来。
<br/>　　“哼。”这时，一个面容霸道的中年人站了起来。
<br/>　　正是五长老，萧若寒之父。
<br/>　　而在他身旁，还有一个面容俊朗的年轻人，正是萧若寒。
<br/>　　“三长老说不出口，那就由我来说。”五长老满脸不屑地看着萧逸，道，“萧逸，经过我们各大长老和执事一整天的商议，决定废除你少家主之位。”
<br/>　　“毕竟，你毫无天赋，资质低下，却占着大量的修炼资源，这对其他出色的家族子弟来说，是很不公平的。”
<br/>　　“当然了，念你父亲，也就是上代家主，以往对家族做出过莫大贡献。我们可以考虑暂时保留你的少家主之位，但，你必须让出半个月后进紫云洞修炼的名额。”
<br/>　　五长老说罢，满脸霸道之色地坐了下去。
<br/>　　“叫我来，原来是为了这事。”萧逸嘴角发出一丝冷笑。
<br/>　　一旁的三长老也是满脸愁容，道，“逸儿，我叫你来，就是想让你来做出决定。到底是保留你的少家主之位，还是保留半个月后的修炼名额。”
<br/>　　三长老满脸愧疚之色，他一直待萧逸视如己出，更是受萧逸父亲之托照顾萧逸，可现在，竟让萧逸陷入如此艰难的境地，他的心不安。
<br/>　　“三长老不必难过，我知道你尽力了。”萧逸看出了三长老心中想法，安慰一声。
<br/>　　他不难想象，三长老必然是尽力了。否则，以五长老赶尽杀绝的作风，现在自己连做出选择的机会都没有。
<br/>　　而是少家主之位和修炼名额都被双双废除。
<br/>　　紫云洞，乃是萧家的宝地和底蕴。平日禁止族人进入，每三年才开放一次，只有最出色的子弟才能进入其中修炼十日。
<br/>　　而且，名额只有十个。
<br/>　　据说，紫云洞内灵气极为充裕，比外面的天地灵气浓郁了数十倍都不止。
<br/>　　所以，每三年一次的开放，那十个名额都是全家族年轻子弟争破头都要抢夺的。
<br/>　　不过，萧家有一个规矩，少家主可无需争夺，直接获得一个名额。
<br/>　　“萧逸，速速做出决定。”五长老霸道的声音再次响起，“我等商议了一天，如今天色已晚，你不要再浪费我们的时间了。”
<br/>　　一旁的七长老也说道，“萧逸，你资质低下，说得难听点的，你就是个废物。家族这些年一直赐予你修炼资源，已是对你莫大的恩赐，得心怀感激才是。”
<br/>　　八长老也训斥道，“萧逸，紫云洞三年才开放一次。你既然没那个能力，进去了也是浪费时间，倒不如将机会留给其他有需要的家族子弟，也算是你为家族出一分力了。别尽当个混吃混喝的废物，让萧家蒙受耻辱。”
<br/>　　“住口。”三长老顿时怒了，“逸儿如何，还轮不到你们来教训。”
<br/>　　萧逸也冷笑一声，这三位长老太过无耻霸道了。这些年来，修炼资源从来都是被人抢夺的，哪来的什么恩赐。
<br/>　　当然，如果是以往的萧逸，今天再怎么被欺凌，也只能打碎牙齿往肚子里吞。
<br/>　　可现在的萧逸，绝不会受这样的气。
<br/>　　“我拥有进紫云洞修炼的名额，是萧家多年来一直的规矩，这是我的权利，我为何要放弃掉？”萧逸冷声反问道。
<br/>　　五长老沉声道，“你的意思是，保留修炼名额，废去少家主的头衔了？”
<br/>　　“好，那就这么定了。”七长老和八长老同时附和。
<br/>　　看他们兴奋的样子，似乎更想看到萧逸废去少家主的地位。
<br/>　　“我有说过不要少家主的位子吗？”萧逸双手一摊，反问道。
<br/>　　“你什么意思？”五长老的语气骤然变得冰冷起来。
<br/>　　“又想保留修炼名额，又想留下少家主之位，萧逸，你不要痴心妄想了。”七长老训斥道。
<br/>　　“是吗？”萧逸丝毫不惧，反问道，“我倒想问问，五长老和七长老，你们有什么资格废除我的少家主之位？”
<br/>　　五长老霸道地说道，“这里是议事厅，众长老和执事决定的事，难道你还想拒绝不成？”
<br/>　　萧逸冷笑道，“一般的事宜，自然由长老和执事们商议即可。但，按照族规，少家主之位，必须要家主废除；若家主不在，则由九大长老联合废除；若长老中有不同意见，则交由全体族人投票决定。”
<br/>　　“只有超过九成以上的族人附议，才可废除少家主之位。”
<br/>　　“胡闹。”五长老怒道，“萧家家大业大，族人们个个都要打理家族的事宜，哪里有空全部聚集起来管你这破事儿。”
<br/>　　“切，真当自己是什么天才了。”八长老不屑道，“你还没那个资格让整个萧家都劳师动众。”
<br/>　　萧逸冷笑道，“那就没有办法了，毕竟，国有国法，族有族规。难道，你们几位长老想罔顾族规，无视祖宗先辈们订下来的规矩，不将我萧家先人们放在眼里，打算亵渎他们吗？”
<br/>　　萧逸一顶大帽子盖在他们头上，让他们哑口无言。
<br/>　　其实，规矩归规矩。毕竟，少家主在萧家是一个很重要的职位。如果是以前，历代的少家主都是天赋极高之辈，当然要全体族人去决定。
<br/>　　可萧逸嘛，本身就不是名副其实，空有名头，却没有任何权利。
<br/>　　只不过，如果萧逸咬定了族规，五长老他们也确实没有办法，除非，他们想罔顾族规，公然无视祖宗定下的规矩。
<br/>　　“你...你...小废物，休得牙尖嘴利。”七长老一时语塞。
<br/>　　“萧逸，你不要胡说八道。”八长老也是一惊。
<br/>　　五长老皱起了眉头，暗道，今天的萧逸怎么忽然变得如此强势，竟然学会反击了。
<br/>(本章完)
