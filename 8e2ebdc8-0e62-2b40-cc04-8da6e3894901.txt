<br/>　　狂龙帝听到这话，大笑着开口道。
<br/>　　这人人谈之色变的天人大劫，对如今的狂龙帝来说，却是根本不被他放在眼中。

<br/>　　更何况是在这无数岁月之中，在跟混沌之雷的对抗之中，现在狂龙帝的力量，也已经是今非昔比了。
<br/>　　度过这天人大劫，狂龙帝有着绝对的自信。
<br/>　　“那我就等着龙帝的好消息。”
<br/>　　楚风眠说了一句，身形一闪，便是离开了宫殿之中。
<br/>　　天人大劫，只有狂龙帝自己度过，楚风眠也不会去参与，不过狂龙帝度天人大劫，楚风眠在一旁看着，对楚风眠来说也是一个机遇。

<br/>　　可楚风眠却是隐隐预约的可以预测到，他的天人大劫绝不会那么好度过。
<br/>　　楚风眠度过逆天大劫，成就天地异种，已经是逆天而行了，为天地所不容。
<br/>　　这天人大劫，乃是天道诛杀楚风眠的最佳机会，只怕楚风眠的天人大劫，才是真正的劫难。

<br/>　　楚风眠离开了宫殿之中，两道身影便是围了过来，正是尹龙仙跟海公主。
<br/>　　尹龙仙虽然是被狂龙帝喊出了宫殿，可一直也是十分在意楚风眠跟狂龙帝的交谈。
<br/>　　她也隐隐约约猜到了，这一次楚风眠跟狂龙帝交谈的内容，绝对是跟狂龙帝身上的伤势有关。
<br/>　　“楚公子，师尊的情况如何？”
<br/>　　尹龙仙看了楚风眠出来，焦急的问道。
<br/>　　“万事大吉，在过几天，你就能见到你师尊了。”
<br/>　　楚风眠看着尹龙仙，笑了笑说道。
<br/>　　尹龙仙，狂龙帝这一对师徒，倒是感情极好，这尹龙仙脸上的担心之色，可不假。

<br/>　　尹龙仙虽然不了解楚风眠跟狂龙帝到底谈了什么，可看着楚风眠一脸笑意的样子，也放松了下来。

<br/>　　尹龙仙有些奇怪，可她也相信了楚风眠的话，急忙的离开了，去收拾这山峰之中的东西。

<br/>　　所有的十几枚龙鳞，都是在吞天秘术之中爆炸开来，化为了一小团太初之气。
