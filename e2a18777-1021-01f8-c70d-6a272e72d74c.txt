        &lt;h3&gt;第三卷第四百章  准备提拔夏颜&lt;/h3&gt;
<br/>
<br/>        洛城的两个项目分了一半夏颜的心思，也拖住了呼延临庆，夏颜在西城的人只有宋长生最贴心，顾恩赐有自己的小九九，许弋澄有自己的事情要做。
<br/>
<br/>        宋长生陪着夏颜一起去了陈家，看到了陈家对夏颜的态度，心里对夏颜多了一丝丝奇怪的同情的感觉，对陈家人的态度很反感，他们连最起码的礼仪都没有，太让人不可理喻了，尤其是陈伟冯，夏颜给陈伟冯那么多钱，还说是一家人，虽然夏颜喜欢用一些手段，想一想也无可非议，工作和商场上拼的是智慧和谋略，谁有本事谁就能站在顶端，世界上其实都是优胜劣汰，强者为尊。
<br/>
<br/>        从冯睿家里出来，夏颜的心情很差，宋长生将夏颜送到单位楼下，对夏颜说：“我下午来接你下班，我们一起去吃点东西？”
<br/>
<br/>        “我今天不想出去，我想休息一会。”夏颜有些疲惫，她努力的扮演着，可是在冯睿的灵堂前，她的心还是很慌乱。
<br/>
<br/>        “那好吧，我来接你，送你回去。”
<br/>
<br/>        “不用，我自己回去，车在车库放了好几天。”夏颜拒绝了。
<br/>
<br/>        夏颜下午要和老马见面，这几天俩个人基本天天在一起，夏颜撞了冯睿之后回来告诉老马了，老马心里当时很吃惊，但是过一会就释然了。
<br/>
<br/>        他和夏颜的事情不能让第三个人知道，尤其是不能让唐红莲知道，唐红莲心思细腻，是做事情不达目的誓不罢休的那种人，唐红莲曾经对老马说：这个世界上没有永远的敌人也没有永远的朋友，唯有利益的等价交换才能产生稳固的堡垒。
<br/>
<br/>        “那么你和我呢？”那个时候老马还年轻。
<br/>
<br/>        “我说的话对任何人都一样，包括我的亲人，亲情和爱情也是一样，不可能只是一方永远的付出，只有相互付出才能达到平衡。我和你，都是需要双方共同付出，我付出你物质上的东西，你会给予我情感上的回报，你付出物质上的东西给我，我也会同意的付出，如果我一味的付出，你一味的索取，时间长了我觉得心理不平衡，你觉得理所当然，这样我们就会产生矛盾。”
<br/>
<br/>        这么多年以来，唐红莲和老马就保持着这样的一种平衡，他们相安无事，相互利用也相互取暖，老马在年轻的时候在唐红莲处得到了不少的帮助，如今身份职位都比唐红莲要高了，自然也就不会忘记唐红莲，顺便也把唐红莲的女儿收纳在怀中。
<br/>
<br/>        当然了，如果不是夏颜主动，打死老马，老马也不会主动的，老马有时候很纠结，但是夏颜年轻，漂亮温柔也很可爱。
<br/>
<br/>        老马问夏颜的车在哪里，夏颜说她的车在车库，她开的冯睿的车，前几天车有问题一直没有时间去修理，冯睿就将她的车开过来给夏颜用。
<br/>
<br/>        老马打电话叫了一个人将冯睿的车送到了冯睿家的车库里，夏颜和宋长生在冯睿家门口站着的时候将车钥匙悄无声息的丢了。
<br/>
<br/>        这几天夏颜心里一直不舒服，老马下班也会推了应酬陪着夏颜。
<br/>
<br/>        夏颜回到家里，老马还没有到，夏颜烦躁的将包一扔坐到沙发上，闭上眼睛。
<br/>
<br/>        可是电话铃声响个不停，夏颜拿起手机看了一下是呼延临庆的，准备挂掉，可是想了又想还是接接起来了：“说，什么事儿？”
<br/>
<br/>        “颜颜，上面让我们停工配合调查，吕严华和胡安娜双双停职接受调查。”呼延临庆在电话里无力的说。
<br/>
<br/>        “那你就停工配合调查吧，我在这里想想办法，你也想办法给吕严华和胡安娜递个话，只要他们坚持住，我保证他们过一段时间无事。”
<br/>
<br/>        “现在合同款已经到了百分之九十五，有些我们已经撒出去了，如果人家要退款怎么办？”
<br/>
<br/>        “你撒给谁了，让谁吐出来不久行了。”夏颜冷冰冰的语气。
<br/>
<br/>        “你........”
<br/>
<br/>        “你给我说这话不是废话？他们都停职了，如果他们承认了他们就要吐出来，不承认就什么事情都没有，你过一段时间还可以继续，现在你应该继续安排人将这个工作怎么进行下去，坚持你该坚持的。别人家一调查你就不打自招。”
<br/>
<br/>        “我怕这一段时间是不能和他们接触的。”
<br/>
<br/>        “你傻啊，让别人去啊，那个刘清影，让她去，你告诉她，她的儿子工作兢兢业业，很有前途，让她给他们暗示一下就行了，大家都是聪明人。”
<br/>
<br/>        .......
<br/>
<br/>        老马到夏颜家里的时候，夏颜正坐在沙发上发呆。
<br/>
<br/>        “颜颜，怎么了，不高兴？”老马走过去搂着夏颜。
<br/>
<br/>        “我去弄点吃的。”夏颜拨开老马的手，朝厨房走去。
<br/>
<br/>        “晚上吃什么？”老马也跟着夏颜走进了厨房。
<br/>
<br/>        “你想吃什么？”
<br/>
<br/>        “我想吃.......”
<br/>
<br/>        “去等着吧，吃饱了才有力气干活。”夏颜朝着老马一笑将老马推出厨房。
<br/>
<br/>        夏颜从小娇生惯养，哪里能做出什么好的饭菜，好的是老马是洛城人，一碗面就可以打发了。夏颜煮了两碗酸汤面，加了一个荷包蛋端到餐桌上：“如果你要长期过来的话，我就得雇保姆了。”
<br/>
<br/>        “雇什么保姆，家里就我们俩个人不好吗？也不是天天都在家里吃饭。”
<br/>
<br/>        “可还能吃？”
<br/>
<br/>        “颜颜做的什么都好吃。”
<br/>
<br/>        夏颜等着老马将面条吃完，看着老马说：“洛城出事情了，吕严华和胡阿娜被调查了，呼延临庆也要去停工配合调查。”
<br/>
<br/>        “为什么要他停工，该干什么就什么，工程不能停。”
<br/>
<br/>        “这个工程我动用了一部分给上头送过去了，所以你看着想想办法。”
<br/>
<br/>        “洛城那边有很多是我以前的人，我问问情况，另外，在事情没有发酵之前，你离开现在的这个岗位吧。”
<br/>
<br/>        “我离开这里我去哪里啊？”
<br/>
<br/>        “陈子乔前几天说她俩个孩子马上要毕业了，她顾了工作顾不了家里，她想调到一个轻松的部门，不行了你就接替她的位置吧，我给她安排一个轻松的位置，在你下属的一个分部，她想上班了就上，不想上了就不要勉强，你不能为难她。”
<br/>
<br/>        “你给上头说了？”
<br/>
<br/>        “这种事情不用说，况且你现在和上头的关系不是很好吗？把你放在哪个位置他们也放心。”
<br/>
<br/>        “行，那你也得将洛城的事情想办法压下来，我不想闹得风言风语。”
<br/>
<br/>        “行，你现在这个位置你有没有合适的人选推荐？”
<br/>
<br/>        “我想一想吧，就这两天再考察一下。”
