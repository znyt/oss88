　　之前对付蝾螈的时候，妙心就把这件法器拿出来过一次，当时叶少阳还在内心吐槽过这是哈里波特用的法杖，很好奇看她怎么用，不过她当时没来得及使用，变故就发生了。此刻再度被她拿在手中，显然做好了应付一切的准备。
<br/>　　哭声越来越近，就在前方。大伙放慢了脚步，慢慢贴过去。
<br/>　　“等下。”毛小方突然轻轻叫了一声，拉住叶少阳，指了指地面。
<br/>　　血迹，一条长长的血迹，洒在砖石铺成的铺面上，叶少阳用脚尖轻轻踏了一下，粘稠的，说明是刚撒上去不久的……
<br/>　　那自然只有可能是曹雨兴的。
<br/>　　他，遭遇了什么？
<br/>　　“过去看看！”哭声就在前面，也不用研究这些血迹了，大伙继续往前走，长明灯的灯光范围之内，总算出现了一个人的影子：蹲在地面，后背朝着众人，双手捂着脸，在那里一抽一抽地哭着。从身上的衣服看，正是曹雨兴。
<br/>　　妙心拦住大伙，一起站住，望着曹雨兴的背影，卢晓清忍不住轻轻唤了一声：“雨兴？”
<br/>　　曹雨兴慢慢地止住了哭泣，然后，缓缓转过身来。
<br/>　　灯光照在了他的脸上，所有人都倒吸了一口气，妙心甚至捂住了嘴。
<br/>　　灯光之下，曹雨兴没有了脸皮，肌肉暴露在外，因为没有眼皮的缘故，两只眼睛显得格外的大，瞪着众人，血不断从他脸上滴落下来。
<br/>　　这画面……格外的惊悚，幸好在场的都是法师，在这方面心理承受能力异于常人，如果是普通人的话，看到这一幕不吓死也得吓昏。
<br/>　　怎么会这样？
<br/>　　卢晓清咽了下口水，喃喃说道：“雨兴，你怎么了？”
<br/>　　曹雨兴竟然站了起来，撇着嘴，发出轻轻的哭声，似乎想朝他们这边走过来，然后却猛然转身，朝远处飞奔了起来。
<br/>　　“雨兴你回来！”卢晓清怕他走失，情急中追了出去。大伙也跟着奔跑起来。
<br/>　　叶少阳追着卢晓清，说道：“我跟你说，人被撕了面皮，不可能活下来！失血过多，就算不死，也早该昏迷了，不可能还有力气奔跑。”
<br/>　　卢晓清一下站住，瞪着叶少阳：“你是说，他已经死了？”
<br/>　　“你觉得呢？”
<br/>　　法师都有阴阳眼，普通的鬼妖逃不过他们的眼睛，但是方才，他们看到的曹雨兴，的的确确是他，但肉身是他的，内在可就不一定了，叶少阳打死也不相信一个人被撕了面皮，还跟没事人似的跑那么快，再说，如果真是曹雨兴，见到他们应该求救才对，为什么要跑？
<br/>　　“不管怎么说，我们也得追上他，我们得知道，在他身上到底发生了什么事！”妙心说完，又追了出去。
<br/>　　大伙沿着血迹一路跑，直到脚下的砖石都没了，成了山岩，四周的空间也越来越大，前方再度听见了流水的声音，妙心感觉不好，让大家停下，朝四周打量过去，但由于长明灯的亮度不够，照不亮太多地方，只能依稀看到，他们好像是走到了一个山洞里，周围十分宽敞，似乎是一个很大的空间，前方不远处有潺潺流水声。
<br/>　　“你们站着，我去看看。”叶少阳拿着长明灯就往前走。
<br/>　　“我保护你。”妙心拿着她的法杖跟了上去，叶少阳无奈笑了笑。
<br/>　　没走多远，长明灯的灯光照到了前方不远处的一条地下溪流，比之前看到那一条要宽，水流也更急，脚下这条路到头之后，前方有有砖石修出来的一座拱桥。
<br/>　　曹雨兴的血迹，往左侧一直延伸到水边，然后就没了。
<br/>　　叶少阳和妙心互相看了一眼，走到桥下，往桥上看去，这才发现不是一般的桥，而是一个两头尖、中间鼓起来的一座建筑，在桥的中间最宽阔的地方，有一座花坛，长明灯照过去，发现花坛里长着很多花，一共有两种，一种是血红色的，叶少阳一眼就认出这是彼岸花，只有阴司才有的长，人间除了极少数例外情况，是不可能见到的。
<br/>　　眼下就是例外情况。
<br/>　　彼岸开人间，必有妖邪。这是法术界的一个共识。不过，这里除了彼岸花，还有一种黄色的花，上面有几个黑点，看上去像是人的笑脸，只不过笑得有点诡异。
<br/>　　“这红的是彼岸花，我认得，这黄色的是什么花？”
<br/>　　“阴生蔷薇。”叶少阳道，“这种花在阴司都是没有的，只有荫水河北岸深处才有，传闻是吸收荫水河中的残精魄所生长出来的，有说法，每一株花，都是一只魂灵的化生之形。”
<br/>　　妙心皱眉道：“你怎么知道的？”
<br/>　　“我去过荫水河北岸，离阴司至少有几百里的地方，只有在那里才能见到这种花。”
<br/>　　妙心一听，更加狐疑。“荫水河北岸，那不是阴司的地面了，你为什么去过那里？”
<br/>　　一般法师，走阴的范围仅仅局限在阴司之内，极少会有越过荫水河北岸，更不要说纵深几百里……叶少阳这么说，不可能不引起妙心的怀疑。
<br/>　　“你现在要听我说故事吗？”叶少阳耸了耸肩。
<br/>　　“我只是好奇，你一个真人牌位，怎么会去那种地方，你怎么敢去？”妙心说完，走上了拱桥。
<br/>　　看不起自己……叶少阳暗自撇了撇嘴，追了上去。再往前走了几步，长明灯照的范围更广，可以隐约看到花坛的中间，有一株巨大的花草，叶子巨大，至少有两米高，叶子看上去像是芦荟，只不过是红色的，但不是彼岸花那种血红，而是鲜红色，叶片扭曲，纠缠在一起。
<br/>　　什么鬼东西？
<br/>　　两人怔了一下，继续往前走，光线更加明亮，叶少阳突然愣住，道：“这根本就不是叶子，这尼玛是花瓣！”
<br/>　　在这些巨大的花瓣下面，有一丛叶子，墨绿色的，那才是真正的叶子。
<br/>　　也就是说，被叶少阳和妙心当成叶子的，其实是一朵花……两米来高，如此巨大的花！简直闻所未闻。
<br/>　　仔细观察之后，两人发现，这朵花有点含苞待放，从形态上看，有点妖艳，也有一种……说不出来的诡异。
<br/>(本章完)
