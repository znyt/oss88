　　“我宣布，这一届火能猎捕赛，正式结束。”
<br/>　　见最后一支新生队伍出来，苏长老目光在乱石地带扫视一圈，见无一缺席后，满意的点了点头。
<br/>　　“不管你们有何恩怨，在进入内院后，可以直接向老生发起挑战，只要对方同意，内院竞技场随时敞开。”
<br/>　　说完，苏长老特地瞥了眼最后一支新生队伍。
<br/>　　随后，目光落在慕清霄与小医仙身上：“作为这一届火能猎捕赛胜利者，在场新生，各自获得一周的火能，慕清霄，小医仙两人，额外奖励“青火晶卡”两加两周火能。”
<br/>　　“青火晶卡？”
<br/>　　苏长老话语刚落，老生队伍中一阵哗然，纷纷向慕清霄与小医仙两人，投来一连串羡慕的目光。
<br/>　　在内院中，火晶卡由低到高，火晶卡等级越高，越能深入天焚炼气塔，而想要升级火晶卡，就必须交纳火能来换取。
<br/>　　升级越高，需要的火能就越多，而两人直接获得青火晶卡，就相当于节省了三百天的火能。
<br/>　　三百天的火能，相当于他们这群普通老生一两年的积蓄，他们又怎能不羡慕。
<br/>　　此时，苏长老已经来到台下，手中捏着两张青火晶卡递了过去，道：“奖励已经在里面，你们把黑火晶卡中的火能划过去，再将黑火晶卡还给我。”
<br/>　　慕清霄与小医仙微微点头，将巨款火能划进青火晶卡后，便将黑火晶卡还给了苏长老。
<br/>　　接过黑色火晶卡，苏长老微微一笑，道：“你们表现的不错，期待你们在内院的表现，现在都跟我进入内院吧。”
<br/>　　说罢，他便率先转身，踏上一处碎石阶梯，朝着山坡上走去。
<br/>　　望着苏长老的背影，慕清霄叹了口气，距离他来到这个世界，已经有一年零一个月，距离目标也是又近了一步。
<br/>　　“仙儿，我们也上去吧。”
<br/>　　朝着小医仙招招手，慕清霄也跟在学员后面，踏上碎石阶梯，小医仙微微点头，快步跟了上去。
<br/>　　碎石阶梯并不长，也没有什么讲究，仅仅两分钟的时间，两人便来到了顶端，放眼望去，映入眼中的是一片奇异景象。
<br/>　　阶梯的后面，是一处奇异的凹陷盆地，盆地占地极广，仿佛是被一颗巨大陨石给砸出来的。
<br/>　　盆地内部，高耸的建筑林立其中，居高临下的观看，能够看见一道道犹如跳蚤一样的黑影，不断在建筑物上闪掠跳动。
<br/>　　盆地大的出奇，视线达到尽头，可却依旧只能看见高耸建筑与葱郁绿林。
<br/>　　望着眼中的景象，慕清霄眼底闪过思念之色，暗道：“多么熟悉的场景，缺少的唯独只有火影岩而已，也不知道小妮子现在修炼的如何。”
<br/>　　看到此地的迦南学院，慕清霄脑海中不禁浮现，在火影位面中的一幕幕。
<br/>　　当初，站在火影岩上方，所能看到的景象，与现在是何其的相似，只是这里的盆地，比木叶忍者村，实在大的太多。
<br/>　　他当然不可能忘记雏田，只不过那小妮子炼化过龙元，资质经过改造，而且长生不老，寿命千年，实力也不需多操心。
<br/>　　“看来，得尽快回去一趟看看，不过还是先将当下的事情处理完吧。”
<br/>　　微笑的望着那些新生面孔上，难以掩饰的震撼，苏长老拍了拍手，道：“呵呵，小家伙们，欢迎来到迦南学院的核心地带，内院！”
<br/>　　慕清霄嘴角牵起一丝笑意，拥有九霄琉璃焱的他，明显能感觉到，内院中某处，确实有一股炽热的气息。
<br/>　　这股气息，陨落心炎，不会有错。
<br/>　　“行了，猎捕赛结束，所有新生都跟我回内院吧，等将你们都安顿好，你们便真正的成为内院的一员了。”
<br/>　　说着，苏长老目光在新生队伍中扫视一圈，道：“相信我，只要你们能够在这里坚持下去，等以后出去，你们将会为自己的进步，感到一切都是物有所值。”
<br/>　　所有新生下意识的点了点头，在那些老生身上，他们已经体会过，在内院中修行后，效果是多么的明显，虐他们不要太简单。
<br/>　　同时，他们也非常期待，下一届火能猎捕赛。
<br/>　　“所有学员都跟上。”
<br/>　　手一挥，苏长老身形便化作一道模糊影子，朝着视线中巨大的盆地掠去。
<br/>　　其后，所有新生也是立刻行动，犹如蝗虫过境，自山坡上俯冲而下，心中都抱着期待与兴奋。
<br/>　　......
<br/>　　......
<br/>　　进入内院后，所有人立刻就感觉到了，内院中带来的压迫。
<br/>　　内院中确实是弱肉强食，实力为尊，在这里面，拳头大才是王道，哪怕连住的地方亦是如此。
<br/>　　内院中可没有什么宿舍楼，这里建筑林立，环境自然也有所不同，往往实力强悍的学员，是绝不可能住在最外围的。

<br/>　　因此，越能激发出他们的动力，因为越是里面，就越是备受瞩目，越能接触到强者。
<br/>　　而慕清霄与小医仙的表现，在火能猎捕赛中比较优秀，所住的地方自然要稍微靠里面一点。

<br/>　　轻轻推开木门，光线顺着缝隙，率先偷偷溜进房间中，木门敞开，三道人影也走进其中。
<br/>　　领头的是一名中年男子，目光在小阁楼中扫视一圈，随后落在慕清霄与小医仙身上，目光有些暧昧。
<br/>　　“小阁楼正好有两间屋子，客厅与厨房都有，环境非常的不错，正好适合两人居住，相信你们也会满意。”
<br/>　　慕清霄在小阁楼中扫了一眼，道：“多谢，夏管事。”
<br/>　　“不用客气，以后你们两人，便是属于内院的学生了，特别提醒你们一下，在内院中独行可不好混，因为内院中势力划分明确，不用我多说，再过不久，你们就会明白。”
<br/>　　势力划分，慕清霄对于这种东西在清楚不过，弱者依附强者，强者同样也需要弱者，当然，完全与他扯不上关系。
<br/>　　说着，夏管事微微一笑，道：“不过，我看你们也没有团结合作的想法，猎捕赛的事情，我也听说了一些。”
