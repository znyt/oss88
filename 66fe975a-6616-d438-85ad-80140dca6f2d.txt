　　幽深的空间通道中，空间壁障坚固如铁，苏莫和秦不死在通道中大步而行，两人的面色凝重如水。
<br/>　　这空间通道极长，两人已经走了上百步，还没有走到尽头。
<br/>　　少倾之后，前方出现了明亮的光芒，预示着他们即将走出空间通道。
<br/>　　“前辈，准备好了吗？”苏莫停下了脚步，深深的吸了口气，声音低沉的问道，他们即将走出空间通道，面临生死一刻。
<br/>　　“嗯！”
<br/>　　秦不死郑重的点了点头，道：“苏莫，你一定要逃出去，若是有可能，我会为你争取时间！”
<br/>　　秦不死已经做好了打算，若是他逃不掉，会尽力帮助苏莫逃掉。
<br/>　　因为，只要苏莫能逃掉，他就能在外界复活。
<br/>　　“好！”
<br/>　　苏莫点了点头，随即他意念一动，身前一阵波动，一名女子出现在了身前。
<br/>　　这名女子不是别人，正是巫娴。
<br/>　　唰！
<br/>　　大手一张，巫娴白腻的脖颈瞬间被苏莫扣在了手里。
<br/>　　巫娴的眼眸扫视了一番周身的情况，发现自己身处空间通道中，顿时对苏莫冷笑道：“你必死无疑！”
<br/>　　此女面色倒是坦然，没有丝毫惧色。
<br/>　　“你会比我先死！”苏莫淡漠的说道。
<br/>　　秦不死手臂一摆，他的身边同样一阵波动，瞬间出现了十三人。
<br/>　　这十三人全部都是秦不死擒拿的巫族试炼者，个个身受重伤，躺在通道中犹如一条死狗一般。
<br/>　　这些人修为被封印，无法疗伤，虽然被抓了很久，依旧没有恢复过来。
<br/>　　“放了我，我求求你们了！”
<br/>　　“不要杀我！”
<br/>　　“我带你们出去，我带你们出去！”
<br/>　　一众巫族天才满脸惊恐，脸色惨白，出来之后立刻求饶了起来。
<br/>　　“苏莫，你再抓几个人质，更保险一点！”秦不死没有理会这些人，转头对苏莫说道。
<br/>　　苏莫见此，另一只手掌一张，将一名瘦弱的巫族天才抓在了手中。
<br/>　　有两名人质，对于他来说足够了。
<br/>　　而秦不死同样手抓两人，其身上气息卷动，直接将剩下十人卷起，围绕在他的周身。
<br/>　　如此的话，谁若想杀他，这些巫族天才，必将陪葬。
<br/>　　做完这些，秦不死大口一张，一枚玉佩飞到了苏莫的面前。
<br/>　　“苏莫，这是我的空间宝器，二弟和三弟都在其内，你比我的实力强，逃走的机会更大，你收下吧！”秦不死沉声道。
<br/>　　苏莫没有拒绝，意念一动，便将玉佩收了起来。
<br/>　　嗡嗡~~
<br/>　　这时，他们脚下的空间通道，开始剧烈的颤动了起来，显然即将关闭。
<br/>　　“走！”
<br/>　　苏莫和秦不死对视一眼，身形一动，急速向外掠去，瞬息间便冲出了通道。
<br/>　　……
<br/>　　无尽的星空之中，星河璀璨，亿万星辰发出朦胧的星光。
<br/>　　一处略显黑暗的星空深处，有一颗荒芜的星辰。
<br/>　　这颗星辰不大，直径只有数百万里，其上没有丝毫灵气，自然也没有生灵生存，有的只是数之不尽的荒山和怪石。
<br/>　　这种星辰，在无尽星空之中，属于微小的废星，没有任何价值。
<br/>　　但是此刻，在这颗废星之上，一座荒山之巅，却是聚集着五十余名武者。
<br/>　　这数十名武者之中，大多都是年轻的武者，个个面色苍白，眸中露出心有余悸之色。
<br/>　　这些武者，正是巫天御等一众试炼者。
<br/>　　而在众人的前方，虚空中伫立着两道强大的身影。
<br/>　　这两人眉心之间，都有一个三叉戟的标记，其中一人是一名中年人，留着八字胡，身形伟岸无比，仿佛能支撑整个天地，其身上的气息浩瀚苍茫，眸中神光耀目，威势吞天摄地。
<br/>　　而另一人，是一名老妪，身形瘦弱，弓腰驼背，阴冷的目光中略显浑浊。
<br/>　　这名老妪的气息，虽然不如那名八字胡中年人，但依旧骇人无比，其瘦弱的身躯，仿佛蕴含着经天纬地的力量，睥睨天下。
<br/>　　此刻，中年人和老妪眉头紧皱，面上的杀机浓郁到了极致，让得方圆数万里之内，都冰寒刺骨。
<br/>　　因为，他们巫族天冥星支脉试炼的后辈子弟，居然只出来了五十余人。
<br/>　　一千名后辈天才，进入苍穹世界历练，居然陨落了九百多人，只剩下了五十余人！
<br/>　　这简直是让两人无法接受，这是他们天冥星支族巨大的损失，此事传到族中，必然举族震怒。
<br/>　　而且，最让两人无法接受的是，大祭司的曾孙女巫娴，居然也没有回来。
<br/>　　望着前方即将关闭的空间通道，中年人和老妪锐利的目光，仿佛洞穿了无尽的虚空，看到了其内的人在快速飞出。
<br/>　　若不是空间通道即将关闭，两人就直接杀进去了，哪里会在此地等待！
<br/>　　“梵尊者、尹尊者，那个土著就快出来了，千万不能让他逃了！”巫天御眼眸赤红，死死的盯着即将关闭的通道，急声说道，他已经隐约看到了苏莫的身影。
<br/>　　现在的巫天御非常凄惨，全身浴血，披头散发，连左臂都被斩掉了。
<br/>　　八字胡中年人和老妪默不做声，两人眸中惊天的杀机，便说明了一起。
<br/>　　唰！唰！
<br/>　　下一息，光芒一闪，两道人影从即将关闭的通道中冲了出来。
<br/>　　苏莫和秦不死乍一离开通道，便立刻止住了身形，因为他们知道，根本不可能直接逃掉。
<br/>　　目光转动，扫视四方，苏莫的眼眸明亮无比。
<br/>　　他看到了一望无际的荒芜大地，也看到了星光璀璨的浩瀚星空，更看到了前方的一众巫族之人。
<br/>　　他的目光，瞬间便落在了为首的八字胡中年人和老妪的身上，脸色凝重如水。
<br/>　　他感受到了铺天盖地的杀机，重如山岳的磅礴威严，让得他的身体瞬间僵硬，冰寒入骨。
<br/>　　“这就是尊者吗？”苏莫心中自喃，牙关紧紧的咬在一起，尊者，好像并没有他想象的那般强大！
<br/>　　“就是他！”
<br/>　　“就是这个土著，杀了我们这么多族人！”
<br/>　　“两位尊者大人，杀了他们，碎尸万段！”
<br/>　　“为所有死去的族人报仇！”
<br/>　　“……”
<br/>　　一众巫族天才，见到苏莫和秦不死出来，顿时放声大喝了起来，个个面上显出怨毒之色，恨不得喝苏莫的血，吃苏莫的肉，方能解心头之恨。
<br/>　　他们都是巫族天才，平日了顺风顺水，哪里吃过如此苦头，简直将苏莫恨到了极致。
