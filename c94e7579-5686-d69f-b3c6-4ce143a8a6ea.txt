　　后土部落中，女娲思索良久，说道：
<br/>　　“道友，你这个条件太苛刻了，恕我不能答应。不过，若是改变一下，改为在只要是妖族不出现灭族危机，我都不会插手两族之争。如何？”
<br/>　　女娲嘴上虽是如此说道，但心里却在想，我妖族已经创出了可以克制你巫族的大阵了，若是再次大战，谁胜谁负还不一定呢？
<br/>　　主要是她对于自己的妖族是非有信心，她自信妖族不灭被灭。但尽管如此，她也不敢将话说得太满，所以，还是留下了一个限制，这个限制就是妖族灭族之危。
<br/>　　后土深深看了看女娲一眼，思索良久，觉得可行。她想了想，道：
<br/>　　“到时就算出手，你最多就是庇护你妖族，但不可出手灭杀我巫族。若是觉得这可行，那就可以交换。”
<br/>　　“可以。”
<br/>　　女娲思索了一会儿，觉得以妖族的实力，巫族不可能将其灭族。但若是出现意外，妖族真的接近灭族，到那时，巫族也可能将近灭族了。因为为了攻打妖族，巫族决不可能毫发无损，最大的可能就是两败俱伤，同归于尽。既然如此，那时自己杀不杀剩余的巫族又有何区别呢？还不如就此答应她的条件。
<br/>　　“道友欲以何物来交换？要知道，九天息壤可是土之祖源，先天法器，是天下间最为顶级的灵物。你也不用拿灵宝来换，你是知道的，我巫族并无元神，可不能使用你们的灵宝什么的。”
<br/>　　后土见她答应，也不奇怪，毕竟这关系到她的圣道。她现在关心的是女娲能够拿出何种宝物来交换她手中的九天息壤。
<br/>　　女娲沉吟片刻，接着有些心疼地取出一滴闪烁着青色光芒的血滴。这一血滴出现的瞬间，四周的天地仿佛变得沉重起来，莫大的威压出现在天地间，就仿佛不周山上的威压。
<br/>　　“这……这是——”
<br/>　　后土脸色大变，惊骇道。因为在这滴青色液体出现的瞬间，她不仅能够感受莫大的威压，更能感受到其中那股与自己十分亲切的感觉。这就仿佛是自己遇到了自己最亲之人的那种感觉。而洪荒之中能够给她这种感觉的人，只有自己的巫族。但这股气息比起其他祖巫给她的感觉还要亲切，这是盘古父神身上的气息。
<br/>　　女娲微微颔首，道：“不错，这是盘古脊骨中的一滴精髓。这对于你们巫族来说，没有什么比它更好的了。”
<br/>　　“竟真是盘古父神的精髓。”后土低声呢喃道，语气之中，还有一种不可置信。
<br/>　　“想必，我以此作为交换，可以了吧。”
<br/>　　其实女娲拿出这件东西来说，还是很心疼的。但这东西她得到好些年了，她都不能将它炼化，虽然她知道这是一件好东西，但不能炼化又有何用。尽管有些不舍，但她还是将它拿出来，作为交换。
<br/>　　“可以。”
<br/>　　后土深吸一口气，缓缓道。
<br/>　　接着，后土从祖巫宝库之中，取出一团脸盆般大小的黄土，上面闪烁着似金非金的黄光，一股浓郁至极的土源气息扑面而来。
<br/>　　“给，道友。这就是你要的九天息壤，你可要注意了，这九天息壤遇风而涨，变幻无穷。”
<br/>　　“多谢道友成全。”
<br/>　　女娲也是有些激动的接过后土手中的九天息壤，之后将那团闪烁着青色玄光的盘古脊骨精髓送给后土。
<br/>　　“道友，告辞。”
<br/>　　接着，女娲求圣心切，直接告辞离开。
<br/>　　在她走后不久，与后土祖巫比较靠近的玄冥祖巫来到后土部落。
<br/>　　“妹妹，你没事吧，我听说那妖族的女娲来到你的族地，她有没有欺负你了？”
<br/>　　“没事，你不用着急，倒是姐姐，你一身雨之法则内敛于身，好似大海汹涌，看来是开辟法则空间，开始积累法则之海了。已经快要达到祖巫圆满了。”后土看着玄冥，笑道。
<br/>　　“妹妹你不也一样。对了，你对都天神煞大阵参悟得如何？”玄冥祖巫笑着问道。
<br/>　　“已经掌握了。”
<br/>　　……
<br/>　　不周山之前的那个山谷之中，女娲去而复返。此刻，调整好心态，她娇俏的小脸上露出凝重之色，目带坚定，还有一股决绝之色。显然，创造一族这种事情，可不像嘴里说的那般简单。
<br/>　　女娲以三光神水、天一真水混合九天息壤形成泥土，而后开始捏泥造人。不过片刻时间，女娲便已捏好两个泥人。这神土与神水都是洪荒之中顶级之物，最是非凡，这泥人刚一成形，女娲的元神轰鸣，体内的鸿蒙紫气开始散发出浓郁的造化之光，此光玄之又玄，弥漫到四面八方。
<br/>　　当此光接触到那两大泥人之时，嗡鸣声中，那两个泥人全身绽放出彩色之光，下一瞬，地上便跳出一男一女，两个有着先天道体的生灵。
<br/>　　“真的成功了？！”
<br/>　　女娲惊喜异常，看着两个目露茫然之色的生灵，感受到那元神之中在剧烈震动的鸿蒙紫色，急忙继续捏造起来。
<br/>　　就在女娲继续一边捏泥造人，一边感悟元神之中的鸿蒙紫气时，谁也没有注意到，于九天之上，飞下一道微小的灵光，落入到那第一个被女娲创造出来的男生灵脑海之中。紧接着，男子眼神发出异样的神采，但随即，便被敛去，恢复原样。
<br/>　　女娲继续在捏泥造人，渐渐地，随着时间流逝，泥人越来越多，被造化玄光扫过的一个个都化作相貌各异的先天道体生灵。这些生灵初生，眼神中露出迷茫之色，他们看着女娲的动作，立刻就知道，是眼前这人创造了自己等人，于是他们一个个朝着女娲之处，虔诚朝拜。
<br/>　　渐渐地，女娲身上的造化玄光越来越强，她的全身散发着璀璨的光芒，她身上的气息也越来越强，越来越浓郁。可是，随着不断的捏泥造人，感悟鸿蒙圣机，她的元神消耗也越来越多，她的脸上开始露出疲劳的神色。
<br/>　　这个时候，她突然感觉到自己之前在不周山扯下的那株先天葫芦藤发出异样的震动。她神情一动，若有所思，取出葫芦藤，将自己的精血滴入泥土之中，与之混合。她的精血中蕴含着她的造化大道，这与用手捏有着同样的效果。
<br/>　　用藤条沾些泥土，挥臂甩出，随即就有造化玄光笼罩过来，山谷之中，顿时出现一个个相貌各异的生灵，他们带着虔诚和感激，不断向女娲叩拜。
<br/>　　女娲不停，继续沾泥挥洒，一个个生灵就这样被创造出来。山谷之中被创造出来的生灵越来越多，不一会儿，便将整片山谷占满了。一直到第十二万九千六百个生灵被创造出来，共合一元会之数，女娲才停下手了。
<br/>　　她沉默不语，静静地感悟着元神之中的鸿蒙紫气，浩大的紫气从她身上传出，璀璨耀眼，磅礴无边，光亮无比，将她衬托的无比高贵。
<br/>　　“拜谢圣母娘娘，感谢娘娘造化之恩！”
<br/>　　山谷之中，一众被创造出来的新生灵，望着那犹如神女般笼罩在神光内的女娲，朝着她不断地虔诚叩拜。
<br/>　　“自此以后，尔等便称之为‘人’，尔等皆是人族！”一阵沉默后，女娲大声宣布道。
