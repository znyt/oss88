　　萧逸恭声说着，行了一礼。
<br/>　　随后，等待着总殿主的首肯。
<br/>　　在这等人物面前，萧逸虽拒绝，可也不敢放肆。
<br/>　　再者，也不知这总殿主性格如何。
<br/>　　若是那等性格乖张之辈，一个不高兴，怕是一巴掌就能扇他出去。
<br/>　　孰料，总殿主只是淡淡一笑。
<br/>　　“太高看吗？”
<br/>　　“呵呵，年纪轻轻，便是双殿总殿副殿主头衔，更兼四方城城主。”
<br/>　　“百院之争上，一战成名，各大学院首席甚至不敢上场与之交手。”
<br/>　　“你若太谦虚，谦虚过头，可就是狂妄了。”
<br/>　　萧逸看着总殿主那不怒自威的面庞，脸色微变，连忙拱手道，“不敢。”
<br/>　　“不敢？”总殿主再次一笑。
<br/>　　“我看过你的资料卷宗，可没见有什么事是你不敢的。”
<br/>　　“再者，得了我修罗殿的最强体修功法之一，又得了那些珍贵的修炼之物，你觉得你还可以拒绝？”
<br/>　　萧逸脸色一惊，手中连忙齐出一个乾坤戒以及那本功法。
<br/>　　“禀总殿主，乾坤戒内之物，小子未动分毫。”
<br/>　　“帝岳不灭体，小子也只是初窥门径，并未详细修炼。”
<br/>　　萧逸心头不禁松了口气。
<br/>　　幸好他向来谨慎，并未用那些修罗太荒石。
<br/>　　他早便猜测，自己与总殿主非亲非故，又无半分交情，总殿主给他这份大礼，必有原因。
<br/>　　“你…”总殿主看着桌案上的乾坤戒与功法，眉头一皱。
<br/>　　下一秒，皱着眉头一松，眼中闪过一丝精光。
<br/>　　“不错，倒是个精明的小子。”
<br/>　　“不过。”总殿主语气一沉，“若老夫说，非要你帮此忙不可呢？”
<br/>　　“这…”萧逸脸色微微有些难看。
<br/>　　刚想说些什么。
<br/>　　忽然，一股莫名的气势，重重地压在了他身上。
<br/>　　总殿主淡淡一笑，“你若不回答，老夫便当你答应了。”
<br/>　　萧逸脸庞抽了抽，他现今不仅动弹不得，甚至是口不能言。
<br/>　　那股莫名气势，毫无疑问来自于总殿主。
<br/>　　总殿主的气势压制之下，他能回答才怪。
<br/>　　“好，此事，便算你应允下来了。”总殿主面庞上闪过一丝笑容，亦收回了气势。
<br/>　　萧逸身体一松，脸庞抽了抽。
<br/>　　总殿主看了萧逸一眼，轻笑道，“小子，不必这副表情。”
<br/>　　“老夫又不是让你去做些太麻烦的事，不过是让你进入古帝之墓时，顺带寻些东西罢了。”
<br/>　　“寻些东西？”萧逸面露疑惑之色。
<br/>　　现在，他也只能答应下来。
<br/>　　“嗯。”总殿主点点头，脸色，恢复了一贯的威严与严肃。
<br/>　　“接下来要说之事，事涉一些秘辛。”
<br/>　　“首先，老夫先问你，你可知晓，修罗殿为何称之为修罗殿？”
<br/>　　萧逸按捺下心头疑惑，点点头，道，“自是知道的。”
<br/>　　“上古之时，修罗殿武者，人数最少。”
<br/>　　“但修罗殿武者，却又个个勇猛，一身肉体力量惊人无比。”
<br/>　　“抵御妖兽时，往往悍不畏死，哪怕是以凶狠著称的妖兽，亦为之胆寒。”
<br/>　　“往往一人，便可以一当百，浴血奋战。”
<br/>　　“每每一场大战下来，孤身一人，踏于无数妖兽尸体之上，浑身浴血，宛若修罗。”
<br/>　　“修罗殿，故有此称。”
<br/>　　对于修罗殿，萧逸还是比较了解的。
<br/>　　孰料，总殿主摇了摇头。
<br/>　　“那是其一罢了。”
<br/>　　“再者，那是外人对我修罗殿武者的看法。”
<br/>　　“我修罗殿，怎可能因外人的看法，而定下名字。”
<br/>　　“那是？”萧逸皱眉问道。
<br/>　　“源于一物。”总殿主脸色凝重。
<br/>　　“记住，今日之事，你是总殿认可的副殿主，故能知晓，但不得外传。”
<br/>　　“咕。”萧逸闻言，点了点头，脸色也微微有些难看。
<br/>　　能让总殿主如此凝重之事，不，秘辛，怎可能会是普通秘辛。
<br/>　　这等秘辛，若不知道也就罢了，若知道了，指不定是祸是福。
<br/>　　他现在只能祈祷，别是太过麻烦之事。
<br/>　　“总殿主请说。”萧逸亦凝重地点了点头。
<br/>　　总殿主沉声道，“此物，可追溯之上古之时，甚至更久，由我修罗殿的某位创立者偶然得之，乃是一门功法。”
<br/>　　“功法？”萧逸忽然想到了什么，脸色有些怪异。
<br/>　　总殿主并未注意到萧逸脸色，自顾道，“这是门体修功法，名唤修罗战体。”
<br/>　　“此门功法，堪称逆天，乃是我修罗殿有史以来最强之功法。”
<br/>　　“我修罗殿，因此而得名。”
<br/>　　说着，总殿主看了眼萧逸。
<br/>　　“不过很可惜，无数年来，这门功法，不知何时起，忽然遗失。”
<br/>　　“遗失的，还是最重要的前几层。”
<br/>　　“整门功法，共计12层，层层逆天。”
<br/>　　“偏偏前七层，就此遗失。”
<br/>　　“后虽经我修罗殿历代强者修补，却无补于事。”
<br/>　　“但，仅仅是通过后5层功法，便已让得无数殿内强者领悟并创造出一门门绝世体修功法。”
<br/>　　“你那本帝岳不灭体，便是其中之一。”
<br/>　　萧逸闻言，心头闪过一丝恍然，但脸上的表情，仍旧不可抑止地怪异着。
<br/>　　总殿主没有理会萧逸的脸色，直接道，“我要拜托你之事，便是进入古帝之墓，查探是否有这前7层的下落。”
<br/>　　“古帝之墓内，莫测滔天。”
<br/>　　“虽每隔15年开启一次，但无数年来，无数次开启，亦无数为天骄获得莫大机缘。”
<br/>　　“但，至今为止，还从未有人能真正窥其奥秘。”
<br/>　　“墓内主局，亦还未有任何一位天骄能堪破。”
<br/>　　“但在老夫看来，你若当破局者，必能有一线希望。”
<br/>　　萧逸脸色怪异，并未详细听总殿主后面的话。
<br/>　　只是喃喃自语。
<br/>　　“修罗战体前7层遗失，后5层在总殿内？”
<br/>　　总殿主点了点头。
<br/>　　“若真能查探到下落，你便是立下修罗殿之莫大功绩。”
<br/>　　“当然，若不能，老夫也不会与你计较。”
<br/>　　“古帝之墓，本就是你的一份机缘，老夫拜托你之事，也不过是让你随手为之。”
<br/>　　“若不能破局，老夫亦不勉强。”
<br/>　　总殿主认真地凝视着萧逸。
<br/>　　萧逸沉默了一下，半晌，道，“敢问殿主，若能查到前7层下落，如何？”
<br/>　　总殿主笑笑，“若能查到，你有何要求，修罗殿一切应允。”
