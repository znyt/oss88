　　他把自己这一连串的想法提出来，大伙都沉默不语，过了一会，四宝瓮声瓮气地说道：“还是要去做的，未来的世界，也是我们这样一步步走过去的，如果我们什么都不做，未来的我们，也会跟我们一样，因为我们的‘因’，而产生某些结果。”
<br/>　　叶少阳道：“我当然知道，但是这样一来，你不觉得这一切都逃不出宿命吗？”
<br/>　　大家又沉默了，过了一会老郭说道：“那个，咱们之前不是讨论过吗，应该是平行宇宙，每一个选择的节点都会衍生出不同的世界，所有人的命运也都不一样了，所以，这没什么好说的。”
<br/>　　道理，叶少阳自然是懂得，但心里还是有些疙瘩。
<br/>　　这时候，四宝突然深吸一口气，分别看了看他们，道：“我想到一个很可怕的事情……少阳，你现在最想做什么？”
<br/>　　“现在？吃饭，我饿了。”叶少阳不好意思地揉了揉肚子。
<br/>　　“靠！我在说正事！”
<br/>　　“我怎么知道你要问什么，只好如实回答你。”
<br/>　　“我意思是，你现在得到了山海印，最想做什么！”
<br/>　　叶少阳看着手中这块其貌不扬的大印，认真思索起来，道：“老实说，我很想用它去穿越时空，看一看未来的我在做什么。”
<br/>　　四宝满意地点点头，道：“还有还有。”
<br/>　　“还有……”叶少阳也深吸一口气，道：“我想回到过去，有很多事要做，第一件就是……回到海眼之战之前，去救下秋莹……”
<br/>　　突然心头一颤，望着四宝的目光也变得尖锐起来，他猛然明白了四宝的意思。
<br/>　　四宝点了点头，道：“你能想到的，未来的你一定也能想到，但是，秋莹还是死了。”
<br/>　　大伙也都反应过来，感到了事情的恐怖，老郭琢磨半天说道：“那就有三种可能，第一，你一直没找到使用山海印的法子，也就没法穿越时空，第二，你能用山海印，但是没有这么做……第三，你做了，但是改变的是别的时空的进程，我们这里并没有改变，该发生额还是发生了。”
<br/>　　四宝补充道：“还有一种可能，你回到过去，试图改变，但被某种不可知的力量给阻止了，很多科幻电影里就是这么演的嘛。”
<br/>　　叶少阳认真思索起来，第一种……不太可能，徐福机关算尽，花费了这么大的力气才把山海印送到自己手上，不可能送个废物给自己，否则那几句所谓的口诀也没用了。
<br/>　　至于后面三种可能，叶少阳更偏向第二种：他觉得以自己的性格，就算真能穿越时空，也未必会去改变历史，因为一切都已经发生了，贸然改变过去，会引发什么一连串的反应，谁也不知道，自己这方面是谨慎的，应该不会去冒险……
<br/>　　想到最后脑袋都大了，摇头无奈道：“你们说，我们这一群吃死人饭的法师，在这讨论科学问题，是不是有点跨界了。”
<br/>　　四宝道：“现在都流行跨界合作。”
<br/>　　“滚吧。”
<br/>　　半晌没做声的吴嘉伟，突然说道：“我有个问题……你们说哎，既然徐福有山海印，他也去过了未来，那一定是知道自己走将来会被伏击然后被捕，他为什么提前规避呢，总没人想自己被抓起来吧？”
<br/>　　这个问题，还真是把大伙给难住了，半晌老郭说道：“其实这还是我刚才分析的那些，他没有改变过去，无非就是出于那四个原因。”
<br/>　　叶少阳把玩着手中的山海印，喃喃道：“这些问题先放着吧，我至少先弄明白它是怎么用的，然后想办法问问徐福，就什么都清楚了。”
<br/>　　老郭严肃地说道：“小师弟，你一定要慎重使用，这东西虽然好，但可能带来无穷后患。”
<br/>　　叶少阳点了点头，老郭这真不是危言耸听，这山海印的存在，本来就是一种破坏，而且是终极的破坏：破坏了天地时空的规则，就连徐福也说过，他不敢多用这东西，更不敢用来穿越到未来（但他还是这么做了）。双刃剑。叶少阳看着山海印，心想。
<br/>　　不过，能得到这东西，自己内心深处还是很兴奋的，当下强压着兴奋，把它收起来，跟大伙一起出去吃饭，中间又讨论了一会山海印的话题，后来大家一致认为，应该先研究如何使用它，等真的能用了，再来讨论其他，否则都是纸上谈兵，没有意义。
<br/>　　吃完饭，大伙各忙各的了，生活又暂时回到了常态：四宝去接王旭雯约会去了，吴嘉伟跟老郭一起去他店里，继续研究和实验他的剑道——最近几次战斗，他那一手快剑总是起到了一剑封喉的作用，对他大为鼓舞，要继续研究和实验。
<br/>　　瓜瓜则情绪有点低落，出去散心去了，叶少阳也不管他，主要是不知道怎么劝，秋莹死了不能复生，面对死亡，终究还要自己去慢慢接受，对他将来面对更悲伤的事也有好处。总还是会有更悲伤的事发生……叶少阳有预感，但也无可奈何。
<br/>　　他把自己关在房间里，开始研究山海印。
<br/>　　他牢牢记着徐福告诉自己的咒语：
<br/>　　八卦二生相，菩提树下生，山海乾坤定，四象占灵根。
<br/>　　这咒语到底啥意思，是要怎么运用？
<br/>　　他回想徐福以前运用山海印的情况，好像也就是直接托在手中，于是按照常规方法，托着山海印，释放灵力在手掌上，包裹着山海印，试图让灵力进入它。但它跟一般法器完全不同，就像一块普通的顽石，灵力丝毫无法进入，只能放弃了。
<br/>　　任何法器，本质上都是一样的，操控起来无非就是那几种方式，叶少阳都试了一遍，一点用没有。
<br/>　　他冥思苦想，也没什么好办法，只好暂时先放在身上——三界之内，各方势力对山海印的觊觎之心，他是体会过的，放哪里都不放心，只有自己身上最安全。
