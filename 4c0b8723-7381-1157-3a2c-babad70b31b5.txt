　　第88章
<br/>　　孟子云一直看不起山贼，今天他是头一次发现，做贼也不是一件容易的事情。
<br/>　　毕竟是做一件坏事，周围静悄悄的，一丝光亮都没有。脚踩在地上的每一步，仿佛都压着心跳，除了自己心脏砰砰乱跳的声音，周围没有任何声音。
<br/>　　他摸黑来到粮仓前，看看左右无人，从怀里掏出火石。打火时，火石摩擦出的光芒，照亮一张惨白的脸。
<br/>　　烧了吧，烧了吧……烧了就得干净，想到蛤蟆寨因为没有食物，在酷寒中走投无路，冻饿而死的模样，孟子云心里就冒出一阵异样的快感。
<br/>　　而如今，他距离成功已经很近了。
<br/>　　咔嚓……
<br/>　　咔嚓……
<br/>　　咔咔嚓嚓
<br/>　　一下又一下，今夜手抖的厉害，总是点不燃。
<br/>　　突然，一束火把猛地亮起，在完全黑暗的环境中，显得如此刺眼。
<br/>　　紧跟着，又是一束火把……瞬间，孟子云就被炙热的光芒包围，秦蛮，林少羽，孙九，柳芷，刘悲，关鱼等人举着火把出现在他周围。
<br/>　　迎着火光，一人从黑暗中走到光明处，他拥着大氅，长风将他的黑发吹起。
<br/>　　“孟少侠，是缺火么，用不用我向你借个火。”
<br/>　　一束束火照亮山寨，织成一道密不透风的网。这张网像是已张开良久，只等着自己自投罗网。
<br/>　　这一瞬间，孟子云想起义父曾对自己说起过的一个故事。
<br/>　　春秋时期郑庄公有个弟弟名叫公叔段，其母姜氏宠溺公叔段而讨厌庄公。后公叔段的行为俞加跋扈，庄公却没有惩罚。直到公叔段试图谋反时，庄公才大义灭亲动手铲除。
<br/>　　世人皆赞庄公纯孝，可是也有人说，如果郑庄公及早约束自己弟弟，也就没有后来公叔段谋反的事情。
<br/>　　纵容你，就是为了有朝一日名正言顺的杀掉你。
<br/>　　孟子云昂首，从齿缝里迸出三个字：“程大雷！”
<br/>　　狠下一条心肠，孟子云手持钢刀扑向程大雷，今天就算拼着性命不要，也要宰了这人面兽心的畜生。
<br/>　　程大雷眼神里流露出怜悯的目光，轻轻摇了摇头。
<br/>　　“多行不义……必自毙。”
<br/>　　也就在钢刀挥过来的刹那，程大雷一矮身避过刀锋，锵啷一声拔出腰上佩剑，将孟子云逼退。
<br/>　　这剑是李行哉临走时留下的，真正使用程大雷还是第一次。
<br/>　　一剑便将孟子云逼退，眨眼之间两人已经拆了七八招。孟子云心中惊骇莫名，上次动手他一脚就能将程大雷踹飞，怎么间隔短短时间，程大雷已经变得如此了得。
<br/>　　山寨升级后，程大雷也升了一级，速度、力量、反应均有大幅度提升。
<br/>　　此时的他，并不弱于孟子云。
<br/>　　“不必大当家动手，由我宰了这恶贼。”
<br/>　　秦蛮一枪便分开二人，从程大雷手中接过孟子云。程大雷毕竟不擅长用剑，再和孟子云打下去有些悬，况且今天也不用他动手。
<br/>　　交手不过三个回合，孟子云便被击败，身子倒在地上，手中却还紧紧握着那柄钢刀。
<br/>　　火光明灭间，看清一双双愤怒的眼睛。即便曾经的柳家人，此刻也对他恨之入骨，实在今天孟子云犯了大忌。民以食为天，多少人为一口饭奔波，孟子云今天却要烧毁粮仓，这是要将所有人赶尽杀绝。
<br/>　　恨吧，痛快恨吧，假若我今日不死，来日必将你们个个诛尽，刃刃杀绝。
<br/>　　可今日已经是上天无路，入地无门，想逃，谈何容易。
<br/>　　目光盯着周围，见自己距离地牢的入口不远。孟子云突然站起来，不顾一起的冲向地牢。一人拼命十人难敌，真被他不顾性命闯入地牢。
<br/>　　地牢门重重关上，显然是他在里面将牢门反锁了。
<br/>　　众人没明白怎么会事，要逃也似向外逃，怎么有人自寻绝路往地牢里钻。
<br/>　　程大雷第一个反应过来，开口大喊：“快，别让他把那怪物放出来。”
<br/>　　孟子云气喘吁吁跑入地牢，杜茂正在酣睡，看到手持钢刀的孟子云，奇怪的皱起眉头。
<br/>　　“怎么，你来杀我么？”
<br/>　　“我放你出去，你救我一命。”
<br/>　　连续三刀砍在锁门的铁链上，哐当一声铁门落地，牢门洞开。
<br/>　　杜茂一楞，随即哈哈大笑：“姓程的，老子出来喽！”
<br/>　　程大雷正让大家撞门，只听咚的一声，牢门从里面被一脚踹开。
<br/>　　大家只看到一个披头散发衣衫褴褛的怪物站在门口，宛若顶天立地一般，放声大笑。
<br/>　　“姓程的，你在哪里，老子要来杀你了。”
<br/>　　程大雷心中一惊，真的让孟子云将这怪物放出来了。如今山寨能打的不过五十几人，碰上这百人敌的猛汉，怕还真没有多少胜算。
<br/>　　“大家快快散开，别白白上前送死。”程大雷急忙喊着，像杜茂这样的猛人，杀七个八个并不如斩瓜切菜困难多少。
<br/>　　人群散开，已有人去仓库去弓箭出来，碰上这样的人，最好不要和他近战。
<br/>　　关键时刻，秦蛮手持大枪站了出来。
<br/>　　“哈，我认得你，上次没打，这次分个输赢。”杜茂。
<br/>　　“好。”秦蛮一抖枪身，宛若一条黑龙在空中舞动。
<br/>　　杜茂目光转了一圈，最好一脚踹在身后的地牢的木门上，三下两下便将半扇木门从墙上拆下来。
<br/>　　双方相对而立，就如两只野兽对峙，寻找着对方的弱点，等待发出致命一击。
<br/>　　一个土匪，一个恶霸，上山虎撞着下山虎，出海龙碰见入云龙，似乎迟早该有这一战，双雄会，龙虎斗。
<br/>　　程大雷也让人把自己的鬼面斧抬过来，说不得关键时刻，自己这三斧头也能发挥点作用。
<br/>　　电光火石之间，杜茂和秦蛮已经战在一处。杜茂挥舞着门板，大开大合，沉重的木门在其手中如若无物。秦蛮使的是梅花枪，一枪挑出，花蕊绽放，每一朵枪花中，都藏着杀机。
<br/>　　程大雷提着大斧头逡巡在周围，老实说，帮不上什么忙。不单单是他，林少羽也是提着枪，不知道该往那里扎。
<br/>　　整个山寨灯火通明，今夜一场龙虎斗，杀得难分难解，难分雌雄。
