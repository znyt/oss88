蓝烟拿着遥控器的手一抖，她本来打开电视是想分散夏夕绾的注意力的，不让她去想今晚陆寒霆和厉嫣然的订婚，但是哪里想到这条财经新闻竟然直接推送了这条消息。
<br/>
<br/>        “呵，绾绾，我们换个台，现在财经新闻竟然也报导婚礼了。”蓝烟迅速换了一个台。
<br/>
<br/>        这一次换成了娱乐频道，里面的主持人正在报导今日的娱乐头条今天这一场陆厉联姻万众瞩目，陆寒霆作为如今的商界第一贵胄，于四个月前回归帝都城，正式接管了陆氏集团，这个十几岁就玩转了商界的一代大佬在最新一期的福布斯排行榜上已经位列前茅，引爆了多少名媛千金的芳心，今天他与首富千金，帝都第一美人厉嫣然订婚，简直男才女貌，天作之合。
<br/>
<br/>        说着电视的画面迅速切换到了豪庭酒店，今天帝都所有的杂志周刊记者都已经围堵了过去，这时一辆加长版的劳斯莱斯商务豪车疾驰而来，后车窗缓缓滑落，露出了一张精致英俊的容颜。
<br/>
<br/>        是陆寒霆。
<br/>
<br/>        陆寒霆已经现身这场订婚宴了。
<br/>
<br/>        今天的陆寒霆穿了一件纯手工的白色衬衫，外面罩着一件黑色的商务马甲，马甲外面是一件同色的黑色西装，西装衣袖里露出小半截白色衬衫还有名贵的男士腕表，举手投足都透着上流贵族的倨贵和考究。
<br/>
<br/>        此时金黄的夕阳镀在他那张完美如雕琢的俊脸上，英俊的一塌糊涂。
<br/>
<br/>        他一出场，全场骚动。
<br/>
<br/>        记者们蜂拥而去，“陆总，我们可以采访一下你吗？”
<br/>
<br/>        这时一群工作人员出现了，拦住了记者，“不好意思，我们总裁不接受任何的采访。”
<br/>
<br/>        加长版的劳斯莱斯商务豪车消失在了众人的视线里。
<br/>
<br/>        夏夕绾看着电视，电视里面的画面被定格了，那是陆寒霆的最后一个镜头，夏夕绾看着男人半张侧脸，他的侧脸线条立体凌厉，抿着薄唇，整个人透着一股高高在上的威慑和不近人情的冷漠。
<br/>
<br/>        这一下蓝烟嘴角都抽了，她从财经新闻换到娱乐新闻，竟然都是在报导这一场陆厉联姻的。
<br/>
<br/>        “呵，绾绾，我们再换个台，这一次换个…动物世界！”
<br/>
<br/>        蓝烟麻溜的换台，她就不信动物世界也在报导这场联姻！
<br/>
<br/>        这时夏夕绾出声道，“蓝妈，算了，把电视关掉吧，今天恐怕所有电视台都在报导这一场联姻，我不想看。”
<br/>
<br/>        夏夕绾发话了，蓝烟就没有坚持，她将电视给关掉了，“绾绾，你不要受这些事情的影响，这世上什么都缺，就是不缺三条腿的男人。”
<br/>
<br/>        夏夕绾巴掌大的小脸看着很苍白，她垂下纤长的羽捷，点了点头，“我知道了蓝妈，我想再睡一会儿。”
<br/>
<br/>        “好，快躺下吧。”
<br/>
<br/>        夏夕绾躺下，闭上了双眼。
<br/>
<br/>        蓝烟在床边守了一会儿，这时她的手机响了，来电话了。
<br/>
<br/>        见夏夕绾已经熟睡了，蓝烟给她盖好了被子，然后拿着手机出去接电话了。
<br/>
<br/>        蓝烟一走，床上的夏夕绾就睁开了眼。
