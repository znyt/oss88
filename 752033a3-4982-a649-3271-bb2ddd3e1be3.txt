　　没过多久，苏莫便回到了自己的阁楼，一脸阴沉的盘坐了下来。
<br/>　　“太史轩！”想到之前的那名紫衣青年，苏莫心中杀机一闪浓郁。
<br/>　　不过，他现在修为太低，估计不可能是对方的对手。
<br/>　　少倾，苏莫暗叹一声，不管在哪里都要有实力，没有实力均会被人压迫，这是这个世界的法则。
<br/>　　强者为尊，弱者蝼蚁！
<br/>　　随即，苏莫抛却脑中杂念，开始仔细思索今日孙导师所讲的知识。
<br/>　　他这一坐，便是三日时间。
<br/>　　这三日时间，苏莫不仅将孙导师所讲的知识全部消化一空，更是将未来的一段时间，做了一个简单的规划。
<br/>　　首先，他当务之急就是提升武魂的等级，将武魂提升到天级九阶，然后才能想办法进化战魂。
<br/>　　其次，他的不朽奥义，明显不太适合吞噬武魂，要感悟新的奥义之力。
<br/>　　还有就是，接下来他要调查清楚关于巫族的情况，以及巫天御的所在。
<br/>　　这个比较难办，因为巫族根本不在古周星，想查出巫天御的行踪，实在是太难了。
<br/>　　坐了三日之后，苏莫又离开了阁楼，在南苑中闲逛了起来。
<br/>　　他是准备前去找木璃导师。
<br/>　　他想要提升武魂，显然不可能靠静修，准备出去历练。
<br/>　　但他对神武学府以及古周星，都一无所知，更不认识任何人，只能去找木璃了。
<br/>　　木璃是南苑的导师，平时就居住在南苑之中，苏莫询问了一番，便的得知了其具体的住处。
<br/>　　值得一提的是，或许是上次广场一事，让得苏莫的名声变得极差，他一连询问了数人，才有人告诉他木璃导师的居所。
<br/>　　木璃导师的居所，是一处幽静的院落，远离了南苑中众弟子的居所，建在一处空旷之地。
<br/>　　院落不大，占地方圆百丈，但建造的极为精美，院墙不知道是用什么材料建成，仿若是翡翠一般，闪烁着迷蒙的莹光，美轮美奂。
<br/>　　“何事？”
<br/>　　苏莫还未走到院落的大门前，一道清冷的声音，便传到了他的耳中。
<br/>　　苏莫闻言一怔，急忙向阁楼方向抱拳，道：“弟子来此，一是要感谢木璃导师上次解围，二是想向导师请教一些事情！”
<br/>　　苏莫言罢，院落中便没了回音。
<br/>　　咯吱!
<br/>　　少倾之后，院落的大门打开了，一身白衣的木璃走了出来。
<br/>　　“感谢就不必了，有什么问题你就问吧！”木璃声音淡淡，却有充满了空灵之感。
<br/>　　“哦，弟子是想问问，这神武学府中都有什么合适的历练之地！”苏莫沉声道。
<br/>　　木璃闻言，略一沉吟，便玉手一翻，手中出现了一枚玉简。
<br/>　　“这是神武录，给你！”木璃玉手一抛，便将玉简抛向了苏莫。
<br/>　　这种玉简，每一名新晋的神武学府弟子都会发放，只不过苏莫不是那种正常招收的弟子，所以便没有发放给他。
<br/>　　随即，木璃转身返回了院落中，大门再次关闭了。
<br/>　　苏莫接过玉简，见对方不愿意与他多聊，便无奈的摇了摇头。
<br/>　　随即，他开始查看玉简中的内容。
<br/>　　这玉简内记录的信息量极大，有神武学府详细的介绍，还有各种能辅助弟子修炼的地方。
<br/>　　苏莫顿时大喜，拿着玉简便返回了自己的阁楼中。
<br/>　　随后，苏莫将玉简中记载的神武录，从头到尾，仔细的查看了一遍。
<br/>　　神武学府，乃是由赤霞洞天、紫薇圣地、九霄圣地、云顶圣门、太史世家、苍云圣宗、等十数个大势力组成。
<br/>　　只招收古灵星河人族顶尖天才，当然，这十几个大势力，也会挑选各自宗门强大的天才进入学府修炼。
<br/>　　所有进入学府的弟子，只能在学府中待三十年的时间，三十年之后必须离开学府，加入这十几个大势力之一。
<br/>　　如若不然，便是背叛神武学府，会遭到十几个大势力的联合绞杀。
<br/>　　可以说，若是背叛了神武学府，整个古灵星河，将再无容身之地。
<br/>　　当然，若是不想加入这些势力，那还有一个选择，就是留在学府担任导师。
<br/>　　不过，导师的名额非常有限，一年才有一个名额。
<br/>　　而神武学府，对于府中弟子的培养也是不遗余力，各种上好的功法武技，应有尽有。
<br/>　　每一名进入学府的外府弟子，都有机会挑选一门帝级上品功法、一门帝级上品武技，一剑帝级上品宝器。
<br/>　　而弟子晋升到武尊境之后，不仅奖励更为丰厚，还可以进入内府修炼，拥有一座自己的山峰。
<br/>　　根据神武录的介绍，神武学府的内府，居然就是这颗陨落星辰的内部，当初奇妙无比。
<br/>　　神武学府的弟子，分为四个级别，从低到高，分别是：外府弟子、内府弟子、圣子、神子。
<br/>　　圣子，是内府中的精英弟子，预示着将来可以冲击武圣之境，是神武学府中重点培养的弟子。
<br/>　　目前，整个学府中，圣子也只有十八人而已，个个都是人中龙凤，万中无一的存在。
<br/>　　而神子，更是不得了，预示着将来有望晋升虚神境，成为浩瀚星空中无敌的存在。
<br/>　　目前，神武学府中，只有一名至高无上的神子。
<br/>　　神子的地位非常高，在学府中超过了绝大部分的导师，也只有火元圣师那种武圣境的导师，能稍压神子一头。
<br/>　　学府中有很多辅助修炼的地方，比如星辰修炼室，奥义圣山等等。
<br/>　　而适合弟子历练的地方，神武录中也记载了不少。
<br/>　　“魔云海、九枯界、万屠山……！”苏莫口中呢喃自语，查看着适合历练的地方。
<br/>　　他的历练和别人不同，别人的历练无外乎是寻宝和探险。
<br/>　　而他这次的历练说白了就是去杀人、杀妖兽，然后吞噬他人的武魂或者兽魂。
<br/>　　毕竟，他这次历练，是要提升武魂的等级。
<br/>　　很快，苏莫便选中了一处适合的地方，魔云海。
<br/>　　据记载，魔云海生存着一支邪恶的种族，魔甲族。
<br/>　　魔甲族是天荒星域众多魔族中的一支，实力较弱，龟缩在魔云海。
<br/>　　而魔云海，自然就成了神武学府外府弟子理想的历练地之一。
<br/>　　只不过，这魔云海极为危险，学府有强自要求，必须三名以上的弟子结伴而行，才能允许去魔云海历练。
<br/>　　这让苏莫有些无奈，他吞噬别人武魂或兽魂，可不能让学府其他弟子知晓。
