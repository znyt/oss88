　　原剧情中，在天帝宝库未开启的时候，黑水玄蛇与黄鸟都守候在死泽，静等宝库开启，而它们的目标，毫无疑问正是其中的神药。
<br/>　　而在这天帝宝库中的这面木台，杯中所盛的，正是传闻中天帝宝库密藏的神药，这木台和木杯，都是与脚下这棵奇树连为一体，以这棵奇树本身万载灵气，来保存神药。
<br/>　　原著中，能让黑水玄蛇这般上古异兽垂涎三尺，甚至不惜与守卫灵药的天敌黄鸟对阵的，也正是这看上去不起眼的一杯灵药而已。
<br/>　　正当两人愣神之际，木杯中浮在水面上的那颗石子忽然腾起一道细细的金光，笔直向上射去，罩在那片光幕之上，顿时将整片光幕染成金色，随即光芒忽盛。
<br/>　　伴随着光芒的闪烁，木杯中的石子，居然漂浮起来，逐渐离开木杯，慕清霄灵气一卷，石子就落入手中，晶莹剔透，看不出何种材质。
<br/>　　可就在落入他手中的瞬间，仿佛是上古神明的法力，石子上忽然冒出比刚才要粗大和耀眼许多的金色光芒，霍然成柱，直接照射在天帝宝库的穹顶上。
<br/>　　宝库之中，突然响起了神秘而悠远的声音，就像是灵山胜境里的神秘焚唱，又像是九幽孤魂的轻声低语。
<br/>　　在两人的注视下，金色耀眼的光芒中，神秘的声音再次响起，好似在吟唱，斗一般大小的金色古拙的文字，在金光的照耀下，不断的凌空出现：
<br/>　　天地不仁，以万物为刍狗！
<br/>　　慕清霄身躯一震，眼中闪过一丝喜悦，道：“天书，终于见到最后一卷了。”
<br/>　　片刻之后，十个笔劲沧桑的金色字体消失，但那神秘的吟唱声却是越来越响亮，越来越盛。
<br/>　　下一秒，那耀眼的金色光柱突然如爆发一般，炙热无比地射向天帝宝库所有的空间，四周的墙壁上，在金光照耀下，一个又一个金色字体凌空出现，笔势苍劲。
<br/>　　字体围绕着慕清霄与陆雪琪，在轰鸣声中，环绕飞翔，两人也是为这夺人心魄的奇异景象所震慑，但很快就被四周神秘的文字所吸引。
<br/>　　陆雪琪粉唇轻启，道：“这就是最后一卷天书吗？”
<br/>　　“没错，这便是最后一卷天书。”
<br/>　　说罢，慕清霄心神停留在其上，脑海中‘轰’的一声作响，墙壁上的金色文字，全部向他狂涌而来，最后一个不剩的莫入他的脑海中。
<br/>　　“叮，收集五卷天书，任务完成，奖励50000掠夺积分，宿主信息数据化中......”
<br/>　　宿主：慕清霄（龙骑士）
<br/>　　资质：未知
<br/>　　血脉：未知
<br/>　　武器：魂剑
<br/>　　战技：暴风剑
<br/>　　修为：元婴五重中期
<br/>　　心法：《御女心经》
<br/>　　伴侣：王语嫣，阿碧，阿朱，波雅.汉库克，欧阳紫嫣，日向雏田，小依，陆雪琪。
<br/>　　掠夺积分：180000
<br/>　　储存空间：400

<br/>　　脑海中响起系统的提示声，慕清霄嘴角牵起一丝笑意，整整六年多的岁月，本位面的任务终于在今天全部完成。
<br/>　　回过神来，慕清霄摊开右手，心脏‘砰砰砰’的跳动起来，只见他掌心握着一颗晶莹剔透的石子，散发着浓郁清香。
<br/>　　“丹药，这就是天帝宝库中的神药，原来传闻都是真的，神药真的存在。”
<br/>　　慕清霄语气有些激动，由不得他不激动，这可是神药啊，他能感受到，这颗小小的丹药中，所包含的能量绝对是恐怖的，若是他现在服用，唯有爆体而亡一个下场。
<br/>　　难怪原著中，黑水玄蛇与黄鸟在天帝宝库未开启的时候，便守在死亡沼泽中，看来它们的目的就是这枚神药，毫无疑问。
<br/>　　原本，他的目标是天帝宝库中的最后一卷天书，却没想到有此机遇，此刻神药就在他的手中，他怎能不激动，仅仅是一颗神药，绝对能让他的修为飞跃！
<br/>　　自己的丈夫获得神药，陆雪琪心中自然无比的开心，她也是修士，自然知道神药代表什么。
<br/>　　慕清霄取出储存空间中的容器，小心翼翼的将神药封印在容器中，其中含有的能量，不是他现在能够炼化的，但他不也着急。
<br/>　　当封存好神药，慕清霄转过头，露出一丝微笑。
<br/>　　“雪琪，接下来我要开始参悟天书至理，木杯中的透明液体是液化的灵气，你尝试着炼化，记住一次不要太多，否则肉身无法全部炼化，只会适得其反。”
<br/>　　闻言，陆雪琪美眸中尽是喜悦，她的修为停留在筑基九重巅峰已经有半个多月，早已有突破的趋势，现在有灵液，不仅能够让她突破金丹，她的修为也会暴涨。
<br/>　　慕清霄就地盘坐而下，心神守一，掌心向天，心神停留在墙壁上，开始参悟最后一卷天书至理。
<br/>　　而另一边，陆雪琪也照着慕清霄的话，随手一卷，五分之一的灵液就顺着粉唇莫入口中，灵液入口瞬间流淌进四肢百骸。
<br/>　　其中包含的能量极其恐怖，陆雪琪不敢怠慢，快速运转心法，开始炼化灵液，随着时间的推移，一股恐怖的气息自她体内向四周扩散出去。
<br/>　　金丹成！
<br/>　　“这就是太清境，真是恐怖的力量。”陆雪琪睁开美眸，轻声细语，温柔的看了一眼慕清霄，继续炼化体内浓郁的灵液。
<br/>　　而另外一边，慕清霄则仔细的参悟着最后一卷天书，气息也隐隐波动起来。
<br/>　　天地自然，万法不离其宗。
<br/>　　星辰年轮，引灵有迹可循。

<br/>　　慕清霄身躯一震，一股无形的波动，在空中荡起一阵涟漪，睁开一双沧桑眼眸，道：“天书果然不是本位面的东西，而是来自于一位天外强者！”
