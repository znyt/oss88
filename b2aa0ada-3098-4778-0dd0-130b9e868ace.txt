　　这一战结束，楚风眠已经击败冷面魔君，挺进了四强。
<br/>　　只要在经历两场战斗，便是足以夺得这届武斗大会的冠军。
<br/>　　下一场，夜天君上场，他对面的，不过只是一个武道世家的弟子。
<br/>　　几乎是被夜天君，轻而易举的给击败。
<br/>　　夜天君也是跟楚风眠一样，挺进了四强之中。
<br/>　　四强名额两个，如今已经是定了下来。
<br/>　　“下一战，葬天对寒月漓。”
<br/>　　看台之上，通天阁老者开口，宣布了下一场战斗。
<br/>　　葬天，对寒月漓。
<br/>　　“居然是碰到他了。”
<br/>　　楚风眠看着一眼擂台，只见葬天的身影，已经是一跃跳上擂台。
<br/>　　在这葬天站在擂台上的一刻，他的目光，似乎是故意看了楚风眠几眼，从这目光之中，楚风眠甚至是看出了不少仇恨，以及怨恨。
<br/>　　这是一种充满仇恨的挑衅眼神。
<br/>　　“小心一点。”
<br/>　　楚风眠看到这目光，暗自对着旁边的寒月漓开口道。
<br/>　　“嗯。”
<br/>　　寒月漓点了点头，她的身形一动，便是直接一跃跳上了擂台之上。
<br/>　　寒月漓的实力，还没有真正展现过，所以对于这一战，在场众人也是无比期待的。
<br/>　　两匹这一届武斗大会的黑马，终于碰撞起来。
<br/>　　寒月漓一袭白衣，站在擂台之上，而她对面的葬天，正是一身漆黑。
<br/>　　“轰！”
<br/>　　几乎是没有任何的言语，葬天猛然出手了。
<br/>　　在看到寒月漓的一刻，葬天的眼神中，都露出了几分狰狞的眼色。
<br/>　　没有丝毫的犹豫，突然出手，他的一拳已经是轰击过去。
<br/>　　这葬天一路晋级，几乎都是以纯粹的力量便是将对手击败。
<br/>　　面对着他的力量，寒月漓也没有丝毫放松，她手一动，在她的面前，便是化为了一道冰镜。
<br/>　　正是跟葬天的力量，碰撞了起来。
<br/>　　“碰！碰！碰！”
<br/>　　眨眼的瞬间，几乎是无数声音，同时响起，葬天足足是连续数十道拳风，向着寒月漓轰击过去。
<br/>　　但是寒月漓却是轻而易举的将这些拳风，全部抵挡住。
<br/>　　几乎一个瞬间的交手，将两人的真正实力，都给爆发了出来。
<br/>　　“好强的力量，果然当初寒雪阁的第一人，先天神体的天才，没想到楚风眠那个蠢货，居然是没有将你身上的先天玄寒之力给夺走。”
<br/>　　葬天打量着寒月漓，突然阴冷的开口道。
<br/>　　“既然如此，那我就先将你的力量夺走，再去对付楚风眠那个小子！”
<br/>　　这葬天的声音，嘶哑无比，令人听见，都不由的想要捂住耳朵。
<br/>　　“你到底是谁？”
<br/>　　寒月漓听着葬天，喊出她的名字，尤其是提到寒雪阁的一刻，寒月漓也明白过来。
<br/>　　眼前这葬天，的确也是来自于武胜国度的人物，而且听起来，还跟楚风眠有着极大的仇怨。
<br/>　　葬天冷哼一声道。
<br/>　　“我是谁？对一个将死之人，没有任何解释的必要！”
<br/>　　“不愿意说？那我就将你镇压，看看你到底是谁！”
<br/>　　寒月漓听到葬天的话，脸色平静，不过隐约之间，她的脸上都是露出了几分怒意。
<br/>　　这葬天的杀意，几乎是毫不掩饰的展现出来。
<br/>　　对待敌人，寒月漓也没有这任何手下留情的想法。
<br/>　　“冰绝！”
<br/>　　寒月漓大手一挥，一股灵力在她的手中便是化为了一把冰剑。
<br/>　　突然之间便是挥动，向着那葬天，斩杀而去。
<br/>　　虽是冰剑，却是锋利到了极点。
<br/>　　冰剑一出，整个擂台上都是被一股寒意包围。
<br/>　　“破！”
<br/>　　一层层寒冰，随着寒月漓这一剑斩出，同时向着那葬天斩杀过去。
<br/>　　这一层层寒冰，都是锋利无比，寻常武者若是触碰到，都会被直接斩裂。
<br/>　　这一剑中，已经是蕴含杀意，寒月漓毫不留手，向着葬天斩杀，其中的心思，也同样是想要逼出这葬天的真正实力。
<br/>　　葬天一直以来，都在隐藏着自己的实力，一旦是他暴露实力，他的身份也就很容易猜到。
<br/>　　无论如何，这葬天，绝对是敌人，不能斩杀他，也要至少逼出他的身份。
<br/>　　“不知死活！你想要逼出我的身份，那好！”
<br/>　　葬天突然冷冷一笑。
<br/>　　在他的身边，一股股的魔意，爆发出来。
<br/>　　这魔意出现的一刻，席卷整个擂台，无数向着他斩杀过去的剑锋，都被直接吞噬。
<br/>　　“魔意？”
<br/>　　“此人的地魔门的人？”
<br/>　　无数目光看向葬天身上爆发出来的魔意，都是一愣。
<br/>　　这魔意，也就意味这，葬天的地魔门的人？
<br/>　　不过不对。
<br/>　　地魔门的人，似乎都不认识这葬天，那这葬天，到底是什么来历？
<br/>　　“嗯？”
<br/>　　寒月漓的目光，都是微微变化这，显然这魔意的突然出现，似乎是在她的意料之外。
<br/>　　本里寒月漓是打算先将葬天的真正力量逼出来，好知晓他的真正身份的。
<br/>　　但是现在这魔意一出现，更是让寒月漓有些疑惑，这一股力量，似乎并非是她之前认识的任何人。
<br/>　　可是葬天可以认出她的来历，她的身份，显然葬天肯定是武胜国度的人。
<br/>　　“魔意，这是？”
<br/>　　擂台下，楚风眠死死的盯着葬天，看到这魔意的一刻，他的脑海中突然显现出来一个人影。
<br/>　　只是他还没有确认的一刻。
<br/>　　在这擂台上，却是突然发生了逆转的变化。
<br/>　　只见寒月漓大手一挥，手中冰剑再一次斩了过去，在她面前的魔意，都被纷纷斩碎。
<br/>　　但是这魔意破碎，葬天的脸上却是没有出现任何的慌张，反而是露出了几分笑意。
<br/>　　这笑意，被楚风眠看在眼中。
<br/>　　“不好，这下糟了！”
<br/>　　楚风眠猛然站了起来。
<br/>　　“轰！”
<br/>　　寒月漓的身形，站在原地，突然之间，她的身躯震荡，双眼之中都露出了几分迷茫。
<br/>　　整个人，似乎是站在原地，彻底呆滞了起来。
<br/>　　“心魔？这是大心魔决！”
<br/>　　楚风眠惊喝一声道。
<br/>　　这葬天打出的魔意，并非是寻常的魔意，而是心魔。
<br/>　　这心魔，随着寒月漓的斩杀，居然是直接钻入到了寒月漓的身躯之中去，引动了寒月漓身上的心魔，令她一下子陷入到了短暂的呆滞中。
<br/>　　就在这瞬间，葬天的身形动了，一掌之间，已经是向着寒月漓轰击过去。
<br/>　　正是打算趁着寒月漓对付心魔的瞬间，来彻底轰杀寒月漓。
<br/>　　局势危机到了极点。
