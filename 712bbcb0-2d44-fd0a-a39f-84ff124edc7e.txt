<br/>　　大长老等龙族之人，心中暗暗的松了口气，他们还担心苏莫遭遇不测，没想到顺利的脱身了。
<br/>　　嗖！
<br/>　　神昆小世界的入口，一直没有关闭，所有被淘汰的弟子，之前都从入口处出来了，这时，身姿挺拔的龙腾，身形一闪，飞出了神昆小世界。

<br/>　　“不知！”龙腾微微的摇了摇头。
<br/>　　“什么？”
<br/>　　魔长烽闻言，顿时面上露出怒色，大声质问道：“你之前去追逐太子，现在你出来了，太子没有出现，你怎么会不知晓？”
<br/>　　他可不信龙腾不知，毕竟，第八城都破了，古魔太子被淘汰了，那么肯定会出来。
<br/>　　“难道我不知晓，还要骗你说我知晓？”龙腾面色平静的反问道。
<br/>　　“你……！”魔长烽顿时被龙腾怼的语塞，他深吸了口气，暂时没有动怒。
<br/>　　毕竟，古魔太子没有出来，并不代表遇到了什么危险，有可能是在虚空深处迷失了方向。
<br/>　　待千选战结束，他去将其找回来便可。
<br/>　　其它各大势力之人，也都没有多想，毕竟，正常情况下来说，无论是龙腾还是苏莫，都不太可能伤了古魔太子。
<br/>　　……
<br/>　　神昆小世界之中。
<br/>　　“好，既然如此，那我们就商议一番，如何击败第九城！”涂赢见苏莫等人都同意他的话，顿时朗声说道。
<br/>　　“陵公子精通阵法，这第九城之中，绝对蕴含无尽的危险，想要攻破第九城，不容易！”阎魔鬼圣说道。
<br/>　　“那我们怎么办？”公孙长绝问道，这第九城之人躲在城中不出，他们根本没有办法。
<br/>　　“我觉得，我们先试一试能不能从外部摧毁阵法！”阎魔鬼圣说道。
<br/>　　“那我们就试一试！”苏莫道，无论成功与否，都可以试一试，毕竟，不到万不得已，不能贸然进入。

<br/>　　一众近两百人，开始疯狂的攻击第九城，各种强大的攻击，铺天盖地的向第九城之中袭杀而去。

<br/>　　“根本毫无破绽！”
<br/>　　“简直让人无法下手！”
<br/>　　一众人个个面色难看，这第九城之中，到底有什么阵法，如此的诡异莫测。

<br/>　　“我们这么多人，难道还破不了他的阵法？”
<br/>　　“不太妥当，一旦无法破阵，我们就要全军覆没！”
