        但最终，傅昼景还是保持着一份理智，没有做对不起童筱颖的事情。
<br/>
<br/>        那么……接下来，他该怎么去面对童筱颖？！
<br/>
<br/>        当傅昼景回到家的时候，已经是凌晨了。
<br/>
<br/>        家中一片漆黑，他关上门开灯后，并未看到童筱颖的身影。而就在转身之际，刚巧碰到童筱颖打开家门从外归来。
<br/>
<br/>        她面色红润，大汗淋漓。
<br/>
<br/>        傅昼景看到她这个样子，脸色瞬间阴沉了几分。
<br/>
<br/>        “你去哪儿呢？”
<br/>
<br/>        “你回来了！”
<br/>
<br/>        两人几乎异口同声。
<br/>
<br/>        童筱颖理了理乱糟糟的长发，抹了把额头上的汗水，咧嘴一笑，撒了个谎替自己打掩护：“我、我睡不着，跑步去了。”
<br/>
<br/>        “哦。”傅昼景的语气忽然间变得格外冷硬，心情也越来越糟糕，甚至脑子里还在怀疑筱颖是不是刚卖完身回来。
<br/>
<br/>        “你饿不饿？要不我去给你做点……”
<br/>
<br/>        “我累了，想睡觉。”不等童筱颖说完，傅昼景若有所思地打断了她的话，然后自顾自地往沙发上一趟，并翻了个身，背对着童筱颖睡了过去。
<br/>
<br/>        童筱颖愣了愣，脸上的笑容渐渐收敛，只好聋拉着脑袋，默默地去洗澡。
<br/>
<br/>        她不明白，他今晚为什么要睡沙发？
<br/>
<br/>        可是他已经睡了，她不好再打扰他。
<br/>
<br/>        或许，他是真的工作太累了。
<br/>
<br/>        第二天，傅昼景早上起来后便看到童筱颖已经将早餐端上了桌。
<br/>
<br/>        “早上好！”童筱颖正在盛粥，看到傅昼景已经洗漱完并穿戴整齐好从卫生间内出来，不禁微笑着问了声早。
<br/>
<br/>        傅昼景却只是干笑着扯了扯嘴角：“早上好。”
<br/>
<br/>        随之，他来到餐桌前坐下。
<br/>
<br/>        “以后，你多睡一会儿。不要这么早起来给我弄早餐。”他接着说道，像是在关心她。
<br/>
<br/>        童筱颖欣慰地一笑：“我没有睡懒觉的习惯。”
<br/>
<br/>        是啊！
<br/>
<br/>        她确实没有睡懒觉的习惯。
<br/>
<br/>        哪怕很晚很晚才睡觉，早上她仍旧会在固定的时间点起床，不过，也有特殊的情况才会起得晚。
<br/>
<br/>        “随你。”傅昼景淡淡地应了声，端起粥喝了一口，“味道好像淡了点。”
<br/>
<br/>        童筱颖随即自己也端起粥碗尝了尝，不以为然地说道：“不会啊！我觉得甜度刚刚好！”
<br/>
<br/>        “对了，这是我们的生活费。”傅昼景说着，从衣服里掏出一个皮夹，然后抽了几张钞票出来，放在了餐桌上。
<br/>
<br/>        “不用，我身上还有钱用。”童筱颖微笑着说道，顿了顿后，又补充了一句，“上次你给我的，我还未用完。”
<br/>
<br/>        傅昼景淡定地收回钱，并未多说什么。
<br/>
<br/>        吃完早餐，他准备去上班的时候，童筱颖连忙上前帮他提公文包。
<br/>
<br/>        忽然间，童筱颖又想起一件事情，便对傅昼景说道：“可不可以，帮我弄一份实习报告回来？我学校那边还没有毕业，过完这个月。我得回学校去补学时了，因为期间是请了假出来实习的，所以得弄一份盖了章的实习报告回去！”
<br/>
<br/>        “意思是，你有很长一段时间，都不会在家？”傅昼景下意识地反问。
<br/>
<br/>        童筱颖点了点头，接着说：“我每个月会回来。”
<br/>
<br/>        “哦，好。实习报告，晚上我带回来。”傅昼景强颜笑了笑，然后出了门。
<br/>
<br/>        待傅昼景离开家后，童筱颖发现，家里又只剩下她一个人了。
<br/>
<br/>        一如既往，一个人在家的话，童筱颖便会从整理内务开始。
<br/>
<br/>        整理完内务后，她闲着无聊，又会拿出笔和纸，趴在书桌前设计一些比较有主题特色的室内设计。
<br/>
<br/>        虽然她现在，在“夜惑”驻唱的收入还不错，但是她没有忘记自己的所学专业。
<br/>
<br/>        她从书桌前起身，趿着拖鞋，走到门前看了一眼猫眼。
<br/>
<br/>        门外是一个气质高贵的女人，披散着长卷发，浓妆艳抹，红唇诱人。
<br/>
<br/>        童筱颖确定对方不是坏人后，将门打开了。
<br/>
<br/>        “请问，您找哪位？”
<br/>
<br/>        童筱颖温和地问道。
<br/>
<br/>        女人上上下下地将童筱颖打量了一番后，微笑着说：“你好，我是傅昼景的新秘书，是他让我来替他取文件的。”
<br/>
<br/>        “放哪了？我去给您取过来。”童筱颖微微一笑。
<br/>
<br/>        女人不依不饶，咧嘴一笑：“放文件的地方得保密噢！”
<br/>
<br/>        童筱颖无奈，只好让女人进门。
<br/>
<br/>        当女人进门后，童筱颖闻到了一股浓浓的香水味，味道虽然浓，但是这种香水却很好闻。
<br/>
<br/>        童筱颖不知道这是什么牌子的香水，总而言之，这味道闻着很舒服。
<br/>
<br/>        应该是高档香水吧！
<br/>
<br/>        童筱颖一下子就记住了这个味道。
<br/>
<br/>        不用童筱颖提示，季思妍四下打量了一番这小小几平方米的出租屋内陈设后，便直接打开了书桌旁的衣柜翻找起来。
<br/>
<br/>        童筱颖也不知道这个女人在她和傅昼景的衣柜里找什么，季思妍背对着童筱颖，偷偷从包包里掏出一个文件袋拿在手里，然后合上了柜门转过身来。
<br/>
<br/>        “拿到了，谢谢。”季思妍莞尔一笑，摇了摇手里的文件袋。
<br/>
<br/>        童筱颖只是微微一笑，并未对她的举动多疑。
<br/>
<br/>        临走前，季思妍忽然转过身来，问道：“你是昼景请回来的保姆？”
<br/>
<br/>        童筱颖怔愣了一下，刚想说不是，那女人咧嘴一笑就这么走出去了。
<br/>
<br/>        季思妍走出公寓后，不由地笑出了声。
<br/>
<br/>        阿景找的老婆，压根就不及她一半。
<br/>
<br/>        还以为对方会非常优秀，原来只不过是个“城乡结合部”来的“保姆”。
<br/>
<br/>        闲来无聊，童筱颖给发小叶美佳发了条信息。
<br/>
<br/>        美佳，有一种香水闻起来非常好闻，哪怕是从来不喜欢香水的人，都会觉得那味道闻着很舒服，那是什么牌子的香水？——筱颖。
<br/>
<br/>        迪奥？香奈儿？兰蔻？——美佳。
<br/>
<br/>        看到叶美佳发过来的这些香水名字，童筱颖一脸懵，完全不了解。
<br/>
<br/>        于是，童筱颖决定去一趟高档商场，看看这些香水。
<br/>
<br/>        换了身衣服，提起包包，装好钥匙和手机，童筱颖坐公交车去了市中心的高档百货。
<br/>
<br/>        商场的一楼，往往都是珠宝首饰，化妆品那些东西的专柜。
<br/>
<br/>        童筱颖很快便找到了香水专柜，然后低头认真辨识玻璃橱柜里那些一小小瓶就要价上千的香水。
<br/>
<br/>        “大少爷，这个商场，是我们傅氏集团旗下最高档的商场之一。”
<br/>
<br/>        刚接手傅氏集团，傅夜沉便被爷爷派到自己身边的助理郑忠威带着“观光”傅氏集团旗下的产业。
<br/>
<br/>        随从毕恭毕敬地介绍，傅夜沉的眼睛却在走马观花。
<br/>
<br/>        幸好，弃军从商，他的适应能力还比较快。
<br/>
<br/>        视野扫过之际，傅夜沉将目光的焦点停留在了一个香水专柜前。
<br/>
<br/>        只见童筱颖微微弯着腰，像是非常感兴趣地在研究玻璃橱柜里的那些香水一样。
<br/>
<br/>        “这位女士，我给您推荐这款香水吧！这款香水是兰蔻……”站在专柜里的女服务员开始滔滔不绝地做推销。
<br/>
<br/>        她倒是说得口干舌燥，也不见童筱颖有要买的冲动。
<br/>
<br/>        因为，在童筱颖看来。
<br/>
<br/>        太、贵、了！
