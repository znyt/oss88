        童筱颖走到了隔壁的门边，看了一下屋内的场景。
<br/>
<br/>        是刚刚那个给那名叫“季少”的贵公子舔鞋的女人。
<br/>
<br/>        而另一个男人体型臃肿，坐在办公桌前，翘着二郎腿，像是这女人的上司。
<br/>
<br/>        童筱颖看到那女人哭得一脸委屈，刚想进去替那女人说句话的时候，脑海里突然想起了晚霞先前警告的那些话。
<br/>
<br/>        是啊！
<br/>
<br/>        她是谁？在这里也只不过和那女人一样，都是最底层的人。她有什么能力能帮到那个女人？
<br/>
<br/>        童筱颖收了收自己那爱莫能助的心，黯然神伤地转过身去，回到了自己的休息室里。
<br/>
<br/>        她今晚驻唱所穿的服装，唐倩兮已经将留言纸条贴在了梳妆镜上，她只要照做就行了。
<br/>
<br/>        童筱颖关了门，拿起唐倩兮留言条里所指的那件白色广袖留仙裙给换上后，见时间差不多了，便出了门打算先去后台等候。
<br/>
<br/>        “夜惑”里的古风还是比较浓厚，很多在这里工作的女人都穿着像她这样的古装，行走在长廊上。
<br/>
<br/>        在这里工作的女人，有个不成文的规定。
<br/>
<br/>        只要是戴着假面眼镜的女人，都是卖艺不卖身的，其他男人不可窥觊，连顾客也不能打这些戴着假面眼镜的女人的主意。
<br/>
<br/>        上次，就因为她没戴假面眼镜，才会被一个名叫“郭裕”的男人差点占了便宜去。
<br/>
<br/>        童筱颖记得那次，自己整个就是浑浑噩噩地度过的。幸好，是傅昼景及时救了她，恐怕她早就失身于那个不怀好意的郭裕了。
<br/>
<br/>        而其实，那次救了她的人是傅夜沉。
<br/>
<br/>        童筱颖却一直以为救了自己的人是傅昼景。
<br/>
<br/>        她忽然发现，昼景真的是她的救星，一次又一次替她解围。
<br/>
<br/>        这个时候，她心里想起傅昼景，莫名其妙地有种负罪感。
<br/>
<br/>        如果他知道她在这里卖唱，一定会很生气吧！
<br/>
<br/>        童筱颖在后台坐了一会儿，待到有人喊她上台驻唱时，她才心事重重地走上台去。
<br/>
<br/>        季运恒一袭暗红色西装，配着一双暗红色皮鞋，翘着二郎腿，坐在真皮沙发的雅座里。
<br/>
<br/>        他左拥右抱的美女傍身，却仍旧是一脸“吃着碗里瞧着锅里”的心态，遥望着坐在舞台上卖唱的那抹白色飘影。
<br/>
<br/>        季运恒情不自禁地咧嘴，坏坏一笑：“今晚在台上唱歌的那个女人叫什么？”
<br/>
<br/>        坐在一旁的好友男，一边摸着陪酒女的胸部，一边色眯眯地看向台上的童筱颖，笑得一脸淫相：“她啊！叫小谢！好像才来没几天吧！不过，她的歌声不错。”
<br/>
<br/>        刚刚一个女服务员弄脏了他的皮鞋，倒了他的胃口，但是此刻，台上那个穿着白色广袖留仙裙，戴着白色羽绒假面眼镜，歌声优美的女人引起了他的兴趣。
<br/>
<br/>        他有一段时间没来“夜惑”玩了，没想到今天一来，就被一个女服务员扫了兴致。
<br/>
<br/>        虽然，这里的主管跟他赔礼道歉了，但他心里总是有种情愫堵得慌，让他十分不爽。
<br/>
<br/>        “服务员，过来，把这个电子手环拿上去打赏给小谢。”季运恒掏出手机，在茶几的打赏框上扫了码，拿起框内的手环，点了小谢的名字以及打赏金额，便让男服务员给台上的小谢送去。
<br/>
<br/>        “夜惑”里，顾客给卖艺的人打赏，都得通过打赏手环点名点金额后，再由男服务员送到卖艺人的手中。
<br/>
<br/>        童筱颖在这里驻唱的这几晚，很少收到顾客的打赏，而今晚，她竟然收到了一个价值一万的打赏手环。
<br/>
<br/>        而打赏她的人的署名是“季少”，也就是欺负那个女服务员的男人……
<br/>
<br/>        男服务员送上来的手环，她不得不戴在手腕上，也不得不当众道谢。
<br/>
<br/>        童筱颖唱完一首歌后，强颜欢笑地说道：“谢谢刚刚给我打赏的季少。”
<br/>
<br/>        她的话刚落，台下就有人在吹口哨，像是在起哄，又像是在暗示着别的什么她不知道的意思。
<br/>
<br/>        童筱颖在台上把五首歌都唱完后，就被男服务员请到了刚刚给她打赏的那位“季少”的面前。
<br/>
<br/>        当男服务员带着童筱颖来到季运恒的面前时，季运恒看着童筱颖的模样，瞬间惊呆了。
<br/>
<br/>        这个女人虽然戴着白色羽绒假面眼镜，但她却被身上那袭白色广袖流仙裙穿出了不食人间烟火的气质。
<br/>
<br/>        “季少，你好。”童筱颖毕恭毕敬地颔首。
<br/>
<br/>        在这里正式驻唱的第一天，唐倩兮就有跟她交代，“顾客是上帝”，必须对来这里消费的所有顾客毕恭毕敬。
<br/>
<br/>        本来，她唱完五首歌后，是要直接下班回家的。
<br/>
<br/>        但是晚霞姐说，季少既然打赏了她一万块钱，又点名要见她，她就必须得去应酬。
<br/>
<br/>        可童筱颖并不想为了这一万块钱，去耽误自己回家的时间。
<br/>
<br/>        所以，她是被晚霞姐强行让男服务员带到这个季少面前来的。
<br/>
<br/>        季运恒在看到童筱颖后，立即遣散了身边的莺莺燕燕，腾出自己身旁的位置，伸手召唤童筱颖过来坐下。
<br/>
<br/>        “小谢，过来陪我坐一会儿说说话。”季运恒嘴角微扬，目光直勾勾地盯着童筱颖，非常客气地说道。
<br/>
<br/>        童筱颖却站在原地不动，而是毕恭毕敬地微微欠身，委婉地拒绝道：“对不起，季少。我的下班时间到了，我该回家了。今晚谢谢你的打赏。愿季少在这里玩得开心，我先告辞了。”
<br/>
<br/>        语毕，她头也不回地离开了。
<br/>
<br/>        童筱颖只是不想惹是生非，不想给这里的客人留下任何不好的印象，同时更不想间接地当陪聊或是陪酒的。
<br/>
<br/>        明明她有意避讳，却无心地在季运恒的眼里留下了一抹清新又高冷的背影。
<br/>
<br/>        她越是这样桀骜不驯，他便对她越感兴趣。
<br/>
<br/>        季运恒忽然间想要征服这个名叫“小谢”的女人。
<br/>
<br/>        他给她打赏了一万块钱，她都不放眼里，看样子，是他打赏的数目不够多。
<br/>
<br/>        很好，那明天晚上，他就让她见识一下，他这位季大少爷的财力。
<br/>
<br/>        季运恒换女友的速度比换衣服还快，每个女人，他都只玩了那么一两个月就腻了。
<br/>
<br/>        现在，难得有个女人能入得了他的坏心思，他当然要好好地玩一玩，而且还要玩得尽兴。
