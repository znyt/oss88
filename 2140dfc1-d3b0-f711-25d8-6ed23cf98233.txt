　　现在楚风眠被五行天的小天主，设下的天价悬赏，简直就是一座移动的宝藏。
<br/>　　谁可以杀了楚风眠，得到的好处不亚于一座大型宝藏所得到的好处。
<br/>　　这五行天圣子，也曾经听闻过楚风眠的一些事，知晓楚风眠手段众多，所以不跟楚风眠废话，直接出手。
<br/>　　一出手便是绝对的杀招，要将楚风眠捏死在他的手心之中。
<br/>　　一道手掌，铺天盖地之间对准楚风眠笼罩而来，瞬息之间轰击在了楚风眠的头顶之上。
<br/>　　“这种实力也想要杀我？”
<br/>　　楚风眠嗤笑一声，身形未动，从他背后的天涯剑匣之中，两道剑光飚射而出，冲天而起，斩在了这一道手掌之上。
<br/>　　啊！
<br/>　　一声惨叫响起。
<br/>　　那五灵宗圣子的手掌被顿时斩落而下。
<br/>　　“这么强？”
<br/>　　“八劫古帝，此子已经是达到了八劫古帝？”
<br/>　　那另外两名五灵宗圣子盯着楚风眠出手，脸色一变，对视一眼，不需要任何的交谈，刹那之间一起出手了。
<br/>　　轰！
<br/>　　漫天灵力，向着楚风眠轰击而来，三道联手出手，凝聚灵力化为了一道巨大无比的大阵，蕴含毁灭之力，冲着楚风眠便是轰击而来。
<br/>　　楚风眠脸色平静无比，心神一动，又是数道剑光冲天而起，打入到了这大阵之内。
<br/>　　一道道剑光的锋芒，肆无忌惮的毁灭这大阵的阵眼，将这大阵打的破碎。
<br/>　　大阵破碎的余波，一下子将三名五行天圣子都给逼退了出去，余波的力量，更是震伤了他们，一个个口中吐出了几口精血。
<br/>　　“此子实力太可怕了！”
<br/>　　“走，此人不是我们可以对付的了了，去将此人的消息告诉小天主！”
<br/>　　三人被逼退出去，心中都是清楚无比，以他们的实力根本难以对付楚风眠。
<br/>　　在被逼退出去的瞬间，三人便是身形调转，化为了一道遁光直接向着远方逃去。
<br/>　　“不用逃了！都死在这里吧！”
<br/>　　楚风眠眼神冰冷无比，还没有谁可以在他的手中可以逃走。
<br/>　　数道剑光骤然之间飞了出去，风驰电掣之间追上了是三人，剑光闪烁，这三人的身躯都被剑气所贯穿。
<br/>　　吞天秘术随即打出，一道漩涡将他们三人的精血全部吞噬，化为了楚风眠的力量。
<br/>　　同时这三人的空戒，也一下子落入到了楚风眠的手心之中，这三人的实力不错，在这赤红大陆内，有着极大的收获。
<br/>　　楚风眠打开空戒，一眼望过去古圣药，都足足有着数十株之多，其中上古级别的古圣药都有足足七株。
<br/>　　其中品质最好的那一株，已经是接近到了远古级别了。
<br/>　　这些古圣药之中，其中三株都是正适合楚风眠，足以是让楚风眠巩固境界之用。
<br/>　　楚风眠没有任何的犹豫，一把是将这些古圣药从中拿了出来，一下子吞服了下去。
<br/>　　这些古圣药，虽然如果炼制成丹的话，效果要好得多。
<br/>　　但是炼制成丹，凭借这么几株古圣药还是远远不够，还需要许多其他的圣药，作为辅佐之用。
<br/>　　可这些辅佐之药，许多也都古圣药级别的，如今这个时代有没有还不一定，想要得到更是谈何容易。
<br/>　　所以楚风眠便是没有打算炼制成丹的想法，直接吞噬，来提升他现在的实力。
<br/>　　在这赤红大陆之中，只有实力才是最为根本的一切，拥有实力，绝不会缺少宝藏。
<br/>　　更何况现在三大势力已经是开始来针对楚风眠，楚风眠必须要抓紧一切时间去提升实力。
<br/>　　“大型宝藏，天人遗迹，看来应该是类似于九玄仙尊洞府一类的宝藏。”
<br/>　　楚风眠记得三人交谈之中的那些大型宝藏，楚风眠也将这三名五行天圣子的记忆给抽取读取了下来。
<br/>　　据说这五行天的几位太上长老，已经是得到了其中两座大型宝藏。
<br/>　　一是乾坤国主，二是雾岛老人遗留下来的洞府。
<br/>　　这两尊存在，都是九域历史之中赫赫有名的强者，皆是天人境的人物。
<br/>　　九域历史之中的天人境强者，也并不少，如今这赤红大陆开启的大型宝藏，远远不止这两处。
<br/>　　如果有机会楚风眠也得到其中一座大型宝藏的话，那绝对是一大收获，让楚风眠的实力足以暴涨，超级机缘。
<br/>　　像是楚风眠得到了九玄仙尊洞府，便是就得到了巨大无比的好处，得到了九玄仙尊。
<br/>　　还有那当初楚风眠曾经得到过的颜古龙帝的洞府，也得到了祖龙至尊剑这样的灵器。
<br/>　　不过想要得到大型宝藏，可并不容易，前去争夺的，许多都是九劫古帝级别的强者。
<br/>　　归墟天可是出动了不少九劫古帝级别的太上长老，才抢夺了其中的两座，足以看出想要得到的艰难。
<br/>　　楚风眠最多也就能试试，也没有完全的把握。
<br/>　　“先去猎杀其他三大势力的古帝，等到这附近出现大型宝藏，再去碰碰运气！”
<br/>　　赤红大陆的宝藏，都是突然间开启的，如果没有开启，在强大的武者也没有资格进入其中。
<br/>　　就像是那天元商会的宝藏，必须是要得到那七彩晶石的人，才有资格踏入其中。
<br/>　　当然有资格踏入其中，也并不代表可以得到其中的宝藏。
<br/>　　只有拥有足够的实力，才有资格得到其中的宝藏。
<br/>　　楚风眠打开窥神境，又是看出了其中几名古帝的踪影，随即身形一闪，飞了过去。
<br/>　　七劫古帝，八劫古帝，在楚风眠的手中就连逃走的可能性都没有。
<br/>　　短短的半个时辰时间，如今在这方圆百里，三大势力的古帝，都被楚风眠斩杀一空。
<br/>　　大量的精血也都被楚风眠吞噬，化为了楚风眠的力量，在巩固这楚风眠的境界。
<br/>　　不过最大的收获，还是这些古帝空戒之中得到的宝物。
<br/>　　这些古帝，许多都在赤红大陆之中收敛了大量的宝物，现在都落入到了楚风眠的手中。
<br/>　　这些宝物，拿出去，绝对足以建立起一方庞然大物了，许多宝物就算是九劫古帝见到，都会忍不住的出手去抢夺。
<br/>　　楚风眠看了一眼窥神镜上，看到没有了任何的目标。
<br/>　　“该去下一个地方了。”
<br/>　　楚风眠刚刚准备催动遁光，突然间地动山摇。
