        &lt;h3&gt;第三卷第二百三十三章  脱困&lt;/h3&gt;
<br/>
<br/>        寒伯安在去机场的路上乘坐的车子被后面超车，在超车的同时又被追尾，几辆车撞在一起发生了严重的交通事故，路上堵得一塌糊涂，可以说是寸步难行了，眼看着飞机要起飞了，寒伯安叹了口气。
<br/>
<br/>        寒伯安坐的车被追尾了，车后面已经严重变形，而超他的那辆车似乎是故意的将他们截下来。
<br/>
<br/>        他还是低估了对手。
<br/>
<br/>        这些事件综合起来想一想就是一个阴谋，到底是谁在布这么大的局？将落家和他们寒家一起拖入局中？
<br/>
<br/>        真的是甸城谷家吗？可是甸城谷家和他们并没有太多接触，偶尔会有来往，也仅仅是工作层面的事情。
<br/>
<br/>        还有，安男会有危险吗？
<br/>
<br/>        苓希已经成了谷家的人，那么这么多年苓希会干一些什么？他们有很多年没有见面了，他们会怎么样对待安男？
<br/>
<br/>        这几天发生的事情让他措手不及，商场上的事情他能知道对方是谁，可是现在他连对方是谁都不知道。
<br/>
<br/>        他无奈的给李长卿拨打了电话：“长卿，来机场接我。”
<br/>
<br/>        “怎么了？你怎么还没有走？”
<br/>
<br/>        李长卿叫来王璔一起驱车赶往机场。
<br/>
<br/>        现在不知道国内安男的情况，势必寒伯安会受到牵制，公司目前看着一片安静，谁也不知道会发生什么。
<br/>
<br/>        李长卿心里也是很焦虑，他拿起电话想给他的女朋友打电话，想让他女朋友去看看安男，可是想到没有人引见，宫阳连寒家的门都进不去。
<br/>
<br/>        李长卿看着寒伯安，他们现在是在国外，鞭长莫及，但是寒伯安似乎又不愿意惊动别的人，打电话回公司安排人将安男接走也是可以的。
<br/>
<br/>        “为什么不报警或者是让人过去接安男小姐姐？”
<br/>
<br/>        “安男目前没有受到任何威胁，况且苓希是苓家的人，苓家和寒家的关系谁会相信苓希会对安男怎么样？我虽然坐上了这个位置，但是大部分人还是老爷子的手下。我们这边如果有问题，家里恐怕也会闻风而动。”
<br/>
<br/>        “那现在怎么办？”
<br/>
<br/>        “明天只能看情况了。”苍岚雪可是说了要再来拜访他的，似乎是不达目的不罢休，他也只能祈祷安男能自己保护好自己。
<br/>
<br/>        “你找回来的那个东西你检测了没有？”
<br/>
<br/>        “已经送去了，回去我们看看结果。”
<br/>
<br/>        “不用了，我弟弟他们没有任何事情，但是这件事情还是要保密。”
<br/>
<br/>        寒伯安此次出来只带了李长卿一个人，李长卿当兵的出身，虽然也有些防身的功夫，但是在沧岚雪的面前简直是不堪一击。
<br/>
<br/>        不过就目前的状况来看，带了太多的人也只是枉然。
<br/>
<br/>        “你给安男打个电话吧。”李长卿提醒寒伯安。
<br/>
<br/>        “打电话？打电话能解决问题吗？我不打电话证明我不在乎她，打了电话就证明我在意她了。”每个人都有软肋，寒伯安的软肋是安男没错，也许安男的一举一动都被监视起来了。
<br/>
<br/>        安男，对不起，我给你带来了危险，安男，接受我就要接受我的一切，包括家庭，包括危险，我不能时时刻刻陪在你身边，安男原谅我！安男，你是睿智的，你肯定能解决一切，你一肯定能发现一切，是不是？
<br/>
<br/>        寒伯安的心乱了，弟弟虽然安全了，但是妻子面临危险，他被困国外。
<br/>
<br/>        回到分公司，寒伯安叫来王璔，又仔细的询问了和沧岚雪对弈的经过。
<br/>
<br/>        他们将数据调集到一起，发现他们要运送的货物有些和之前吴家的那次事件大同小异，原来他们早就盯上了寒家，原来甸城谷家和吴家是一体的。
<br/>
<br/>        寒伯安在纸上写下了：吴家--谷家---陆家---张家！张函---陆玉---陈子昂--谷强--落雪。
<br/>
<br/>        寒伯安---寒冰玉------落雪-----吴家------落白------张家。
<br/>
<br/>        他放下了手中的笔，如果落雪出问题，围绕着落雪身边的一切都会遭到重创。
<br/>
<br/>        落雪和寒冰玉曾经拔掉了吴家的左膀右臂。他们现在最大的依仗就是寒冰玉和落雪不要出现任何问题。
<br/>
<br/>        “今天，落家的股票有没有问题？”寒伯安问李长卿。
<br/>
<br/>        “下午已经开始动荡了，只是落家根深蒂固，怕是一时三刻动不了。”
<br/>
<br/>        夜已经深了，寒伯安心思沉重。
<br/>
<br/>        “明天见机行事吧。”
<br/>
<br/>        “我给王璔安排过了，沧岚雪如果要来的话就说你的腰扭伤了，卧床不起。”李长卿当时是怕见沧岚雪，拖一天是一天，等寒伯安回到国内确保安男安全了再说。
<br/>
<br/>        “我为什么腰会坏？”
<br/>
<br/>        “你这样的身份身边没有几个莺莺燕燕怎么可能？”
<br/>
<br/>        “李长卿，你.....你简直是败坏我的名声，就我这身体腰会扭伤？”
<br/>
<br/>        “你知我知就行了，明天先应付一下沧澜雪。”
<br/>
<br/>        “要应付也不是这样应付，明天见他，让他随我一起回去洽谈合作方案。”

<br/>        国内

<br/>
<br/>        “安男？”
<br/>
<br/>        “子昂，我好久没有见到你了，很想你！”
<br/>
<br/>        “你在哪里呢？”
<br/>
<br/>        “我在寒家老宅。”
<br/>
<br/>        “这么想我，那我去看看你？”
<br/>
<br/>        “你忙不忙？”
<br/>

<br/>
<br/>        潘禺说：“事情很紧急？我陪你一起去吧。”
<br/>
<br/>        “财叔，春姨，你带着孩子回家，我和潘禺一起去趟上海，明天下午就回来了。”陈子昂和潘禺打车赶往机场。
<br/>
<br/>        “宁雅，帮我买两张最近赶往上海的机票。”陈子昂打电话给宁雅。
<br/>
<br/>        “子昂，怎么回事？”潘禺不明白陈子昂为什么突然要去上海。
<br/>
<br/>        “我有一个朋友遇到了麻烦，我们需要带她离开。潘禺，我怕你也会搅入一些事情中来，但是今天也只能让你陪着我了。”
<br/>
<br/>        “你和我是亲人，母亲很担心你，让我帮她找一个得力的人陪在你身边，小姨，不要什么事情都自己担着，我们是一家人！”
<br/>
<br/>        陈子昂看着潘禺，是啊，他们是她的家人，血脉相连的家人，有他们站在她的身边真好。

<br/>        ......
<br/>
<br/>        安男在傍晚见到了陈子昂，陈子昂来访，让寒老爷子有些吃惊，虽然他知道寒伯安和陈子昂有合作的关系，但是陈子昂却从来都没有见过寒好爷子，今天跑来见他这个老头子有什么事情？
<br/>
<br/>        落座之后，陈子昂对寒老爷子说：“这么冒昧的来打扰寒伯伯实在是不应该，只是我要是不说来找您，我怕是连您这大门都进不了，子昂给寒伯伯陪罪了。子昂其实是来找安男的，这家伙天天的腻在家里，连我这个好朋友都不见，我好不容易来一趟上海，她总该陪我逛逛大上海吧。”
<br/>
<br/>        “哈哈哈，我就说我这个老头子哪里来的魅力，能让你来找我，原来是冲着安男来的，得了，安男啊，你这几天就陪着子昂好好转转大上海，最好也带她去我们寒家看看。”寒老爷子知道寒冰玉和落雪出事的消息，心里悲愤而难过，寒伯安也没有消息，陈子昂此时来肯定有用意。
<br/>
<br/>        “好的，那我就带着子昂出去逛逛。”安男是一天都不想在寒家老宅呆下去了。
<br/>
<br/>        “姑父，我可不可以也随着嫂子一起啊？安男嫂子，我和你们一起吧？”苓希看着安男要出门便有些着急，沧岚雪明天可是要找寒伯安谈合作，而安男就是他们手中的筹码。
<br/>
<br/>        “不好意思，我和安男都好久不见了，我们两人有些私密的事情要说，不想让别人知道。”陈子昂毫不犹豫的拒绝了。
<br/>
<br/>        “那这个人呢？”苓希指着潘禺，这个男人真好看，看样子身手也不错，她想知道是谁。
<br/>
<br/>        “我的侄子！他不是外人。”陈子昂不知道苓希是谁，她没有必要在苓希身上浪费时间。
<br/>
<br/>        “寒伯伯，我们告辞了，我还要去落家一趟，落叔叔让我回去看看，另外落叔叔让我转告您，不可急躁，不可鲁莽！静等！”
<br/>
<br/>        苓希眼睁睁的看着陈子昂带着安男离开，她又无法出手阻止，第一，她没有理由阻止，第二，她的身手不如潘禺。
