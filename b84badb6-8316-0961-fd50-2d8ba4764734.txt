　　苍穹神宫。
<br/>　　宫殿中，苏莫正在静静的修炼，努力的淬炼玄力、扎实根基。
<br/>　　虽然他现在根基已经足够扎实，但积累的越浑厚，待冲击境界瓶颈之时，才能越轻松。
<br/>　　在北海女皇和三长老离开之后，第二日也有一人来到了苍穹神宫，那就是青雅公主。
<br/>　　对于此女，因为修炼的木属性武学，苏莫便命人将其安排在了长青宫。
<br/>　　玄力在经脉中流动，一个周天又一个周天，无休无止。
<br/>　　苏莫静静的修炼，心境如水，他知道北海冰宫会做出让他满意的选择，他只要安心等着对方将灵石双手奉上便可。
<br/>　　有了这些灵石，他便能再次的冲击境界瓶颈了。
<br/>　　对于北海冰宫这等大势力，他是有心想吞并的。
<br/>　　不过，他也清楚的知道，这些大势力，都传承悠远，自有其傲气与坚持。
<br/>　　他想让这些势力臣服简单，但想要吞并，这些势力必定会拼死反抗。
<br/>　　虽然这对苏莫来说也没有威胁，就算对方反抗也无用，但他要的是大融合，可不是将其它势力灭门。
<br/>　　若是他强势吞并了所有势力，这些势力的强者死的死，逃的逃，剩下的人就算同意加入了苍穹神宫，也不是心中自愿。
<br/>　　总之，苏莫不想造成过多的杀.戮，力求保存各大势力的实力，让所有人都自愿加入苍穹神宫。
<br/>　　如此的话，日后的苍穹神宫才有凝聚力，不然，强势吞并这些势力，强者折损就不说了，一个个都是被迫加入苍神宫，那反而是弊大于利。
<br/>　　不过，对于此事，苏莫也没有操之过急，只要他修为再次突破，到时候天下臣服，能够尽皆被他所用，也是一样。
<br/>　　时间在修炼中飞速流逝，不到一个月的时间，北海女皇便再次的来到了苍穹神宫。
<br/>　　其亲手将数十个储物戒交给了苏莫，随后便立刻告辞离去。
<br/>　　离去之时，此女面上充满了落寞！
<br/>　　数十个储物戒中，灵石无数，囊括了极品灵石、上品灵石、中品灵石，各种级别的灵草灵药，神兵利器，甚至是功法武技，简直数之不尽。
<br/>　　除了灵石和灵药留了下来之外，苏莫将其他所有的物品，全部储存在了苍穹神宫的宝库之中，用于苍穹神宫的发展。
<br/>　　……
<br/>　　神图空间中。
<br/>　　苏莫盘膝而坐，无尽的灵石和灵草、灵药形成一座座小山，将他包围在中央。
<br/>　　北海冰宫送来的所有修炼资源，全部在此了。
<br/>　　这些资源，若是全部兑换成极品灵石，不下三百亿之多。
<br/>　　“足够了！”
<br/>　　苏莫深深的吸了口气，虽然他提升修为所需的灵石越来越多，但三百亿的话，也是足够了。
<br/>　　少倾之后，苏莫释放出吞噬武魂，开始了疯狂的吞噬。
<br/>　　霎时之间，灵气喷涌，方圆数万丈之内，瞬间便被浓郁的浓雾充满，苏莫的身形彻底被无尽灵雾淹没。
<br/>　　灵雾涌动，翻滚不休，如同一只太古巨兽，蛰伏之地不断的蠕动。
<br/>　　海量灵气入体，苏莫立刻便开始冲击武皇境五重境界。
<br/>　　大混沌五行神诀运转，灵气不断的被炼化成五种玄力，分别汇入五枚神丹之中。
<br/>　　五枚神丹光华大放，立刻便震颤了起来，冲击更高的层次。
<br/>　　时间在苏莫的修炼中不断的流逝，一个时辰有一个时辰。
<br/>　　笼罩方圆数万丈的浓郁灵雾，不断的缩小，约莫十个时辰之后，灵雾便只有方圆百丈了。
<br/>　　而这个时候，苏莫心中也是稍稍有些急躁了起来，这么大笔的灵石若是都不能突破境界，他真是要吐血了。
<br/>　　轰隆隆！！
<br/>　　苏莫的丹田之中，轰响不断，五枚神丹已经达到了一个极致，到了突破的边缘。
<br/>　　片刻之后，当灵雾还有方圆五十丈左右之时，五枚神丹终于突破了极限。
<br/>　　轰轰轰！！
<br/>　　五枚神丹齐齐一震，随即急速暴涨了起来，苏莫周身的灵雾，顷刻之间，便全部被他吸纳的一干二净。
<br/>　　少倾之后，一切都归于了平静，暴涨的神丹沉寂了下来。
<br/>　　苏莫张口吐出一口浊气，心有有些唏嘘，就差一点就失败了。
<br/>　　不过，他虽然成功的突破到了武皇境五重，但由于最后所剩灵气太少，他的神丹内的玄力增幅并没有多大，导致他根基极为薄弱。
<br/>　　不过，这无所谓，只要他静心修炼一段时间，便能补充起来。
<br/>　　现在，突破到了武皇境五重，苏莫才算是真正的以自身的实力，达到了无敌的地步。
<br/>　　之前他虽然已经足够强大，能够打败巫森，但也是借助秘术之力。
<br/>　　现在，他不动用秘术，只是本身的实力，他也已经无敌了。
<br/>　　随后，苏莫又去探望了一番芊寻月，便离开了神图空间之中。
<br/>　　回到宫殿之后，苏莫静心打坐，稍稍催动武魂，吸纳天地间的灵气。
<br/>　　苏莫这一坐，便是半个月的时间。
<br/>　　半个月后，他的修为便彻底稳定在了武皇境五重的初期。
<br/>　　随即，苏莫便不再修炼，盘坐在宫殿中，仔细思索接下来的事情。
<br/>　　试炼者大劫，已经迫在眉睫，只有几年时间了。
<br/>　　九宫传人还没有凑齐，他必须去一趟蛮荒山脉了，将小八接回来。
<br/>　　再者，他需要不断的提升修为，需要大量的资源，必须想个方法，大量的囤积资源。
<br/>　　思索了良久之后，苏莫便命人传唤海明江。
<br/>　　不过，海明江已经离开了苍穹神宫，亲自出去主持擒拿黄泉魔宗余孽一事了。
<br/>　　于是，苏莫便传唤来了钱副阁主。
<br/>　　“拜见宫主！”
<br/>　　进入宫殿，钱副阁主便立刻向苏莫行礼。
<br/>　　“钱长老，多日不见，看来你的修为大进啊！”苏莫打量了一番钱副阁主，面上顿时露出了一丝笑容。
<br/>　　现在的钱副阁主，分明已经达到了武皇境九重的巅峰，而且身上气息蠢蠢欲动，这是即将进入准帝之境的迹象啊！
<br/>　　“呵呵！托宫主的福！”
<br/>　　钱副阁主微微一笑，这一段时间，他修炼帝级武学，实力提升的很快，他完全有信心，在一年之内达到准帝的地步。
<br/>　　“不错！希望你能尽快更进一步！”
<br/>　　苏莫微笑着说道，对方的实力提升，他也很是高兴，毕竟这是他的班底，也是他日后对抗试炼者的资本。
<br/>　　随即，苏莫继续道：“钱长老，这次找你来，是有一事要你去办！”
<br/>　　“宫主请吩咐！”钱副阁主恭敬的说道。
<br/>　　“替我传出消息，给苍穹世界所有大势力，就说我苏莫要统御整个苍穹世界，顺者昌逆者亡！”
<br/>　　苏莫眸中精光闪烁，言语霸气千秋，继续道：“另外，两个月后，我会备下宴席，邀请各大势力的宗主来苍穹神宫一聚！”
<br/>　　钱副阁主闻言，顿时心中一震，随即，他便感觉胸中骤然升起万丈豪情。
<br/>　　统御整个苍穹世界！顺着昌逆着亡！多么震撼人心的话语。
<br/>　　不过，钱副阁主却是没有任何怀疑，因为他已经看出了苏莫修为再次提升了，完全有统御天下的实力。
<br/>　　“是，宫主，我这就去办！”钱副阁主点头，随即立刻告退了。
