        &lt;h3&gt;第三卷第二百六十六章  子悦子乔&lt;/h3&gt;
<br/>
<br/>        陈子悦开车将陈子乔送回家，这一段时间闫欢走了，她的工作量一下子就大了起来，父亲生病她也是只能抽空来看看，白天都没有时间，还好二哥在，有二哥陪着大家也都能放心。
<br/>
<br/>        今天看着老父亲开心的样子，心里也甚是欣慰，爸爸的心愿总算是了了一件，现在唯一的不足的是老六一直杳无音信。
<br/>
<br/>        这么多年了，也不知道老六还在不在？
<br/>
<br/>        “四姐，你说姑姑都能找得到，六弟是不是也能找得到？”陈子悦问陈子乔。
<br/>
<br/>        “前几年我们也有找过子寒，只是感觉被阻挠了，后来爸爸让我们不要操那份心了，我们也就不再提起这件事情，哎，也不知道老六在哪里。”陈子乔也叹了口气。
<br/>
<br/>        “四姐，你最近工作怎么样？”
<br/>
<br/>        “我最近的工作可能要调动，我不想在这个位置上干了，操着皇帝的心，当着太监的命，太累了，找一个清闲的位置，我还能帮着照顾家里，俩个孩子的学习也很重要。”陈子悦叹了口气。
<br/>
<br/>        她现在离陆家和马总越来越远了，只是她心里明得和镜子一样，如果她想离开怕不是一件小事情，在她手中过的项目资金多得很，掌握的资料也多得很，虽然她不曾伸手，但是她经手了。
<br/>
<br/>        “那你就找个轻松点的工作啊，你又不缺钱，我姐夫将生意做得那么好，还怕你没有饭吃？”
<br/>
<br/>        “说得容易，做起来难，工作是想调就调的啊。”陈子乔突然就笑了，过一天算一天吧。
<br/>
<br/>        “也是啊，那你就好好的上班吧，在你的这个位置上，也累得很呢，而且你这个工作还很重要，证明领导赏识你，放心你。”陈子悦安慰着自己的姐姐。
<br/>
<br/>        “好了，别光说我了，说说你吧。”陈子乔笑着打断妹妹的话。
<br/>
<br/>        “我啊，就那样，一天忙成狗，虽然很辛苦但是很有成就感，只是闫欢走了，我有些不适应，有时候想事情的时候就会对着旁边的办公桌说：闫欢，你看这件事情怎么做？半天没有反应，然后一抬头才发现闫欢不在。”陈子悦的语气里充满了想念。
<br/>
<br/>        “你和闫欢搭档了这么多年，突然一走确实不习惯，不过天下无不散的筵席，人总是会分开的，早一天晚一天也没有什么区别，只是把你的忧伤提前了而已。”陈子乔说。
<br/>
<br/>        “咦，四姐，你一点也不会安慰人，不知道你是怎么做到你这个位置上的，难道是因为你的皮囊？”陈子悦和四姐开了个玩笑。
<br/>
<br/>        “你四姐难道就只有一副好看的皮囊？我不但有好看的皮囊，我还有有趣的灵魂，还有很高的iq、eq。”陈子乔用手捶了一下妹妹的胳膊。
<br/>
<br/>        “得了吧，iq和eq是我们陈家人的共性，你以为只有你一个人有啊？”陈子悦再次打击姐姐。
<br/>
<br/>        “go，你这个小坏蛋，有你这样欺负姐姐的吗？”姐妹俩说说笑笑的就到了陈子乔的楼下。
<br/>
<br/>        “你不上去看看你家俩个外甥？”陈子乔邀请陈子悦去家里。
<br/>
<br/>        “算了，太晚了，俩个臭小子有什么好看的，我还得去接蔷悦放学，今天保姆请假了。”陈子乔要接俩个孩子放学，蔷悦在托管班补课。
<br/>
<br/>        “行了，那你赶紧滚吧，路上开车注意安全。改天我们一起去看三姐。”陈子乔看着妹妹开车离开，转身回家，俩个孩子安静的坐在书桌旁写作业，请的家教老师在给孩子批改作业。
<br/>
<br/>        日子和谐而安宁，幸福而满足，此生一直这样下去该有多好。
<br/>
<br/>        .......
<br/>
<br/>        陈子悦开着车将孩子接回家，最近这一段时间，她忙得连吃饭的时间都没有，闫欢一走，她感觉工作量一下子就增加了，让她都有些适应不了。
<br/>
<br/>        看来这几年确实太过于依赖闫欢了，四姐说得对，天下无不散的筵席，闫欢总有一天要离开的，这不，闫欢一离开她就显得捉襟见肘了。
<br/>
<br/>        闫欢去甸城也不知道是情况，就是给她发了一个信息：“已到，勿念！”
<br/>
<br/>        这都是个什么话？难道就不知道说详细一点，难道她闫欢就不想她陈子悦吗？
<br/>
<br/>        不行，等一会要给她打个电话，好好的教训她一下。
<br/>
<br/>        将蔷悦和楠悦安顿好，看着俩个孩子睡下了，陈子悦松了口气，王浩楠只有周末才回来，周末回来的时候王浩楠会将整个家务活和照顾孩子的任务接过去，她周末才能轻松点。
<br/>
<br/>        她和王浩楠的关系一直就那样不死不活，王浩楠这几年也改变了很多，对她也上心了很多，只是她已经心如死水，为了孩子就维持着现状。
<br/>
<br/>        梳洗完毕，换了睡衣，坐在床上，拿起手机看了一下时间，已经晚上十一点半了，闫欢每次也就是十一点半就休息，不知道这个时候打电话合不合适。
<br/>
<br/>        陈子悦想了想，拨通了闫欢的电话，响了很久，闫欢才接起来。
<br/>
<br/>        “子悦，有事情吗？”闫欢的声音很轻。
<br/>
<br/>        “怎么这么久才接电话？你还好吧？”陈子悦有些担心的问。
<br/>
<br/>        “我好着呢，你不用担心我，我现在在做前期的准备工作，等一切都妥当了我给你说。”闫欢声音压得很低。
<br/>
<br/>        “那好吧，你自己注意点，我很想你。”陈子悦的声音有点哽咽了。
<br/>
<br/>        电话那头沉默了一会：“子悦，世事艰难，你要学会爱你自己，我不在你身边的时候你要坚强点，因为你不是你一个人，你还有孩子，还有父母，还有兄弟姐妹。”
<br/>
<br/>        “我知道了，你记得给我发发信息，让我好知道你的情况。”陈子悦的眼泪顺着脸颊流下来，这么几年，她和闫欢情同姐妹，同甘共苦，这份感情不是一般的深。
<br/>
<br/>        “行，我记得了，没事了早点休息，告诉蔷悦和楠悦要好好学习，我明天还有事情，先挂了。”闫欢急匆匆的挂了电话。
<br/>
<br/>        这让陈子悦的心里更不安了，闫欢一定是有事情瞒着她，要不然不会这么急匆匆的挂电话，而且她好像不是在房间里，电话里似乎有风声。
<br/>
<br/>        闫欢这么晚在外面干什么？在外面为什么不敢大声和她说话？
<br/>
<br/>        闫欢，你到底有什么事情非要去甸城？你去甸城真的是去投资吗？你真的是去为子昂打前站吗？
<br/>
<br/>        陈子悦被自己的一个个为什么折腾得翻来覆去无非入眠，她觉得她始终都没有看透闫欢。
<br/>
<br/>        三姐夫还特意跑她这里来问过闫欢的一些事情，让她更是揪心。
