　　苍穹神殿之中。
<br/>　　苏莫端坐于帝座之上，下方左侧是司空炎、闻人天都等六位九宫传人，右侧则是帝释和海明江等帝玄宫和天涯海阁的强者。
<br/>　　众人早已回到了苍穹神宫，苏莫回来之后，便立刻召集众人议事。
<br/>　　众人目光凝望着苏莫，现在打败了巫森，想必苏莫接下来会有大动作。
<br/>　　只不过，苏莫到底有没有斩杀巫森的战魂，众人心中迫切的想要知道。
<br/>　　不过，众人没有询问，都在等待着苏莫的回答。
<br/>　　“苏莫，外面聚集了好多武者，要求加入我们苍穹神宫！”司空炎率先开口，说起了山门之外的情况。
<br/>　　“此事稍后再说！”
<br/>　　苏莫摆了摆手，随即目光望向海明江，沉声道：“海阁主，有一事需要你帮忙。”
<br/>　　“苏宫主尽管吩咐！”海明江闻言立刻抱拳，言语不仅客气，隐约还带有一丝恭敬。
<br/>　　现如今的苏莫，乃是苍穹世界无敌的存在，虽不是武帝之境，但堪比武帝之境。
<br/>　　任何人面对苏莫，都会感受到压力，即便是海明江都不例外。
<br/>　　“想办法给我查探出黄泉魔宗之人的踪迹！”苏莫吩咐道。
<br/>　　众人闻言讶然，看来苏莫失败了，没有追到地狱三头狼战魂。
<br/>　　“苏宫主，我天涯海阁很多分阁都撤了回来，怕是很难查到！”
<br/>　　海明江皱眉，略一沉吟，他又继续道：“而且，他们肯定会极力隐藏起来，苍穹世界这么大，很难寻找？”
<br/>　　“那就全部再布置出去，动用一切手段查询，不能放过任何蛛丝马迹！”苏莫道。
<br/>　　“没问题！”海明江点头，他自然知道，若是不彻底灭了黄泉魔宗，以后会有大麻烦。
<br/>　　黄泉魔宗，会如同一只隐藏的毒蛇，不知道什么时候就会来咬你一口。
<br/>　　“有任何蛛丝马迹，立刻通知我！”苏莫叮嘱道。
<br/>　　“嗯，老朽现在就去安排！”海明江言罢，便立刻起身，准备前去安排。
<br/>　　“海阁主稍等，不急于一时！“苏莫摆了摆手，距离帝玄山脉之战，已经过去了几日，再去追查黄泉魔宗之人，也不在乎晚个一时半刻了。
<br/>　　“现在我有要事和诸位商议！”苏莫沉声说道。
<br/>　　众人闻言，目光灼灼，静等苏莫道来。
<br/>　　苏莫略一沉吟，目光望向帝释，道：“岳父大人，帝玄宫已经不存在了，你和诸位长老，都加入我们苍穹神宫吧！”
<br/>　　帝释等人闻言一怔，完全没想到苏莫会在这个时候说起此事。
<br/>　　帝玄宫之人，除了帝释之外，只剩下了三、四十人，均是上次来参加苏莫大婚的人。
<br/>　　这些人全部是帝玄宫的高层，如大长老华太山、十三长老等人。
<br/>　　众帝玄宫的强者，均是转头看向了帝释，他们要看宫主帝释的意见，毕竟对方是宫主。
<br/>　　帝释沉默了下来，若是帝玄宫没有被灭，他是绝对不可能同意加入苍穹神宫的。
<br/>　　但现在情况不同了，帝玄宫已经不存在了，虽说以他和剩下的这些人，完全能重建宗门，但根基已毁，百年之内都不可能恢复元气。
<br/>　　“我同意！”沉默了良久之后，帝释点了点头，同意加入苍穹神宫。
<br/>　　帝释心中清楚，加入苍穹神宫对于他们来说，有百利而无一害，尤其是他，作为苏莫的老丈人，肯定不会受到亏待。
<br/>　　“我等没有意见！”见帝释同意，其他三、四十名帝玄宫的强者，纷纷开口了。
<br/>　　他们对于加入苍穹神宫，倒是没有什么抵触。
<br/>　　“很好！”
<br/>　　苏莫见此微微颌首，随即又转头望向了海明江。
<br/>　　海明江以及数十名天涯海阁的强者，见到苏莫望来，顿时心中一惊，难道苏莫也要将他们吞并？
<br/>　　要知道，天涯海阁和帝玄宫可不同，虽然总部被灭了，但还保留了大部分的力量。
<br/>　　而且天涯海阁，也没有帝释和苏莫的这种关系。
<br/>　　“海阁主，天涯海阁也并入苍穹神宫如何？”苏莫淡漠的说道。
<br/>　　果然！
<br/>　　海明江闻言心中一凛，苏莫果然是要吞并他天涯海阁！
<br/>　　海明江没有回答，眉头紧皱了起来，他天涯海阁屹立苍穹大陆数万年了，岂能被吞并。
<br/>　　若是天涯海阁在他手中，被吞并的话，他也对不起天涯海阁那些逝去的师祖。
<br/>　　望了一眼苏莫，海明江发现苏莫脸色冷漠，眸中一片冰冷，顿时将口中要拒绝的话噎了下去。
<br/>　　一时间，海明江心念急转，暗暗思索应对之策。
<br/>　　他之前也想到过，苏莫会不会吞并他天涯海阁，但黄泉魔宗大敌当前，他不得不与苏莫合作。
<br/>　　“海阁主，现在黄泉魔宗虽灭，但巅峰力量都还在，你天涯海阁若是脱离苍穹神宫，必然会遭到疯狂报复！”
<br/>　　苏莫见海明江的神色，立刻开口说道，他知道对方不会轻易同意，所以要软硬兼施。
<br/>　　海明江闻言，依旧闷不做声，但他的脸色却是微微一变。
<br/>　　苏莫此言明着是告诫，但言外之意，却是在告诉他，若是天涯海阁不并入苍穹神宫，对方不会保护天涯海阁的安全。
<br/>　　海明江心中有些恼怒，这是苏莫在给他施压啊！
<br/>　　这时，便见到苏莫微微一笑，再次开口，道：“海阁主，想必你们都知道我苍穹神宫的底蕴，你们若是愿意加入，帝级功法人手一门，你们可以随意挑选最适合自己的功法！”
<br/>　　对于天涯海阁，苏莫志在必得，但他不想对天涯海阁用强，毕竟天涯海阁对他恩情极大，他还不至于恩将仇报。
<br/>　　所以，他要让天涯海阁自愿加入苍穹神宫，只用靠利诱了。
<br/>　　“什么？”一众天涯海阁之人，包括海明江、钱副阁主在内，闻言全部心神一震。
<br/>　　人手一门帝级功法？
<br/>　　随意选择最适合自己的？
<br/>　　这么夸张？
<br/>　　天涯海阁众强者都震惊了，就算是一门帝级下品功法，对于他们来说，也是无价之宝了。
<br/>　　而苏莫，居然说给他们人手一门！
<br/>　　一众帝玄宫的强者，也是个个面色呆滞，眼眸中充满了震惊和渴望。
<br/>　　帝级功法，足以让任何人疯狂了！
<br/>　　功法要比同级的兵器贵重太多了，因为功法才是武者修炼的基石。
<br/>　　众人心中暗想，苏莫给予天涯海阁之人这么多好处，应该也不会亏待他们吧？
<br/>　　再怎么说，苏莫也是帝玄宫的女婿，应该不糊亏待自己人吧！
<br/>　　“海阁主，我看你老年岁也不小了，若是无法突破到武帝之境，终究要化为一杯黄土！”
<br/>　　苏莫微微一笑，开始利诱海明江，道：“我传你一门帝级中品功法，想必能让您老有更大的几率，冲击武帝之境！”
<br/>　　帝级下品功法，还不足以诱惑海明江，所以，他用上了帝级中品的功法。
<br/>　　咕咚！
<br/>　　苏莫话音一落，大殿中不禁响起了吞咽口水之声。
