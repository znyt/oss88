        童筱颖坐在出租车上，眺望着车窗外，思绪上有些出神。
<br/>
<br/>        脑海里全是昨晚大哥傅夜沉那八块腹肌、人鱼线，身材好到爆表的影子，直到现在，她都只觉双腿间还有些隐隐作痛。
<br/>
<br/>        尽管大哥事后已经很温柔地替她上过药了，但是……
<br/>
<br/>        这件事情，让她有了后遗症！
<br/>
<br/>        若不是今早，傅氏集团总部设计部部长孙莉莉给她发了一封电子邮件，通知她在十点之前赶到傅氏集团设计部，不然，她此刻一定还躺在床上起不来。
<br/>
<br/>        因为她现在佩戴了助听器，不敢再轻易自己开车，而大哥先她一步去了公司，所以她现在只能自己搭车去傅氏集团大厦。
<br/>
<br/>        昨天，她过得有些混沌，但记忆却格外犹新。
<br/>
<br/>        生日蛋糕是吃了，同时大哥也把她给吃了。
<br/>
<br/>        因为，他说，她是上天送给他的生日礼物，就是要留在他生日这天才开吃。
<br/>
<br/>        他还说，她为他而生，他守身如玉二十七年，只为她破茧成蝶。
<br/>
<br/>        大哥还跟她说了许多许多的桃色情话，就连他祖传的宝贝，他说他都已经毫不保留、物尽其用地全给了她。
<br/>
<br/>        现在想想都觉得，大哥昨天为了诱哄她跟他上床时所说的那些话，也实在是太好笑了……
<br/>
<br/>        大哥那绝对是从高冷帅气的大总裁秒变饥肠辘辘的大色狼。
<br/>
<br/>        从餐桌上一路吃到了他卧室里的那张大床上，又从大床上吃到了浴缸里，然后又回到了大床上，整个过程他像揉面团一样，将她揉出各种姿势。
<br/>
<br/>        总而言之，场面十分激烈，冰火两重天，让人终生难忘就对了。
<br/>
<br/>        而为了不让刘婶发现，引起不必要的尴尬，她起床后，特意把那条斑驳得不忍直视的床单揉成一团，塞进了大哥的试衣间的储物柜里，打算回头自己再抽空偷偷洗了。
<br/>
<br/>        大哥说，那条国际品牌的天蚕丝床单，价格有六位数，而且还是限量款。
<br/>
<br/>        呃——
<br/>
<br/>        童筱颖想想都觉得肉疼，不就一床被单嘛！虽然睡着确实挺舒服的，但也至于花那么多钱啊！
<br/>
<br/>        含着金汤匙出生的男人，消费观念真就跟她这么一个普通老百姓不在一条线上。
<br/>
<br/>        傅夜沉是富N代不假，虽然很会花钱，但也很会赚钱。
<br/>
<br/>        童筱颖觉得自己这一辈子，几乎不可能达到大哥那种游刃在奢华里，又能将奢华权数赚回来的境界。
<br/>
<br/>        大概，是她上辈子拯救了银河系吧！
<br/>
<br/>        “你获得了市体育馆外形设计的第一名，怎么就没见你主动来跟我提价？”到达设计部部长孙莉莉的办公室里时，孙莉莉的这句话，让童筱颖的思绪有些飘。
<br/>
<br/>        什么第一名？！
<br/>
<br/>        童筱颖眨巴着眼睛，一脸迷茫地看着孙莉莉。
<br/>
<br/>        或许真的是因为大哥傅夜沉太迷人了，让她脑子里至今都是一些想入非非的东西。
<br/>
<br/>        “我跟你说话了！”孙莉莉眉头一皱，怀抱双臂，严肃了起来。
<br/>
<br/>        童筱颖猛然回神，难以置信地反问：“你说我拿了第一名？不会吧！我那天在网上看了，我前十五名都未进，连挂网的资格都没有。怎么可能会得第一名啊！”
<br/>
<br/>        “你得去谢谢傅总，是他特意过来，询问了你的设计图，破格让你有机会挂上去的。”孙莉莉莞尔一笑，其实她打心底地也认同她童筱颖的设计图。
<br/>
<br/>        当时，童筱颖的设计图在公司内投票选就是第一名，但是最终竟然未进入前十五名，倒是让她这个当部长的心里有些纳闷。
<br/>
<br/>        傅夜沉虽然作为外行，但眼光还是不错，她孙莉莉都看得上童筱颖的设计图，他傅夜沉一定也能看上，所以不至于连前十五名都进不了。
<br/>
<br/>        童筱颖顿时喜上眉梢。
<br/>
<br/>        孙莉莉接着拿出一张工作证和一式两份的合同，递给童筱颖，开门见山地说道：“既然已经定下用你的设计图作为市体育馆的外观造型了，那么我现在正式外聘你为我们这个项目的首席建筑设计师，希望你能来我们公司跟进接下来的结构设计与装修设计工作。外聘你的特权，我已经向傅总打报告批准了，你的外聘薪水，扣税后是十二万，可以不用坐班，但是得监督操持全过程的设计工作，每月定期向你支付一万，十二个月支付完成。这是合同，如果没有其它问题，你就把合同给签了。”
<br/>
<br/>        童筱颖双手接过合同和工作证，嘴角情不自禁地微微上扬。
<br/>
<br/>        难道，这就是传说中的爱情与事业双丰收吗？
<br/>
<br/>        她的努力，没有白费，对不对？
<br/>
<br/>        拿起笔来，童筱颖连合同条款内容都未看，就果断地把合同给签了。
