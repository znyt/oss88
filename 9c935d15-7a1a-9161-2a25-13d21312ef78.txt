　　大唐准备举办华夏杯马球赛的消息如同长了翅膀一样向着外界扩散。
<br/>　　首先反应过来的就是一群老兵痞，而后是各大世家，一时间几乎所有的老家伙全都跑进了皇宫，把李承乾堵在书房里面叫嚷着要代表大唐参加比赛。
<br/>　　这帮老头子要资历有资历，要年龄有年龄，要无耻有无耻，一个两个就跟打了鸡血似的，也不知道哪来的那份自信。
<br/>　　程咬金仗着自己国丈的身份，手扶叉腰肌堵在门口，将一群老货挡在身后，用破啰一样的声音吆喝道：“陛下，这次的华夏杯俺老程一定要参加，俺程家人老子英雄儿好汉，保证把那些异族番邦打出屎来。”
<br/>　　“放屁，华夏杯是打马球的，不是打人的，你把人打出屎来就特么犯规了！”李道宗身为皇叔不甘示弱，从后面上来将程咬金挤到一边，横了他一眼说道：“边规则都不懂，你这匹夫还参加个屁的华夏杯，哪凉快哪呆着去。”
<br/>　　“哎我说你个老不死的，是不是有段时间没有揍你，皮子紧了，竟然敢与老夫如此说话！”
<br/>　　“说你又如何！当年也不知是谁，败在老夫手下！”
<br/>　　“当年那是老子没有吃饱，你趁人之危得不得好汉。”
<br/>　　说着说着，程咬金和李道宗两个老杀才便吵出了真火，互相拉扯着出门PK去了。
<br/>　　李承乾揉着有些快要炸开的太阳穴，无奈的看着门口再次涌进来的人群，还没等开口就听一人喊道：“陛下，这华夏杯马球大赛事关国体不可大意，老臣家中五百假子，个个能征善战，不如从他们中间选出几人参加，定能夺魁而归。”
<br/>　　张亮，有五百个干儿子的牛、、逼人物，李承乾对他没有什么好印象，不过眼下人家又没有造反，所以小李也没法说什么。
<br/>　　“张老匹夫，五百假子算的了什么，莫非忘了我薛家兄弟，想当年……”
<br/>　　又是想当年，又是两个老货，两句话的功夫，又一波人出去真人PK了。
<br/>　　李承乾眼见这么闹下去也不是个事，再加上今天早上也不知是吃什么吃坏了肚子，竟然在这个时候腹痛如绞，结果却被一群老家伙堵在房间里出不去。
<br/>　　感受着肠子都快要断开的强烈冲击，李承乾努力夹紧了双股，咬牙看着又进来的几个老人渣：“都有谁想要参加的，自己回去报上名来，然后把参赛球员的家人给朕送到左、右卫大营，将来球赛打赢了好说，若是输了，全部砍头！”
<br/>　　“啊？”几个老头儿刚一进来，还没等开口就被吓了一跳。不就是打个马球儿么？至于么？还带绑架、砍头的？
<br/>　　不过，看李承乾脸色铁青的样子，几个老家伙又不得不相信他说的是真的，一个个全都缩了缩脖子，不说话了。
<br/>　　毕竟皇帝那可是金口玉言，说到就要做到，这万一要是真的踢输了，几十颗人头落地，到时候主动揽下事儿的人岂不是罪大恶极！
<br/>　　李承乾见老头儿们不吵吵了，这才觉得心情好了些，敲了敲桌子继续说道：“这件事情朕已经交给长乐去办了，回头会在大唐境内举办一次选拔赛，到时候你们听通知就好，现在都退下吧。”
<br/>　　见李承乾不提砍头的事情，老家伙们松了口气，也知道刚刚的举动已经惹得李承乾心生不快，此时自然不会再争什么，纷纷点头应道：“是！臣等告退！”
<br/>　　片刻之后，房间里的老头儿走了个精光，李承乾长出一口气，用最快的速度起身然后向房间一角的便桶走去。
<br/>　　然而事与愿违，就在他还没有走出几步的时候，程老货的声音从外面传了进来：“陛下，俺老程赢了，这比赛应该是俺程家来打了吧！”声落，人已经出现在李承乾的面前，堵住了他的去路。
<br/>　　“让，让开！”李承乾此时已经憋的脸都青了，偏偏程咬金这老货却还在夹缠不清。
<br/>　　“咋啦？脸怎么都青了？陛下，虽然你还年轻，但……有些事情要有节制啊，若是长此以往身体怕是要垮了，这可是俺老程的金玉良言，您……”程咬金喋喋不休，跟个长舌妇似的，丝毫不知李承乾此时已经快要炸了。
<br/>　　终于，李承乾还是忍不住了，伸手一指房门：“丈人，麻烦您一下，先从这里走到外面，然后从外面把门关上。”
<br/>　　“呃……”老程愣了一下，完全搞不明白李承乾是个啥意思，照着他的话做完之后才发现，自己竟然已经在房间的外面了，而房间里面则传来了阵惊天动地的“噼里啪啦”声，以及李承乾的抱怨：“一群老不死的，可憋死老子了。”
<br/>　　好尴尬，程老货的一张老脸几乎成了紫茄子，心道：早知道这样自己就不应该出来，多待一会儿才好，说不定某些人就真的爆开了。
<br/>　　天可怜见，如果李承乾知道老程有这样的想法，一定会后悔当初把他调回来，让自家老头子把这老货带走多好啊，多省心啊。
<br/>　　……
<br/>　　两日之后，大唐旬报刊登了两则消息，一是皇历八百六十四年八月中秋（也就是明年）将在长安举办华夏杯马球赛；二是两月之后将要举行的长安杯马球赛。
<br/>　　大唐旬报现在已经是百姓眼中最权威性的报纸，这上面发出的通告基本上证实了此前所传小道消息的真实性。
<br/>　　而这条消息一确定，立刻有人欢喜有人愁，比较乐观的人认为华夏杯马球大赛是一场全世界范围的比赛，可以吸引异域番邦前来比赛，到时候正好可以向他们展示我大唐国威。
<br/>　　而一些悲观的人则认为，这就是在扯蛋，现在大唐已经是这片大陆上的第一强国，根本没有必要玩这些花样，若是那些番邦肯来还好，若是不肯来，玩这种花样只会让大唐丢尽脸面，完全是一场得不偿失的举动。
<br/>　　所以一时间不管是民间还是官方，全都在为这件事情争论起来，大街上，酒楼中，甚至青楼妓馆里面也时不时会传出各式各样的辩论。
