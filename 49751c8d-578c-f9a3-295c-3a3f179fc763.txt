　　无数剑光，漫天闪烁。
<br/>　　论起剑道，现在的上剑宗，远远无法跟剑道门所媲美。
<br/>　　剑道门的剑术，虽然是剑道之主以一己之力开创而出的，可要比起上剑宗的剑术，要玄妙的多，云泥之别。
<br/>　　正因如此，曾经的上剑宗，才会联合其他六大宗门，一同来攻打剑道门，为的就是剑道门之中的剑术。
<br/>　　这枯叶，出身自剑道门，懂得许多剑道门曾经的剑术，不过为了隐藏身份，一直没有动用。
<br/>　　如今在被楚风眠揭穿了身份之后，他便是没有任何的顾忌了，一招一式，皆是剑道门的剑术。
<br/>　　“小子，我本不愿意动用这剑道门的剑术，这是你逼我的！”
<br/>　　枯叶盯着楚风眠，面露狰狞道。
<br/>　　“我知道你得到了剑道门的传承，剑道门的无数剑术，也落到了你的手中，可惜跟我相比，你的剑术，不值一提，给我灭！”
<br/>　　轰！
<br/>　　枯叶手中的剑锋，一把刺入到了虚空之中，旋即就在楚风眠的周围的虚空之中，数百道剑锋，突然想着楚风眠斩杀而来。
<br/>　　虚空刺剑！
<br/>　　剑道门曾经的刺杀剑术之一。
<br/>　　四面八方的虚空之中穿刺而来的剑锋，根本是让人防不胜防，无可抵挡。
<br/>　　“可笑，旁枝末节罢了，枯叶，哪怕是万年之后的你，剑术也不过如此，就凭你也想杀我？”
<br/>　　楚风眠看到这枯叶打出了无数剑术，眼神之中却是露出了几分不屑的神色。
<br/>　　要是让现在九域的剑修，看到这枯叶的剑术，只怕是会觉得玄妙无比。
<br/>　　可在楚风眠的眼中，这枯叶现在的剑术，却是不值一提。
<br/>　　在曾经剑道之主的无数弟子之中，剑术领悟最高的人，便是楚风眠。
<br/>　　如今楚风眠二世为人，他的剑术更进一步，这枯叶的剑术，在楚风眠看来，不过是在玩耍罢了。
<br/>　　“给我破！”
<br/>　　楚风眠脸色没有任何变化，手中剑锋突然向前一指，一抹剑光闪烁。
<br/>　　只见在楚风眠面前的无数剑锋，尽数破碎。
<br/>　　一剑之下，全部化解。
<br/>　　随后楚风眠的身躯，更是向着那枯叶逼近，手中的剑锋之中，无尽杀意，笼罩在上面。
<br/>　　剑锋之上，都蕴含着浓浓的血腥味。
<br/>　　杀戮剑术！
<br/>　　对待这叛徒，楚风眠可不会有这任何手下留情的想法。
<br/>　　他的杀意，全然凝聚在了这剑锋之上，打出杀戮剑术，直指枯叶斩杀而去。
<br/>　　顷刻之间，这一道杀戮剑术，便是化为了一道血色剑气，斩杀在了枯叶的身躯之上，他身边纷纷凝聚而出的屏障，都被无情的碾碎。
<br/>　　剑修交手，比拼的便是双方的剑术，剑道领悟。
<br/>　　而在剑道领悟方面，楚风眠跟枯叶相比，这枯叶不过只是孩童罢了。
<br/>　　楚风眠要杀他，轻而易举。
<br/>　　“剑神境的巅峰！你的剑道领悟已经是到了剑神境巅峰，这怎么可能！这是曾经无数剑道门的真传弟子，都不曾达到过的层次！”
<br/>　　打出的无数剑术，被楚风眠瞬间破解，那枯叶的脸色大变，不可置信的大喊了起来。
<br/>　　他的剑道领悟，也已经是达到了剑神境的层次。
<br/>　　在万千剑修之中，都足以算得上是一尊强者了，可现在楚风眠的剑术，居然足以可以碾压他，这只有剑神境巅峰的剑道领悟，才做得到。
<br/>　　但是这一重境界，曾经剑道门之中，唯有剑道之主，一个人可以做得到。
<br/>　　哪怕是其他剑道之主的弟子，像是剑凌等人，都无一可以做到这一点。
<br/>　　这种逆天的剑道领悟，现在居然是出现在了楚风眠，这样一名不过二十岁的少年身上。
<br/>　　“谁说曾经的真传弟子，无人可以踏入这一层次的？”
<br/>　　楚风眠听到枯叶的话，嘴角却是露出了几分冷笑之色。
<br/>　　“曾经的真传弟子？楚剑白？楚风眠？你是楚剑白？”
<br/>　　听到楚风眠的话，那枯叶猛然一愣，突然想到了什么一样，旋即死死的盯着楚风眠，露出不可置信的神色道。
<br/>　　“楚剑白，你没死？”
<br/>　　“不可能，你怎么可还活着？”
<br/>　　曾经剑道门之中，除了剑道之主之外，的确还有一人，在剑道领悟上，踏入到了这一步，那就是曾经的楚剑白。
<br/>　　只是当时的楚剑白，哪怕是剑道领悟在高，也没有人会在意，因为他没有灵力。
<br/>　　在强的剑术，落到了他的手中，都一文不值。
<br/>　　可这一世的楚风眠，却是截然不同了，他拥有的灵力，更是凝聚神脉，得到战龙决，力量雄厚到了一种不可思议的地步来。
<br/>　　在加上剑神境巅峰的剑道领悟。
<br/>　　在剑道一路上，现在的楚风眠，所向无敌。
<br/>　　“不错，看来你也知道你是死在了谁的手里了，也算是让你做了一个明白鬼。”
<br/>　　楚风眠听到枯叶的话，眼神无比冰冷。
<br/>　　“既然知道了，那就准备好受死了吧！”
<br/>　　楚风眠是楚剑白，只有死人，才有资格知道。
<br/>　　“楚剑白没死，居然没死，还拥有了灵脉，不好，这消息必须要传递出去，要让上剑宗闭死关的老古董出手，诛杀此子！”
<br/>　　听到楚风眠的话，枯叶也明白过来，楚风眠是承认了他的话。
<br/>　　楚风眠，就是楚剑白，这曾经剑道门之中第一的剑道天才。
<br/>　　尤其是现在，这楚风眠，非但是没死，反而是拥有了灵力，再也不是那个，剑道惊人，却没有任何力量的楚剑白了。
<br/>　　这可是要比起当年的楚剑白的威胁，要大得多。
<br/>　　“八方归元！”
<br/>　　轰！
<br/>　　刹那之间，在楚风眠剑锋即将斩杀抵达的一刻，这枯叶的身躯，猛然炸裂开来。
<br/>　　漫天血雾，飘散在空中，从他的身躯之中，一把银色的灵剑，当即飞了出去。
<br/>　　本命剑元！
<br/>　　与其他武者不同，剑修的根本，却是身体之中的本命剑元。
<br/>　　只要是本命剑元还在，哪怕是身躯破碎，也可以重新恢复过去。
<br/>　　这枯叶，居然是舍弃肉身，直接逃走本命剑元，要将楚风眠是楚剑白的消息，传递出去。
<br/>　　这个消息一旦是传递出去，只怕是就连七大宗门之中，那些已经是坐死关的老古董，不惜身死，也要出手，来杀了楚风眠。
<br/>　　可今日，楚风眠既然是敢告诉这枯叶身份，也就不会让这枯叶，将消息传递出去了。
