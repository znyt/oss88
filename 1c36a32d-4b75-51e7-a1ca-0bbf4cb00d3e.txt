　　第260章
<br/>　　草原上下起雨来，初时只是一朵云，很快连成一片，然后铺天盖地，宛若天河开了水闸，大雨倾盆而下。
<br/>　　程大雷病了，失血过多导致抵抗力下降，再加上连日来的高强度工作，他毫无意外的病倒了。身体发热滚烫，眼皮像山一样重。
<br/>　　马车的车顶是油布制成的，可仍旧挡不住这样的瓢泼大雨。李婉儿不停用汗巾擦拭着程大雷额头的雨水，冰凉的雨珠砸在他脸上。
<br/>　　“程大雷，你醒醒，你醒醒，不要睡。”
<br/>　　李婉儿焦急的喊着，看到程大雷整张脸红肿起来，皮肤苍白中透着一股诡异的红。
<br/>　　咚咚咚！
<br/>　　马蹄声疾，一队马匪在大雨中急奔，马蹄落在地上踏出纷乱的水花。
<br/>　　瞬间，马车已被人围住，急促的马蹄声响，他们围绕着马车转圈，雨水打在脸上，发出放肆的笑声。
<br/>　　“人，人来了。”李婉儿脸色苍白如纸。
<br/>　　程大雷握紧匹夫剑，本来已极其虚弱身体像弹簧一样绷紧。他走出马车，看到雨幕如帘，马贼将车围住，然后在程大雷出来的时候，四五柄刀向他脑袋上砍来。
<br/>　　爆烟花！
<br/>　　剑化作雨，从一点绽放，刺破雨幕。
<br/>　　刀在空中脱手，而他们的主人在同一时间被刺穿手腕。程大雷的剑扎在毛驴身上，因受伤而吃痛的毛驴向前狂奔，冲出包围。
<br/>　　而程大雷的身体无力垂到，脸上青红变幻，呼吸越发虚弱。
<br/>　　李婉儿扶住了他，沉重的身躯压在她身上，她摇晃着程大雷的身体，只能得到程大雷虚弱的回应。
<br/>　　马贼已追了上来，他们来的速度很快，像雨幕里黑色的豹子。
<br/>　　李婉儿看到程大雷想握剑，他小臂的肌肉绷紧，可举起的剑像一柄无力的旗帜。他像一张绷紧的弓，可没有一张弓的弓弦可以一直绷紧的，比如现在，这张弓的弓弦拉断了。
<br/>　　他已没有继续战斗的力量。
<br/>　　李婉儿忽然咬紧牙关，将被雨水淋湿的发挽在脑后，系了一个马尾。
<br/>　　“把剑交给我吧，如果，我还没把一些东西忘记的话。”
<br/>　　她去夺匹夫剑，可发现在半昏迷状态下，程大雷仍然紧紧握着剑柄。
<br/>　　程大雷冲她摇了摇头。
<br/>　　她还给程大雷一个白眼。
<br/>　　“抛白眼也这么好看呐。”
<br/>　　程大雷默默的想。
<br/>　　一个手指一个手指将程大雷的手掰开，剑握在李婉儿手中。
<br/>　　马贼追来，最急躁的一个马贼将弯刀砍向程大雷。
<br/>　　回旋斩。
<br/>　　以双脚支撑，以腰部为中心点，借着身体旋转，增加挥剑的速度和力量。
<br/>　　刀和剑相撞，在空气中发出暴躁响声，像虎和豹搏击，谁也没有讨到便宜后的怒吼。
<br/>　　程大雷怔住，瞳孔在一瞬间扩大。
<br/>　　李婉儿姿势不可谓不狼狈，动作甚至有些滑稽，但她的剑术，却是极其正确的发力方式。
<br/>　　一个没有实践，却掌握理论的学霸？
<br/>　　“我虽然不知道你说的吉祥物是什么。”李婉儿挑挑眉毛：“但我真的不是吉祥物。”
<br/>　　程大雷一怔，忽然大喊。
<br/>　　“小心！”
<br/>　　李婉儿一惊，挥剑格挡。
<br/>　　明帝未必是个称职的帝王，自然也不是称职的父亲，但对于子女的教育工作他是从来不肯放松的。李婉儿从小就学了很多东西：弦琴、围棋、书画、茶道，还有……剑术。
<br/>　　李婉儿自然不是一个合格的学生，她的老师却定是帝国最优秀的老师。她杂七杂八学了很多东西，有些到今天已经忘了，但身体却还替她记得。
<br/>　　中平刺，回燕斩，烟波荡，童子三拜，苏秦背剑……
<br/>　　脑海中存储却不曾动用过的剑术，在这一刻被启用，记忆它们的不是脑海，而是肢体。
<br/>　　她念起幼年老师教导时的情景，在大殿之上，自己如何嘲笑老师的发簪像个香炉。然而，此刻却有些恼恨当初没有多学一些，多记一些。
<br/>　　原来所有学到的东西都会被用到，哪怕只有一次，哪怕只用一次。
<br/>　　现在的她就像一个中学生握住兵乓球拍，有人对她说：现在国家无人，拿冠军的任务就交给你了，喔，你的对手是连续三届世界冠军。
<br/>　　球在球拍间来往，连她自己都没想到，她打了足够多的回合，却还没让球落地。
<br/>　　可是，一个中学生是不可能战胜世界冠军的。
<br/>　　她用双手握剑，以期望发出最大力气，可她的速度和力量都在下降，那些泼面而来的刀光，令她越发眼花缭乱。
<br/>　　程大雷的手掌握住了她的手掌，然后，刺。
<br/>　　准备的洞穿一个马贼的手掌，换来大雨中一声凄厉的惨叫。
<br/>　　她微微贴近程大雷，让二人距离更近，程大雷贴着她的耳边。
<br/>　　“要快，要更加快！”
<br/>　　她若在风雨中颠簸的小船，此刻找到了舵手，她双手握剑，爆发自己可能的最大力量。
<br/>　　“要快，要快，要更加快！”
<br/>　　那沙哑的嗓音宛如魔音，引导出她体内最大的力量。
<br/>　　一剑，一剑，一剑……刺穿雨幕，带来血珠。
<br/>　　“我们没有法子了。”
<br/>　　雨打在李婉儿脸上，她仍紧握着剑，小臂却因力乏而有些酸胀。
<br/>　　“其实也不是没有法子啦……”程大雷附在李婉儿耳边说了一句话。
<br/>　　李婉儿一怔，苍白的脸上挂出一种红。
<br/>　　“混账，你现在还想占我便宜。”
<br/>　　她突然暴怒喝了一声，却发现程大雷的眼神越来越虚弱。
<br/>　　“怎么可能，怎么可能，怎么可能有这种法子。”
<br/>　　她来回重复着这一句话，李婉儿脸皮涨红，忽然低下头，狠狠的骂了一声。
<br/>　　“如果不管用的话，我就杀了你。”
<br/>　　如果不管用的话，二人都要死在这里，已经不必李婉儿杀了。
<br/>　　她俯身，吻住程大雷双唇，湿润的发打在程大雷脸上。
<br/>　　只有被公主的吻祝福过的勇士，才能穿越荆棘，斩杀巨龙。
<br/>　　一柄伞在李婉儿头顶架起，挡住了漫天风雨，她感觉剑回到程大雷手中。
<br/>　　他整个人像条巨龙一样暴起。
<br/>　　他跃在车顶，身无彩凤双飞翼，爆烟花，再接直来直往。
<br/>　　他的身子宛若雨中的杀贼，一场干净的屠戮展开。群狼捕羊，可羊却在地上一滚，化成雄狮。
