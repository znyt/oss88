        不知不觉中，童筱颖的意识有些迷糊了，因为被大哥又是吻又是爱得有些软绵绵的感觉，就连她的心也跟着像触电般酥麻了起来。
<br/>
<br/>        这感觉一上来，童筱颖也情难自控地、不由自主地却仍旧带着些许青涩，去尝试着回吻着这个抱着自己的男人。
<br/>
<br/>        傅夜沉的气息，就像是女人最爱的那款法国香水，让她深深地沉迷在其中无法自拔。
<br/>
<br/>        结果，这吻一发就不可收拾了。
<br/>
<br/>        末了。
<br/>
<br/>        童筱颖坐在傅夜沉的大腿上，浑身像棉花糖一样软绵绵的，双手软趴着这男人宽厚的肩头上，指尖在他健硕的肱二头肌上画圈圈。
<br/>
<br/>        “大哥，我想在你的书房里加一张大的办公桌。”理智逐渐被拉回来后，童筱颖环顾着这书房四周时，思绪上开起了小差。
<br/>
<br/>        这书房挺大，所以再加一张办公桌也不会显得拥挤。
<br/>
<br/>        “这就是你想要的奖励？”傅夜沉剑眉微挑，薄唇意味犹尽地扬起一抹弧度。
<br/>
<br/>        而他的大手，在她光滑的背脊上来回抚摸着。
<br/>
<br/>        “我也要赶设计图！体育馆的结构设计，我得做建筑结构荷载计算书。”童筱颖一本正经地说，有意无意地偷偷挪动着身子。
<br/>
<br/>        她刚动一下，想要趁机逃走。
<br/>
<br/>        傅夜沉那原本轻抚着她背脊的大手猛地滑到了她的腰间，一个腕力就将她死死地箍在怀里动弹不得：“嗯？又想逃？”
<br/>
<br/>        “大哥……”
<br/>
<br/>        他压低了嗓音，暧昧地应了声：“嗯。”
<br/>
<br/>        “我们去床上睡觉，好不好？”童筱颖的下巴抵在傅夜沉的左肩上，小脑袋一歪，汗津津的脸颊直接贴到了他的耳后，软绵无力地提议。
<br/>
<br/>        这个时间点，应该已经很晚了吧……
<br/>
<br/>        呃——是时候该睡觉了。
<br/>
<br/>        突然间，傅夜沉的身子向后靠在了椅背上，抬起手来，握住了她的后颈。
<br/>
<br/>        他就像捏着一只小猫的颈部一样，轻轻地将童筱颖的脑袋提到了自己的眼皮子底下。
<br/>
<br/>        然后，他意味深长地捧住了她的脸颊，强迫她与自己四目相对。
<br/>
<br/>        童筱颖怔愣地凝视着傅夜沉的黑眸，才发现他的眼睛竟然是那种深邃而好看的双眼皮，羽睫长而密集，若不是眼神里透着成熟男人的精锐，想必单看眼睛一定雌雄难辨。
<br/>
<br/>        这样的对视太久，又会让人有种想接吻的感觉。
<br/>
<br/>        童筱颖原本红润的脸颊，渐渐地冒起了热气，也开始蠢蠢欲动。
<br/>
<br/>        而此时，傅夜沉的俊脸上却露出了一个意味深长的笑容。
<br/>
<br/>        他轻轻地啄了一下童筱颖的粉唇，剑眉挑了挑：“宝贝，你真是给了我一个大大的惊喜了！”
<br/>
<br/>        “惊喜？！”童筱颖猛然回神，一脸茫然地眨巴着眼睛。
<br/>
<br/>        傅夜沉薄唇微扬：“虽然你这勾引我的技术着实差了点。”
<br/>
<br/>        “诶？！”童筱颖懵然地瞪大了眼睛。
<br/>
<br/>        傅夜沉会心一笑：“不过，勇气可嘉。”
<br/>
<br/>        童筱颖顿时语塞。
<br/>
<br/>        他接着说：“下次再接再厉。”
<br/>
<br/>        听到大哥说完这些话后，童筱颖觉得自己太冤枉了。
<br/>
<br/>        她哪有勾引他啊！
<br/>
<br/>        明明不是这么一回事来着……
<br/>
<br/>        等等，她跑书房里来是来做什么来着？
<br/>
<br/>        完全给忘了！
<br/>
<br/>        童筱颖顿时又羞又恼地从傅夜沉的大腿上下来，趿着拖鞋直奔回卧室。
<br/>
<br/>        傅夜沉看着童筱颖飞奔出去的背影，意味犹尽地笑了笑，然后随意地整理了一下自己身上的衣着，继续埋头看文件。
<br/>
<br/>        从什么时候起，他忽然觉得，这个家，没有以前那么冷了。
<br/>
<br/>        凌晨的时候，他才将文件合上，身活动活动了一下颈部关节后，准备关台灯时，余光扫过地面，看到了一块像是什么布料的东西遗落在了椅子旁。
<br/>
<br/>        傅夜沉随之弯腰将那块布料从地上捡了起来，拿在手中一看，才发现竟然是一条粉色的小裤。
<br/>
<br/>        嗯——
<br/>
<br/>        看样子，他娶了一个会犯迷糊的小女人。
<br/>
<br/>        粉色？
<br/>
<br/>        筱颖似乎很偏爱粉色……
<br/>
<br/>        童筱颖回到卧室里后，趴在床上摆了个“大”字，或许是因为起初的书房运动太过激烈，让她此刻浑身软绵无力，累到大脑都停止了运作，所以才得以很快再次进入梦乡。
<br/>
<br/>        至于大哥有没有回房睡觉，她真不知道。
<br/>
<br/>        因为，当她醒过来的时候，已经日上三竿。
<br/>
<br/>        不过，隐约间，她好像又想起了什么来，大概是天才蒙蒙亮的时候，在这张床上所发生的事情。
<br/>
<br/>        傅氏集团大厦。
<br/>
<br/>        总裁办里，落以柔一边给傅夜沉端上红茶，目光一边有意无意地落在了他那握着钢笔的右手的手背上。
<br/>
<br/>        两排清晰的牙齿印，谈不上血肉模糊，但看着都觉得当时被咬的时候一定很疼。
