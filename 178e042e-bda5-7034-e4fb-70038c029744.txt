        傅昼景眉头紧锁，默不吭声，不知过了多久，他才抓着她的手臂向后推开，冷冷地呵斥道：“思妍，闹够了吗？”
<br/>
<br/>        季思妍声音哽咽，一边抽泣，一边哭诉着自己的愁肠：“我没有闹！我对你是真心的！我是真心的啊！阿景……”
<br/>
<br/>        傅昼景有些不耐烦起来，转身欲要离开。
<br/>
<br/>        季思妍不顾傅昼景的情绪，跨步上前，一把抱住他的脖颈，为他献上了自己那娇嫩的唇瓣。
<br/>
<br/>        她那长而卷翘的睫毛上，全是未干的泪痕，眼泪，让她明白，只要是为了得到面前这个自己心爱的男人，让她做什么，她都愿意。
<br/>
<br/>        只要他不冷落她……
<br/>
<br/>        傅昼景愣在原地一动不动，他的薄唇紧抿，任由季思妍那细细软软的红唇在自己唇上辗转来去。
<br/>
<br/>        他不但不闭上双眼，偏偏还要睁大了眼睛看面前这小女人在自己面前着急的样子。
<br/>
<br/>        季思妍吻得久了，却没有得到回应。
<br/>
<br/>        她的心里狠狠地倒抽了一口冷气，那种痛与嫉妒的感觉蔓延过全身。
<br/>
<br/>        他一定跟筱颖做过了……
<br/>
<br/>        一定是跟筱颖做过了……
<br/>
<br/>        季思妍用自己的灵舌，撬开了傅昼景紧抿的薄唇与牙关，将一粒药丸送入了傅昼景的嘴中。
<br/>
<br/>        傅昼景察觉到有一丝异物入口，想要推开季思妍，却被季思妍紧紧地搂住，她的灵舌在他嘴里肆意搅动，直到他把那里药丸吞入肚中。
<br/>
<br/>        “咳咳咳——”傅昼景差点呛到，用力将季思妍推开后，一边拍着胸脯给自己缓气，一边质问道，“你给我吃了什么东西？”
<br/>
<br/>        “助兴药啊！你信不信？”季思妍红唇微扬，却是一脸苦笑。
<br/>
<br/>        她太了解阿景了，阿景说要与她脱离床伴关系，他定会说到做到。
<br/>
<br/>        可是，她不甘心。
<br/>
<br/>        真的很不甘心啊！
<br/>
<br/>        傅昼景身心一怔，抬起手来想要给自己催吐，却被季思妍伸手制止。
<br/>
<br/>        她再次踮起脚尖，环住他的脖子，送上自己的香吻。
<br/>
<br/>        不知不觉中，她的吻狂热而妩媚，就像一只妖精，极具挑逗地通过他的唇，势必要将他的魂魄都吸出来才善罢甘休。
<br/>
<br/>        傅昼景刚想要推开季思妍，忽然间，意识逐渐迷糊，视野渐渐朦胧。
<br/>
<br/>        “阿景，你有没有碰过筱颖？”季思妍离开傅昼景的唇瓣后，双手捧着他的脸颊，质问道。
<br/>
<br/>        傅昼景的眼神迷离，完全找不到目光的焦点，像是被什么魔法控制了一般，整个人的神情有些恍惚。
<br/>
<br/>        是筱颍？！
<br/>
<br/>        下一个瞬间，他突然扣住她的后脑勺，低头咬上了她的唇，辗转用舌尖抚舔过她唇瓣上的每一丝痕迹。
<br/>
<br/>        季思妍经不住他的挑逗，“嘤咛”一声献上自己的小舌。
<br/>
<br/>        傅昼景顺势弯腰将季思妍打横抱起，一点余地都不留，直接转进卧室将她压进床铺里。
<br/>
<br/>        热血沸腾，从疯狂被碾压的双唇一直到白长的脖颈，然后一路而下。
<br/>
<br/>        她在他带给她的欢愉里起起浮浮，两只纤细的小手死死抓住身下的床单，抓得床单全都褶皱起来。
<br/>
<br/>        季思妍的身体如水蛇一般向上轻耸的瞬间，还是听到他俯低在自己耳畔：“筱颍，今晚，成为我的女人！”
<br/>
<br/>        这句话，犹如晴天霹雳，轰得季思妍心里、耳里、脑海里全都在嗡嗡作响。
<br/>
<br/>        他……
<br/>
<br/>        居然把她当成了筱颖！
<br/>
<br/>        她说不出自己这时这刻的心情，是又气又愤又恨又痛？！
<br/>
<br/>        为什么他要把她当筱颖？
<br/>
<br/>        她望着他的一双星目里面早已满是晶莹的泪水。
<br/>
<br/>        他的心里，真的已经不再有她的一席之地了。
<br/>
<br/>        哪怕她要了他的身体，也无法将他的心从筱颖手中拿回来。
<br/>
<br/>        至于那个筱颖，她也摸不清猜不透她的心思，但从她看他的眼神，她便知道，筱颖也爱着她的阿景。
<br/>
<br/>        一种迷茫，在心头散开，就像现在，却不知道是谁在享受着这样的迷茫，让自己沉沦，沉沦到无法自拔……
<br/>
<br/>        时间一分一秒的过去，半夜里傅昼景突然一个趔趄，睁开双眼后腾地从床上坐了起来。
<br/>
<br/>        “阿景……”一旁的季思妍感觉到这边的动静，侧身开了另一侧的床头灯，正好看见他仰靠在床头，为唇前点了一支香烟。
<br/>
<br/>        傅昼景果断地将手中的香烟往床头柜上的烟灰缸里摁了摁，然后掀开了被褥，站起身来。
<br/>
<br/>        季思妍有些心不甘情不愿地咬了咬自己的下唇，低声下气地问：“阿景是要回到筱颖身边去了是吗？”
<br/>
<br/>        “这一次，我原谅你。我们之间，没有下一次了！”傅昼景淡定自若地捡起地上的衣物穿上。
