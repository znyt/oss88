<br/>				“原来是谢家的公子，你来晚了，我们可都早就来了，快过来坐吧！”<br/>　　<br/>　　    谢玉堂见此便与那阮家大公子一起走上前去，在其中一个位置坐下，而凤九则站在他身后。<br/>　　<br/>　　    “谢家有好几位嫡系公子，你是排第几的啊？以前怎么没见过你？”一名少年问着，有些疑惑的看着谢玉堂。他与谢家的几位嫡系子弟都认识，却不认识眼前这人。<br/>　　<br/>　　    “我不是嫡系，我是旁系的。”谢玉堂说着，摘了颗摆在面前小桌板上的葡萄吃着，想到身后的凤九，便又摘下一串递给他：“给。”<br/>　　<br/>　　    凤九有些意外的看了他一眼，见他示意着，又见周围的人都错愕的看着，便伸手接了过来，也摘下一颗吃着。<br/>　　<br/>　　    “怎么样？甜吧？”谢玉堂问着，又摘了一颗吃着。<br/>　　<br/>　　    “嗯。”她应了一声，示意他看前面。<br/>　　<br/>　　    “你是旁系的？”一名少年奇怪的看了他一眼，继而看向阮大公子，问：“阮夫人请的不是谢家的主母吗？怎么会跟来一个旁系的子弟？”<br/>　　<br/>　　    在他们这些嫡系的眼中，旁系的身份比他们要低一个级别，在座的人也都为嫡系子弟，倒没想到这谢家来的却是旁系子弟。<br/>　　<br/>　　    阮大公子笑了笑，道：“据我所知，家母今天请的谢家夫人是旁系的一位夫人，我听母亲说上与这谢家夫人是旧识。”他的声音微顿，笑道：“正所谓英雄莫论出身，无论是嫡系还是旁系，只要天资出众都将是受人敬重的，诸位说是吧？”<br/>　　<br/>　　    听到这话，众人沉默了一下，倒也都点头赞成：“不错，身份是一个象征，但自身实力也很重要。”于是，众人的目光再度落在谢玉堂的身上。<br/>　　<br/>　　    凤九站在谢玉堂的身后摘着那串葡萄吃着，一边看着众人打量的目光，心下暗忖：这阮家大公子一开口便是让场面缓和下来，这些人现在都在看着谢玉堂的实力修为如何，不过，以谢玉堂的实力来说，在这同龄之中倒也算是中上的。<br/>　　<br/>　　    众人你一句我一句的聊了起来，气氛也渐渐的融合着。凤九站在后面吃着葡萄，当手中的葡萄吃完后，前面的谢玉堂又塞了两个果子过来，于是，不仅是那些站在其他公子后面的小厮脸上流露出羡慕之色，就连坐着的那些世家公子也甚是好奇。<br/>　　<br/>　　    “玉堂，你对小厮一向都这么好吗？”有人忍不住问着，从坐下来到现在，他就看见那后面小厮嘴里一直在吃着水果。<br/>　　<br/>　　    “当然不是，他不一样。”谢玉堂说着。<br/>　　<br/>　　    众人听到这话，不由问：“怎么不一样？这看着也没什么出色的地方啊！”<br/>　　<br/>　　    一名少年听了忍不住大笑起来：“哈哈哈，哪里没有什么出色的地方？你们瞧这小厮从刚才到现在嘴里就一直没停下来，玉堂拿什么他就吃什么，这都吃了多少东西了，你们说，哪个小厮像他这般能吃？”<br/>　　<br/>　　    凤九听到这话，吃着东西的嘴一顿，抬眸看了那名少年一眼，那少年被她这么一看，不由的心头一滞。<br/>
