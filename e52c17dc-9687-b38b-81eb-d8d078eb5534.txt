　　“让我们恭喜荣耀白队以3：1的战绩获得了夏季赛的冠军，进军世界赛。”当比赛结束的一刹那，解说席上的米粒已经是激动的大声的喊了起来。
<br/>　　“现在荣耀蓝白两队都已进军世界赛，华夏赛区只剩下最后一个名额，这个名额是通过春夏两个赛季各个战队的积分来绝对最后尤谁晋级的，在白队取胜过后，皇族和雷霆两支战队则是积分相同，也就是说接下来他们会有一个加赛，谁取得了胜利，谁将获得最后一个进军世界赛的名额。”林雅楠跟着开口说道。
<br/>　　如果这一次白队不是冠军的话，那么他们的积分最多，他们自然还是会获得一个世界赛的名额，而最后那个名额则是会通过冒泡赛的机制来绝对最后由谁进去。
<br/>　　现在白队已经获得了冠军，两个名额都已经决定了出来，那么最后这个名额自然是不用再去由冒泡赛来决议的，只不过恰好雷霆和皇族两支战队积分相同而已，接下来他们还会在打上一场比赛罢了。
<br/>　　“今年真的是荣耀称雄的一年啊，春季赛、夏季赛两个赛季的冠军分别被荣耀蓝白两支战队获得，这真是华夏电竞的一场盛世，希望他们能够在世界赛上继续努力，为我们在夺下这块装满荣誉的世界冠军。”米粒继续开口。
<br/>　　而场下在叶帝他们获胜的那一刻，无数的白队的粉丝已经高高的跳了起来，现场礼花喷洒，已经是燃放到了空中。
<br/>　　虽然旁边的蓝队的众多粉丝因为蓝队没有夺冠而感到遗憾，不过当叶帝他们一行人走出来的时候，这些粉丝还是给予了非常热烈的欢呼声。
<br/>　　蓝白两支战队的粉丝本来就是一家人，在比赛时双方为各自喜爱的战队加油，但是当比赛结束的时候，大家还是会为胜利者以及另一边的人投去自己最热烈的欢呼声的。
<br/>　　听着现场那震耳欲聋的欢呼声，叶帝他们几个也是有些激动的心潮澎湃，在春季赛错失冠军之后，他们终于在夏季赛的时候将这个冠军给捧了回来。
<br/>　　挥手向着现场的观众示意，随后在叶帝的带领下他们一行人便是向着蓝队的比赛室走去。
<br/>　　走进比赛室的时候，楚南他们已经是调整好了心态，一个个热情的和叶帝他们拥抱着，以此来庆祝他们的胜利。
<br/>　　这场比赛他们输的是心服口服，所以心态上倒是也还放松。
<br/>　　“哇，你们这次真的厉害了啊，三个套路，还让不让人活了啊。”蒋念一脸夸张的说道。
<br/>　　“不要怕，就这几个套路了，现在都用出来了，接下来你们还有机会的。”叶帝笑着说道。
<br/>　　“你要是再有什么套路，我真的不想再去打世界赛了。”秦霄也是在一旁开着玩笑。
<br/>　　解说席，
<br/>　　“双方的感情是真的好啊，明天会有一场雷霆和皇族的加赛，以此来决定最后一个出线名额，而在10天后，则是要在我们上海开始世界赛的小组赛阶段，到时候希望大家多多为我们的队伍加油，现在门票已经开始预售，大家可以在网上预订了。”米粒继续开口说道。
<br/>　　“这次的世界赛队伍只有60支，会分成6个小组来进行比赛，到时候每个小组的前两名会出线进行接下来的淘汰赛，接着才是半决赛以及最后的决赛，让我们拭目以待，一会会后白队的颁奖仪式，大家不要走开。”林雅楠也是在那里开口快速的说着。
<br/>　　接下来的事情就简单多了，领奖、采访，虽然打了一天的比赛了，但是夺冠的兴奋劲倒是让叶帝几人乐此不疲。
<br/>　　至于夺冠的奖金，五人合意了一下后，当众便是宣布将这些将近捐给希望小学，毕竟以他们现在的收入，他们倒是也不差这些奖金。
<br/>　　而这件事在事后自然又是得到了大家的好一顿赞赏，而在叶帝他们获得冠军的时候，其他几个赛区也是角逐出了各自的冠军队伍。
<br/>　　韩国不出意料的依旧是ST战队一枝独秀，以夏季赛全胜的战绩获得了冠军，日本则是有着小石兄弟的山口战队，还有法国的EP战队。
<br/>　　那些之前闯入过世界区王者的选手，基本上都在他们各自的赛区拿下了冠军。
<br/>　　赛后各大媒体自然是争相报道，而报道篇幅最多的毫无悬念的自然也是华夏的比赛。
<br/>　　《荣耀双雄初绽锋芒，夏季赛白队夺冠》
<br/>　　《四场比赛三个套路，世界第一人再展锋芒》
<br/>　　《冠军队决定将奖金全部捐给希望小学》
<br/>　　《香蕉跻身世界第一上单》
<br/>　　《荣耀战队即将创造历史》
<br/>　　无数的媒体在那里报道着，不仅仅只是国内的媒体，就是连国外的那些媒体也是如此。
<br/>　　而叶帝他们一行人，在接受过颁奖和采访之后，则是宴请了现场的工作人员以及还留在现场的那些职业战队的队员一起去酒店庆祝去了。
<br/>　　在北京又逗留了一天，叶帝这边陪着苏沐婉待了一天后，便是抓紧赶回了上海，接下来他们要面临的就是世界赛了，这个自然是容不得他们有半点疏忽。
<br/>　　今年是他最后一年征战赛场了，叶帝自然是想在自己退役之前，以一个冠军的身份退役，这样也好让自己的职业生涯没有什么遗憾。
<br/>　　当然了他这个退役也并不仅仅是他身体的原因，其实如果他继续打下去的话，还是可以的，就算是实力退步了，他还是一个一流的选手，加上他本身的天赋，以及逐渐成长起来的香蕉、萝卜他们，说不定还是有机会获得冠军的。
<br/>　　只不过家里的安排却是让他不得不选择退役，接下来他将会选择另外一条路要去走了，毕竟叶家就他这么一个男丁，他徒手挽救了华夏的电竞，只要在接下来的领土战中获胜，那么他就可以功成身退了。
<br/>　　加入到另外一个圈子，有着这些荣誉，在加上叶家的支持，相信他肯定会一步青云的。
<br/>　　虽然叶帝不喜欢那样的生活，但是毕竟他现在已经不是一个人了，他身后还有着那么多人等着他去守护的，他有父母、有妻子、接下来还会有孩子。
<br/>　　所以这条路他不得不走，退役这件事已成定局，甚至连教练他可能都没机会再去当了，顶多也就是偶尔给大家直播一下罢了。
