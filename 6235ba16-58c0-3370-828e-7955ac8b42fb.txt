        &lt;h3&gt;第三卷第一百零八章  宁雅&lt;/h3&gt;
<br/>
<br/>        宁雅见凌辰为了自己竟然跑到陈子昂的公司，而且听她说了公司的危机，二话不说就出手相助，说不感动那是假的，虽然她家里条件很好，在陈子昂的公司上班，什么都不缺，但是她缺温暖。
<br/>
<br/>        从家里的公司退出来后，她突然就感觉失去了价值，当初信心满满的不知疲倦的工作，其实就是为了证明自己，证明宁雅虽然是一个女儿身，但是也会撑起一片天。
<br/>
<br/>        拿着家里的钱跑了很多地方，只是发现自己像一个流浪的人，漂浮不定，钱花了，心也空了，被父亲从国外提溜回来后，一度不知道干什么，后来在网上发现陈子昂他们公司招人，抱着试一试的想法，应聘到了陈子昂的公司，现在到陈子昂公司已经呆了两年。
<br/>
<br/>        有了之前的工作经验，宁雅从一个小职员不到两年就升为办公室主任，不得不说她的工作能力很强。
<br/>
<br/>        这两天带着大家将公司整个流程全部做完，心里感觉轻松了很多，想起陈子昂将她叫到办公室说的话：“做你自己！”
<br/>
<br/>        她现在难道不是自己吗？她自己的身份又是什么呢？宁家嫡长女？陈子昂办公室主任？似乎什么都不是。
<br/>
<br/>        下班，开着车回到自己在北京的家里，看着偌大的房间，冷清清的，心里没理由的一阵荒凉，忙碌的时候还好，一旦闲下来她就感觉很慌。
<br/>
<br/>        父亲打电话给她，让她回去相亲，她都不知道相了多少次亲，感觉没有一个能入眼的，而且父亲提议的相亲对象都是他们那个圈子里的人，像宁雅这样的女孩子，自身能力过硬，长得又漂亮，一般的花花公子那能入得了她的眼？
<br/>
<br/>        每个女孩心中都住着自己的男神，宁雅也不例外。
<br/>
<br/>        宁雅为自己做了精致的晚餐，倒了一杯红酒，自斟自饮。无论生活怎么样，自己一定要活得精致。
<br/>
<br/>        冬天的北京街头寒冷，室内却温暖如春，橘色的灯光打在宁雅的脸上，泛着暖融融的光，桌上的红酒滴滴入喉，激起千分情丝万分愁。
<br/>
<br/>        夜，阻挡了人们的交往，将情绪扩大化。
<br/>
<br/>        手机的滴滴声响起来，不用想，宁雅都知道是凌辰的，只有凌辰会时不时的给她发消息，尤其是在夜深人静的时候，有时候宁雅都觉得奇怪，凌辰哪里来的那么多情话可说？
<br/>
<br/>        打开手机，凌辰的消息映入眼帘：
<br/>
<br/>        雅的凌：“亲爱的雅：贵州的天是晴的，贵州的水是蓝的，没有你在身边到哪里都是雾蒙蒙的天。”
<br/>
<br/>        这是告诉她，他已经到了贵州了。
<br/>
<br/>        凌辰到北京她确实是很忙，赶着要将资料整理出来，加班加点的干活，哪有时间陪凌辰？
<br/>
<br/>        还有一点，凌辰追求了她这么多年，放一般男孩子早都放弃了，可是凌辰还是锲而不舍，其实，凌辰在大学期间也是高冷范儿的。家世也不错，人长得帅，只是总是冷着一张脸，似乎天下他最大。
<br/>
<br/>        俩个人都是高冷范儿的，怎么能擦出火花？尤其是宁雅看不惯凌辰那一张冷得怨天尤人帅气的脸。
<br/>
<br/>        当好朋友们谈论凌辰的帅气的时候，宁雅不屑一顾的说：“长得人模人样的，就是冷得像六月雪。”
<br/>
<br/>        “喂，宁雅，你也冷得像六月雪，你们真是天造地设的一对。”朋友说。
<br/>
<br/>        “真是一对的话就结成冰了，哪里能擦出火花？”宁雅也笑着说。
<br/>
<br/>        外表冰冷的凌辰其实内心是温暖的，对宁雅来说是这样的，尤其是最近的这几年。
<br/>
<br/>        他们是同学，互有联系。原来在学校俩个人来往并不是很多，每个人都有自己的生活圈子，学校也是一个小社会，尤其是他们这样的孩子们，家里的情况决定了圈子的情况，也决定了他们同学之间来往的密切度。
<br/>
<br/>        凌辰刚开始都是象征性的问候，不冷不热，毕业后也是发发消息，但是在过年过节的时候，宁雅总是会第一个收到凌辰的祝福语。
<br/>
<br/>        时间一长宁雅也就记得了凌辰这个人，凌辰的问候就像是那朵在初春里刚刚冒出来的迎春花，泛着淡淡的不经意的温暖。
<br/>
<br/>        后来，从简单的问候变成了一句句动听的撩拨人的情话。
<br/>
<br/>        再后来，就会时不时的打打电话，这两年更是宁雅忠实的听众，宁雅的欢乐、宁雅的忧愁，宁雅今天买了什么东西，宁雅今天喝了什么酒，吃了什么饭，今天和谁相亲，明天和谁逛街等等凌辰都会知道。
<br/>
<br/>        想到凌辰，宁雅心里划过一丝温暖，这个世界上能听你唠叨的人真的不多呢。
<br/>
<br/>        “外面飞舞着雪花，很冷，但是有你，很温暖。”宁雅也发了一条信息过去。
<br/>
<br/>        雅的凌：“吃饭了吗？”
<br/>
<br/>        小雅：“正在吃，红酒陪简餐。”
<br/>
<br/>        雅的凌：“少喝点红酒，美容养颜，多了就会泛滥。”
<br/>
<br/>        小雅：“你就是麻婆，管得真宽。”
<br/>
<br/>        雅的凌：“不管你，哭鼻子的时候我又不能陪在你身边。”
<br/>
<br/>        雅的凌：“乖乖的吃饭，吃完饭我陪你聊天。”
<br/>
<br/>        小雅：“嗯，你吃饭了吗？”
<br/>
<br/>        雅的凌：“餐桌上狼藉一片，酒气冲天，可是小雅还是最重要的那一个点。”
<br/>
<br/>        小雅：“好了，你也少喝点。”
<br/>
<br/>        凌辰可能是在应酬，没有再回信息过来，宁雅收拾了餐具，准备做个sap，接到了死党胡婷的电话：“小雅，你在哪里？”胡婷永远都是一副大嗓门，喊得宁雅耳朵疼。
<br/>
<br/>        “我在家，刚刚吃过晚饭，怎么了？”宁雅说。
<br/>
<br/>        “我要去你家里。”
<br/>
<br/>        “这么晚了，你过来？”
<br/>
<br/>        “老娘已经在你家门口了，赶快开门！”胡婷的大嗓门充满着愤怒。
<br/>
<br/>        宁雅打开门，胡婷两只眼睛就像是水蜜桃，提着皮箱，站在宁雅的门口。
<br/>
<br/>        “怎么回事啊？”宁雅问。
<br/>
<br/>        “哇，小雅，我好难过。”胡婷直接扑，到宁雅的怀，里。
<br/>
<br/>        “怎么了？”宁雅不解。
<br/>
<br/>        “我好饿，我还没有吃饭。”胡婷哭唧唧的。
<br/>
<br/>        “那我去给你做。”
<br/>
<br/>        宁雅给胡婷煮了一份面，可是胡婷却拿出宁雅的酒：“小雅，你陪我喝酒。”
<br/>
<br/>        “你先告诉我怎么回事啊？”
<br/>
<br/>        “你还记得凌辰吗？”胡婷问宁雅。
<br/>
<br/>        “记得啊？”宁雅心里想，刚刚还在我和聊天呢。
<br/>
<br/>        “他们俩个，他们俩个.......”
<br/>
<br/>        “他们俩个怎么了？”宁雅不解
<br/>
<br/>        “他们俩个面......”
<br/>
<br/>        “怎么可能？”宁雅不相信。
<br/>
<br/>        “你看，他们的照片。”胡婷喝着酒吃着面条，她似乎要将心里的愤怒吃出来，一点也没有了平时的温柔。
<br/>
<br/>        “这也没有什么啊？不就是一个压着，了一个，他们穿着衣服呢。”宁雅看着照片觉得好笑。
<br/>
<br/>        “你也别生气了，我刚刚和凌辰聊天呢，等会我打电话给你问一问。”宁雅安慰着胡婷。
<br/>
<br/>        “你不是说你和他擦不出火花吗？怎么到现在还在联系？”胡婷的注意力立马转移，手中的酒也放下了。
<br/>
<br/>        “是没有擦出火花啊，就是普普通通的聊天而已。”宁雅说。
<br/>
<br/>        “我要看，我要看。”胡婷抓过宁雅的手机。
<br/>
<br/>        “啧啧，凌辰的话真甜，狸猫从来都没有给我说过这么甜的话。宁雅，你沦陷了没有？我可告诉你啊，男人嘴太甜不靠谱。你千万不要被他骗了。”胡婷直接将凌辰拍死在了沙滩上。
<br/>
<br/>        凌辰因为和宁雅聊了几句，心情大好，酒量也大了很多，看着被喝得东倒西歪的人，他拿起电话给宁雅打电话。
<br/>
<br/>        “小雅，今天又是想你的一天。”
<br/>
<br/>        “啊，亲爱的.....不.....不要。”宁雅的电话里传出来的声音。
<br/>
<br/>        凌辰手中的电话跌在了地上。
<br/>
<br/>        宁雅从厨房出来，胡婷笑眯眯的说：“我给你解决了凌辰，他以后不会骚扰你了。”
<br/>
<br/>        “要你解决什么呢？我们就是聊聊天，也没有别的想法。”宁雅微笑着说。
