　　而立之年的长孙冲再也不复以前的那种冲动，整整一夜的等待让他显的憔悴了许多，进了李承乾书房的第一件事情就是把窗帘全部拉开，然后坐到他的斜对面：“高明，看开些，这种事情大家都不想看到，长乐已经去了，你不能再把自己拖垮了。”
<br/>　　“长乐的事情以后再说。”李承乾看了长孙冲一眼，并没有跟他具体说明长乐的事情，只是微微眯起眼睛说道：“拜占庭人这段时间你要盯紧些，那帮混蛋既然想要搞事情，那么我们就要给他们一点教训。”
<br/>　　“高明，你想要和拜占庭开战？”长孙冲敏锐的从李承乾的话语中找到了重点，略有些紧张的问道。
<br/>　　“开战又有何不可，你去忙吧，另外，大食人那里的动静也要注意一下，至于其他人随他们去折腾，只要不过份就不用理他们。”
<br/>　　长孙冲犹豫了一下，但是考虑到李承乾现在心情必然不好，便也没多说什么，答应一声便起身离开，琢磨着以后有时间再好好劝他一下。
<br/>　　“小白，你过来一下。”等到事情安排完了，李承乾再次将白月宁叫到身边：“去把那五姓七望在长安的话事人都找来，朕有事情要跟他们谈谈。”
<br/>　　小白点点头，却没有马上离开，深深看了李承乾一眼：“陛下，您……还是休息一下吧，已经一夜没有休息了，再这样下去您的身体会撑不住的。”
<br/>　　“朕的身体还没有糟到那种程度，你去忙吧！”李承乾的嘴角抽了抽，算是勉强笑了一下，长叹一声用只有自己才能听到声音喃喃说道：“必须抓紧时间把这件事情的影响降到最底啊，否则那丫头就算是回来了只怕也没办法做人了。”
<br/>　　“是，臣这就去通知几位世家家主。”白月宁知道无法改变李承乾的意志，神情微微一黯，低头退了出去。
<br/>　　……
<br/>　　“小夜啊，你说……朕是不是很失败？”四下无人，安静的房间中，李承乾呆呆看着脚下微微泛着暗金色的地砖，不知是有意还是无意的问道。
<br/>　　夜魅看着李承乾颓废的样子，心中也是微微发酸，叹声说道：“陛下，此事非人力所能及！”
<br/>　　但李承乾却好像没有听到夜魅的话一般，依旧保持着刚刚的姿态：“朕贵为大唐皇帝，人间帝王，却连自己的妹妹都无法守护，呵呵……，难道这就是传说中的有得有失？”
<br/>　　夜魅说起来是陪在李承乾身边时间最久的护卫，远比任何人的时间都长，虽然两人之间没有什么惊天动地的事情发生，但如果说十余年的接触下来没有一点感情那绝对是骗人的。
<br/>　　所以李承乾心灰若死的样子被夜魅看在眼中，心里泛起丝丝抽痛，那份一直压在心底的感情再也控制不住，转身来到他的面前，屈膝蹲下，抓住他的一只手，狭长的眸子里闪动着异样的光芒：“高，高明，长公主的事情并不能怪你，谁也想不到长公主竟然会有那样的想法，就算是太上皇来了，也不见得会比你处理的更好。”
<br/>　　李承乾并没有因为夜魅不一样的表现露出什么异色，惨然一笑说道：“小夜，长乐是我妹妹啊！事到如今我也不想瞒你，我有一半的把握可以让长乐再活过来，可是……，可是我以后要怎么面对她？！她昨天那样……世人又会怎么说她！若是将来父皇和母后回来，我又怎么跟他们说？”
<br/>　　夜魅听着李承乾类似说呓语的感慨，只当他的精神已经处在崩溃的边缘，抱着不想再刺激的想法，顺着他说道：“但总会有解决的办法的，眼下最重要的还是你的身体，你也不想将来长公主回来了，却发现您的身体垮了吧？”
<br/>　　“那你帮帮我吧，我知道你们都有那种可以让人失去意识的手段。”李承乾微微侧过头，与夜魅对视着，没有一点开玩笑的意思。
<br/>　　夜魅从他的眼中看到了一抹寻求解脱的味道，心中暗暗长叹一声，抬起手按到了李承乾的颈侧。
<br/>　　片刻之后，精神高度紧张下的李承乾终于因为大脑供血不足，意识开始昏沉，最后双眼微微一垂昏了过去。
<br/>　　……
<br/>　　马球赛的开幕式上发生了这么大的事情，接下来的活动只能不了了之，所有人都被严重警告之后遣散。
<br/>　　但不管怎么说，长乐与李承乾之间的事情还是不可控制的扩散了出去。
<br/>　　一时间整个长安瞬间就炸了，以文人领袖自居的各大世家和士族一个两个全都跳了起来，站在道德至高点开始声讨皇室的龌蹉行为。
<br/>　　而与那些文人领袖不同的是，坊间传言却是说什么的都有思想并不统一，有的人认为长公主敢爱敢恨，爱情本身是无罪的，况且人家兄妹之间并没有什么事情，如果只是单纯的爱似乎并没有什么。
<br/>　　而另外一部分人则与那些文坛领袖的论调一致，认为这是违背人伦的事情，这份感情根本就不应该存在。
<br/>　　总之一时间众说纷纭，莫衷一是。
<br/>　　至于那些异族番邦，对于他们来说这件事情与他们并没有什么太大关系，聪明一些的干脆就是闭口不谈，就好像这件事情根本没有发生过一样，蠢一些的也只是为长乐的结局惋惜了一下，觉得她如此轻易结束自己的生命过于草率。
<br/>　　唯有长安城玄都观主殿之中，所有灯火彻夜未熄，一群老道士一个两个眉头紧锁的坐在一起像是在等着什么消息。
<br/>　　老道士秦风这几年老的已经不成样子，看上去犹如风中残烛，不过那一双老眼却依旧明亮，如果没有意外，显然还能活不少年头。
<br/>　　而在他的旁边，袁守城却依旧还是老样子，看不出与以前有什么变化。
<br/>　　至于袁天罡，李淳风，他们在这两个老道士面前完全就是晚辈，只能老老实实的站在他们身后。
<br/>　　但这一群老道士中间，却有一个看不出年龄的道姑，童颜鹤发，面容清正，正是那传说中一直没有露过面的水月！
