　　　　晚上七点的时候，还能看到没有彻底下沉的夕阳，但是一旦过了这个时间点，天色就在很短的时间内暗了下来。不到一个小时的时间，黑暗就彻底笼罩了这片大地，夜幕降临了。
<br/>　　七点钟，就像是光明与黑暗的分界点。
<br/>　　战乱，让这里的女人沦为男人的玩物；孩子沦为手握AK的“士兵”；至于男人和老人，则是成了战乱的牺牲品。
<br/>　　天空中的乌云很厚，没有月亮，也没有星光。街道上一片漆黑，竖在马路街道两侧的路灯，早就被子弹打成了摆设，街区中的建筑中，也同样没有灯光穿透出来，原本居住在这里的居民也许是因为战乱而搬离了；也许是因为害怕，而不敢打开电灯。
<br/>　　在这样的地方行动，必须格外谨慎，因为就算是平民也有可能在你不注意的时候，突然拿出藏在衣服下的AKS-74U，对着你进行扫射。
<br/>　　罗昊、李然、李毅峰、慕容飞四个人手持武器，呈四人战术队形，在漆黑的街道中行走着。罗昊固定在左手腕的单兵PDA屏幕亮了一下，紧接着又立刻暗了下去，罗昊的耳麦中传来了云熙的声音，“Alpha-1，你要的资料已经发送到你的PDA上。”
<br/>　　“看到了。”罗昊按着耳麦，对云熙说道：“比起以前，这次的效率似乎有些低。”
<br/>　　“但是我给你弄来了最详细的资料。”云熙说道：“我的男人，这个解释你还满意吗？如果不满意的话……”
<br/>　　“如果不满意，你打算怎么样？”罗昊问道。
<br/>　　“不打算怎么样。”云熙露出一抹狡黠的微笑，随后认真的对罗昊叮嘱道：“一定要注意安全，活着回来。”
<br/>　　从罗昊让自己调查“八爪鱼”的资料，云熙就对罗昊后面的计划猜了一个大概，结合联合行动司令部强大的情报系统，云熙也就确定了罗昊采取什么样的行动。
<br/>　　“放心吧。”罗昊说道：“猫有九条命，我有十条。”说完，罗昊就中断了跟云熙的联系，竖起左手握拳，示意其他三人停止前进，并且向自己靠拢，然后推开街边一间屋子的门，走了进去。
<br/>　　“安全！”
<br/>　　“安全！”
<br/>　　确认屋子里的情况后，罗昊摘下了自己左手腕上的单兵PDA，打开里面云熙传给自己，有关于“八爪鱼”的资料，快速的浏览起来。
<br/>　　慕容飞和李毅峰两个人手持ACE-GAP39SB短突击步枪，半跪在地上，枪口指向屋子外的街道，构筑起防线。
<br/>　　罗昊把手中的PDA丢给李然，说道：“这混蛋是个彻头彻尾的色-鬼，没有女人就活不下去，就连还不满十岁的小女孩都不放过！”
<br/>　　“混蛋！”
<br/>　　李然紧握着拳头，压抑着内心的愤怒，继续查看着资料。Parker毕业于西点军校，在绿色贝雷帽服役过，还去德国和列颠国学习过特种作战，本身的实力不可小觑。
<br/>　　“有什么意见？”罗昊对李然问道。
<br/>　　“如果你是想要问我有什么作战计划，我只能很抱歉的告诉你，暂时还没有。”李然摇了摇头，说道：“这混蛋虽然好色，喜欢女人，但是他从来不留女人在自己身边过夜，而且他玩的女人都是他手下帮他从不同地方找来的，充满了不确定性，想要在他玩的女人上做文章，几乎是不可能的，而且我们根本没有那样的时间。”
<br/>　　“所以我们现在只有打进去这一种办法？”慕容飞瞥了眼李然，说道：“这实在太刺激了。”
<br/>　　“这似乎是目前来说唯一的办法。”李然把单兵PDA仍还给罗昊，抓起挂在身上的G36C，打开了武器的保险，说道：“别忘了，我们是龙隐部队的兵。”
<br/>　　李毅峰打开屋子的门，闪身走出屋子，确认街道上的情况后，示意屋子里其余三人情况安全。
<br/>　　吧嗒！
<br/>　　罗昊刚走出屋子，一颗雨珠便是从天空中砸落下来，落在这片干燥的地面上，留下一个印痕。
<br/>　　吧嗒！吧嗒！吧嗒！
<br/>　　豆大的雨点不断的砸落下来，很快就将行走在街道上的四个人完全淋湿。
<br/>　　“天气预报没说要下雨。”慕容飞嘴里嘀咕了一句，说道：“这种被淋成落汤鸡，从外套湿到内裤的感觉，实在是种差到让我想要爆粗口的体验。”
<br/>　　“其实我还是比较喜欢下雨天。”罗昊手中握着G36C，说道：“坐在阳台上，泡一壶茶，再加上小熙做的小点心，这才是生活！”
<br/>　　“不过这样的生活跟你无缘。”李然开口对罗昊打击道：“至少在你35周岁，达到强制要求退出作战部队前，这种安逸的生活跟你无缘。当然，你想要提前享受这种安逸的生活，还有另外一个办法……”
<br/>　　“得了吧。”罗昊打断了李然的话，说道：“我拒绝这种办法。”
<br/>　　李然咧嘴一笑，不再继续说话。
<br/>　　一个小时后，四个人抵达了那座被影子部队控制，现在已经变成了“八爪鱼”老巢的军事监狱。
<br/>　　就罗昊、李然、李毅峰、慕容飞四个人视野所能看到的地方，他们发现了两座哨塔，以及一个机枪堡垒。
<br/>　　两座哨塔上各有一个穿着雨衣，握着狙击步枪的狙击手在警戒，机枪堡垒中有两个士兵，一个在重机枪的防弹钢板背后，另外一个坐在旁边的子弹箱上，两个人似乎是在交谈着什么。
<br/>　　监狱大门紧闭着，正对着大门的那条马路上还有负责对进入监狱车辆进行检查的士兵，总共有两个岗亭，每个岗亭中有三个士兵，装备全自动武器。
<br/>　　李毅峰放下M2010，对罗昊说道：“这里的防御还真是固若金汤。根据情报，这个监狱只有正门，所以我们如果要想进去，只能从正门进去，但是就目前的情况来看，想要从正门进去而不被发现的可能基本为零，只要我们开枪，就一定会触发监狱中的警报。这可不是在玩游戏，杀了人，地上的尸体还会自动消失，不会被人发现。”
<br/>　　“看那里！”
<br/>　　李然指了指监狱大门左侧，说道：“我们可以尝试着搭一辆‘顺风车’进去。”
