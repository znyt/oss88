　　第478章
<br/>　　李星一时间三尸神暴跳，五灵豪气腾空。这九十七具尸体就像一个又一个巴掌，结结实实拍打在脸上。两军对垒，还未开战之前，程大雷就狠狠地给了自己一个下马威。
<br/>　　现在李星看自己的手下的眼光，就像是看一群蠢猪，某种程度上，连猪都不如。
<br/>　　“大人息怒，息怒……”军师道。
<br/>　　“息什么怒，我怎么能息怒，我怎么养了你们这群废物。”
<br/>　　李星在那里破口大骂，将手下一群人骂得狗血喷头。军师耷拉着脑袋，一直等李星骂完人，这才开口说话。
<br/>　　“大人，我们顺着摸查过一群，在外面还发现了一件东西。”
<br/>　　“什么东西？”李星黑着一张脸：“是你们丢掉的脑子么？”
<br/>　　“呃……”
<br/>　　军师尴尬的沉默了一会，挥挥手，有手下双手捧着一物走过来，用包袱皮裹着，里面是散碎的木块。
<br/>　　“这是我们昨夜搜查发现的，大人你请看，这些尸体大部分都是喉部中箭，弩箭上淬有剧毒。经过我们比对，弩箭就是由这种机关发射的。”军师解释道。
<br/>　　李星盯着那堆散碎的零件，道：“这么说，他们有我们不知道的武器，能不能由军中的工匠还原出来？”
<br/>　　李星终于冷静下来，目光充满期待的注视着军师。
<br/>　　“呃……”军师摇摇头：“已经让他们看过，破坏得太碎了，根本不知道原来是什么模样？”
<br/>　　李星眼中期待的光芒散去，黑着一张脸：“那你和我聊这些有什么用？”
<br/>　　“呃……”
<br/>　　军师咳嗽了一阵，道：“大人，我们还是聊一聊如何打好眼前这一仗吧？”
<br/>　　“通知相府和将军府，今日总攻。”李星握紧佩剑，一瞬间雄姿英发：“我要今日攻破蛤蟆军，砍下程大雷的脑袋当尿壶。”
<br/>　　在正午时分，帝国军开始调动，战鼓擂响，全军出动，将兵线往前压了五里。
<br/>　　程大雷在营门看着这一幕，微微眯起了眼睛：“这个时候开打，是要和我们打夜战呐。”
<br/>　　“大当家，现在怎么打？”刘悲问。
<br/>　　“还能怎么打，放开了打。”程大雷道：“一万蛤蟆城的精锐，拿不下五万帝国军，我也就不用打了。”
<br/>　　帝国军不停向前推进，在黄昏时分集结完毕，向蛤蟆军发动了总攻。
<br/>　　投石机、重弩往前推，骑兵跟在远程队伍后面，再之后，是密密麻麻的步兵。帝国军打仗不过就是三板斧，先用远程武器攻击，之后骑兵往前冲，骑兵冲完后步兵冲。
<br/>　　当然，假若细分的话，骑兵可以分为轻骑兵、重骑兵，负责巡逻警卫的游骑兵。步兵又分为重甲兵、弓兵、长矛兵，大刀兵、盾牌兵等等。
<br/>　　程大雷立在营门上，手中握着剑，亲临战场指挥。
<br/>　　帝国军已经开始用投石机攻击，天空中石弹纷飞，程大雷眼睛望着，一时神情肃穆。
<br/>　　徐神机和程大雷立在一起，他看着这一幕，表情有些无语。
<br/>　　“这他们打不到这里啊，为什么不往前推一推？”
<br/>　　程大雷看着他，无可奈何道：“刘悲，你给他解释一下吧？”
<br/>　　刘悲也一脸困惑的看着徐神机，顿了顿道：“咱们在地形上占据有利位置，他们由下往上打，自然打不到我们，可若他们继续向前推，就会进入我们的射程。我们在高处，他们在低处，我们的射程永远比他们远。”
<br/>　　“喔，原来如此。”徐神机做出恍然大悟的表情：“大当家高呀！”
<br/>　　“也就一般高。”程大雷摆了摆手：“放心，他们打不了多久，也就三分半的热度。”
<br/>　　“这又是为什么呀？”徐神机。
<br/>　　程大雷彻底无语，刘悲再次解释道：“并州的地形大部分是草原，想要运送投石机需要的石弹其实并不容易，他们自然是准备了一批，但撑不了太长时间。当然……”
<br/>　　刘悲目光看着程大雷，道：“这一点大当家当然也早就想到了。”
<br/>　　“呃……哈哈……”程大雷笑一阵：“当然，一切都在某家的掌握之中。”
<br/>　　事实上，关于这一点，程大雷之前还真没有想到。他派出秦蛮等人在草原上兴风作浪，的确阻碍了帝国军筹备物资，不过，这纯粹是无心之举，并不在程大雷事先的计划之中。
<br/>　　和刘悲估计得不错，帝国军的投石机攻击了一阵，便偃旗息鼓，蛤蟆军的大营前堆满了石块。
<br/>　　“大当家，天黑了。”刘悲道。
<br/>　　“嗯。”程大雷看了一眼夜色：“天黑了，就该我们动手了。也让帝国军好好学着，该怎样去打一场仗。”
<br/>　　帝国军在黄昏时发起攻击，没有起到怎样的效果。李星等人也在讨论着，该如何打这一战。
<br/>　　他们选择继续推进，一直推进到一个可以发动正面攻击的距离，与蛤蟆军展开白刃战。
<br/>　　程大雷站在营门上，盯着山坡下，只见密密麻麻的士兵如蚁聚，他们摸黑推进，手中的兵器时不时闪过道道寒芒。
<br/>　　程大雷也对帝国军做了一定的了解，他们大部分是由民夫组成，如这种冲锋在前的事情，自然都是由敢死队来完成，说不得，这些人怀里还揣着没捂热的银子。
<br/>　　在他们到达蛤蟆军的射程后，程大雷下令开始攻击。
<br/>　　先用弩箭射杀敌人，再推下滚木擂石清除障碍，饮马坡下惨叫声一片连着一片。不过，这些人也的确称得上悍勇，拼了命的往上冲。
<br/>　　程大雷摆摆手，开始下一轮攻击。
<br/>　　营门上忽然点起篝火，投石机开始发射，他们带着火光轰出去，落地时轰然爆炸，点亮了整个夜空。
<br/>　　蛤蟆城通过火药研制出的武器，不仅只有火雷一种，如今投石机抛出去的，便是另外一种。和火雷的结构大同小异，用明火点燃后，通过投石机抛出去。
<br/>　　帝国军见识过火药，却未必见识过这种武器。大部分普通战士，还以为是某种法术，一时间人心惶惶，军队的士气泄了大半。
<br/>　　这时候，营门打开，赵子龙的龙字骑率先出动。
<br/>　　帝国军有三板斧，程大雷同样也有三半斧，在远程武器玩够之后，双方开始正面冲突。
<br/>(本章完)
