　　苏莫目光瞥了一眼慕冰雪，他心中清楚，看来此女已经认出了他的身份。
<br/>　　这倒不是说明对方眼力高明，而是他并未过多隐藏身份。
<br/>　　在来到青俞岛之时，苏莫只是稍稍改变了面貌，收敛了气息而已。
<br/>　　所以，慕冰雪能认出他的身份，再正常不过了。
<br/>　　“冰雪，你说什么疯话呢？”三长老的脸色黑了下来，对方居然要他向前方的青年道歉！
<br/>　　真是可笑！
<br/>　　若不是此言出自慕冰雪之口，他都要一巴掌乎出去了！
<br/>　　他堂堂武皇境九重武者，这个武皇境四重的蝼蚁，受的起吗？
<br/>　　“三长老，道歉方可保命？”慕冰雪的目光，一直在凝视着苏莫，注意着苏莫的表情。
<br/>　　她很担心，苏莫一怒之下，大开杀戒。
<br/>　　不错，她已经认出了苏莫，虽然苏莫改变了容貌，但身上逸散的气息，却做不了假。
<br/>　　慕冰雪没有说出苏莫的身份，毕竟苏莫既然改变了容貌，就不想让人知道。
<br/>　　她若是揭穿，很可能会让苏莫恼怒，所以，她只是提醒三长老道歉。
<br/>　　不过，慕冰雪好心好意提醒，三长老却是根本不领情。
<br/>　　“慕冰雪，我看你是疯了，给我滚！”三长老彻底怒了，直接出声呵斥慕冰雪。
<br/>　　道歉方可保命？
<br/>　　简直是笑话，整个苍穹世界，除了那十数名准帝级的存在，谁能杀他？
<br/>　　言罢，三长老手中无尽寒气凝聚，刹那间便凝聚成了一杆玄冰大枪，欲将苏莫一枪击杀。
<br/>　　慕冰雪见此，顿时面色一变，她不用想都知道，三长老若是对苏莫出手，必死无疑。
<br/>　　三长老作为武皇境九重的超级强者，是北海冰宫的支柱之一，绝对不能死。
<br/>　　慕冰雪眼疾手快，身形一闪，便拦在了三长老的身前。
<br/>　　而后，她立刻向苏莫躬身一拜，道：“苏……苏宫主，我等失礼，还望苏宫主莫怪！”
<br/>　　事到如今，慕冰雪不得不点名苏莫的身份了，虽然很可能惹恼了苏莫，但她不得不救三长老。
<br/>　　所有人见此，都是心中愕然，不明所以。
<br/>　　慕冰雪为何如此惧怕这个无名青年？
<br/>　　苏宫主？
<br/>　　哪个苏宫主？
<br/>　　一时间，众人根本反应不过来，更没有联想到苏莫的身上。
<br/>　　毕竟，以‘宫’为宗门后缀名称的势力，实在是太多了。
<br/>　　正欲出手的三长老见此，立刻双眉一挑，满脸惊疑。
<br/>　　不过，慕冰雪如此姿态，他也不得不重视了。
<br/>　　苏宫主？
<br/>　　三长老自喃一声，下一息，他立刻瞪圆了眼眸，不可思议的望着苏莫。
<br/>　　“你……你是苍穹……！”三长老惊骇的说道，他已经猜到了苏莫的身份，因为能让慕冰雪如此低声下气的苏宫主，唯有苍穹神宫的苏莫了。
<br/>　　不过，三长老话未说完，便戛然而止，双眼几乎瞪出了眼眶。
<br/>　　倒不是有人打断了三长老，而是此刻，苏莫面目肌肉蠕动，恢复了本来面目。
<br/>　　随着苏莫恢复了本来面容，天地间突兀的安静了下来。
<br/>　　青俞皇帝青谷、青雅公主、一众皇室之人、北海冰宫的强者、以及皇宫之外天空中那无数的观看之人，所有人都是瞬间石化了。
<br/>　　所有的人都是瞪大了眼眸，面色凝固，嘴.巴大张，怔怔的望着苏莫。
<br/>　　一息！
<br/>　　二息！
<br/>　　三息！
<br/>　　九息！
<br/>　　十息！
<br/>　　足足过去了十个呼吸的时间，众人才猛然回过神来，刹那之间，整个青俞皇城都沸腾了！
<br/>　　“什么？居然是苏莫？”
<br/>　　“这……这怎么可能？”
<br/>　　“真的是苏莫！”
<br/>　　“天啊！我不会眼花了吧！”
<br/>　　“我看到了什么？这是真的吗？”
<br/>　　霎时之间，整个青俞皇城沸腾了，无边的声浪冲霄而起。
<br/>　　苏莫是什么人？
<br/>　　是苍穹世界的奇迹！
<br/>　　目前苍穹世界公认的第一强者，无敌的存在，能打败武帝的神话般的存在。
<br/>　　苏莫之名，早已传遍了苍穹世界每一个角落，无人不知，无人不晓。
<br/>　　苏莫的画像，也是早已在整个苍穹世界流转，就算是三岁的孩童，都认识他。
<br/>　　甚至，有些家族和宗门，都将苏莫的画像高高挂起，用以激励族中弟子或者是门下弟子。
<br/>　　青俞岛距离中洲亿万里之遥，对于青俞岛的武者来说，苏莫只是传闻中的人物，虚无缥缈的强大存在。
<br/>　　对于一些年轻武者而言，苏莫是他们的偶像，是他们为之奋斗的目标。
<br/>　　但现在，苏莫居然来到了青俞岛，真实的出现在了众人的面前，众人如何能不震惊！
<br/>　　皇帝青谷、清雅公主，一众皇室的强者、北海冰宫的强者，无数青俞皇城的武者，所有人都感觉这一幕极不真实。
<br/>　　但这一切又真实的存在，绝不会有错！
<br/>　　难怪慕冰雪让三长老认错！
<br/>　　原来慕冰雪早就认出了苏莫的身份！
<br/>　　苏莫的手中，屈覃国师脸色狂变，但他在苏莫的手中，没有半分反抗之力。
<br/>　　此刻，除了苏莫本人之人，最镇定的人，莫过于慕冰雪了！
<br/>　　她提前认出了苏莫，所以并没有太过震惊。
<br/>　　“三长老，你还等什么？”慕冰雪见身后的三长老愣神，立刻出声提醒道。
<br/>　　“啊！”三长老闻言立刻惊醒，他的脸色急速变换，若是他得罪了其他准帝强者，凭借北海冰宫这个靠山，他还能出言周旋一二。
<br/>　　但是得罪了苏莫，他连周旋的底气都没有。
<br/>　　因为在目前的苍穹世界，已经没有任何人、任何势力，能抗衡苏莫了！
<br/>　　即便心中再不甘，三长老也不得不认栽。
<br/>　　很快，三长老便咬了咬牙，向苏莫抱拳，沉声道：“苏宫主，刚才多有得罪……！”
<br/>　　三长老正要开口道歉，但话未说完，便被一声淡漠的声音打断了。
<br/>　　“三息早已过去，道歉已经晚了！”苏莫淡淡的说道。
<br/>　　言罢，他手掌一抬，玄力涌动，就欲将对方击杀。
<br/>　　三长老见此，顿时面色大变，心中惊恐不已。
<br/>　　他的实力虽强，但他根本没有信心，能在苏莫的手中撑过一招。
<br/>　　“苏宫主息怒，老夫错了！不……在下错了！”眼见着苏莫就要出手，三长老顿时急声大喝。
<br/>　　现在的三长老，哪里还有刚来之时那种霸道的气势，简直就是一头受惊的羔羊，全身都在颤抖。
<br/>　　他没有选择逃走，因为他知道，一旦选择逃走，绝对没有活路。
<br/>　　唯有真诚认错，他才有一丝活命的机会。
