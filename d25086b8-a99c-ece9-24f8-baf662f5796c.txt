　　　　被囚禁绝望的人，重获新生，颤栗的朝着洞口那道曙光走，一个个搀扶着我，我搀扶着你。
<br/>　　他们也算一起死里逃生过。
<br/>　　一个个经出黑斑蟒的身旁时，此蟒不复之前的凶煞，不敢阻拦他们，只是匍匐在地，不敢动弹一分。
<br/>　　当所有的人都离开了这座山洞，涂小安的系统之中又疯狂的涨了一波信仰之力。
<br/>　　信仰分轻重，越是信，贡献的点数就越多。
<br/>　　这山洞中走出去的三十来号，所提供的信仰之力都是百位数以上的。
<br/>　　算起来，一共五千多点。
<br/>　　信仰点数累积：512000。
<br/>　　自从蛇王庙出世，涂小安的信仰点数就暴涨了数倍，从原来的十五万，暴涨到了五十万有余。
<br/>　　光在白镇上空盘旋数秒，就让他一下子收获三十二万信仰之力。
<br/>　　此刻，昏暗而阴森森的山洞中独留一条金银色的小蛇，跟一条十五米的黑斑蟒。
<br/>　　黑斑蟒臣服在涂小安的蛇王气息下不敢放肆，一副任由宰割的样子。
<br/>　　涂小安冷淡无比的看着，意念一动，变异双头蛇凭空出现，出现在了黑斑蟒的面前。
<br/>　　“交给你了”
<br/>　　这十五米的黑斑蟒也不能浪费，收服又对涂小安无用，他要那么大的蟒有何用，不过用来培育双头蛇还是不错的。
<br/>　　变异双头蛇刚不久才吃了一头成年老鹰，如今十五米的巨蟒摆在它的面前，小家伙看到既然也丝毫的不怕，反而变的异常兴奋。
<br/>　　这对它来说绝对是一顿美餐。
<br/>　　它双头摇曳缭绕，通灵摄厉，三米长的蛇躯朝着黑斑蟒的血盆大口中爬了进去。
<br/>　　涂小安静静的看着，也不阻止双头蛇这种“作死”的行为。
<br/>　　任由它整个蛇躯爬进去，看上去像是被黑斑蟒给吞了似的。
<br/>　　反到是黑斑蟒铜铃般的蛇眸满是惊恐之色，匍匐在地的庞大蛇躯瑟瑟发抖，颤栗不止。
<br/>　　旋即，就见黑斑蟒的气息在衰弱，生命力开始迅速的在流失，这双头蛇爬进它的体内，在吸食它的能量。
<br/>　　“吼...！”
<br/>　　匍匐在地上的黑斑蟒发出了一道凄惨痛苦的吼叫声，听的涂小安都有点头皮发麻，他清晰的受到了黑斑蟒变的越来越弱，他十五米的蛇躯居然在缩水，变的干枯起来。
<br/>　　这变异双头蛇也得亏是落在涂小安的手中，要是放任在外面，它会比吸血蛇更加的残忍可怕。
<br/>　　几分钟之后，只见双头蛇从黑斑蟒的蛇尾处钻了出来，原本三米长的蛇躯，变成了五米。
<br/>　　而黑斑蟒气息全无，死于非命，原本庞大的蛇躯，此时看上去像是被风干的尸体。
<br/>　　“叮...变异双头蛇等级提升到15级”
<br/>　　此时，只见钻出来的双头蛇非常兴奋的用两颗蛇头在涂小安的面前摇曳，浑身散发这一股恐怖而冷冽的可怕气息。
<br/>　　它吞噬了一头成年老鹰跟一条十五米的巨蟒，就赫然成为了十五级，实力暴涨，而且最可怕的是它成长无上限，等级提升太快。
<br/>　　双头蛇本身具备上古蛇种的血脉，又已人胎孕育而出，出生之后，雷电加身，成为超强变异体，现在又有涂小安培育它成长。
<br/>　　简直是含着金钥匙一样。
<br/>　　都说出生是门技术活，这条双头蛇将来前途不可限量，会成为天地间无比恐怖的存在。
<br/>　　重新将双头蛇收回系统，涂小安飞出了山洞，朝着蛇沟深处前进。
<br/>　　泥土地面一条长沟顺着山林深处蜿蜒的蔓延，从天上往下看，就好像是一条拥有无边无际蛇躯的巨蟒在游曳。
<br/>　　飞行了数公里，这条原本空荡荡的长沟里却布满的各种各样的毒蛇，这些毒蛇顺着长沟朝着前方深处地带游曳，争相恐后，浩浩荡荡。
<br/>　　无形之中，涂小安既然在空气中闻到了一股异香，好像从很远很远的地方飘荡而来。
<br/>　　如今这里才算真正意义上的蛇沟地带，毒蛇的踪影已经遍地都是。
<br/>　　很多毒蛇的品种甚至连涂小安都叫不出来，生平也没有见过。
<br/>　　顺着长沟的方向足足飞了一个小时，涂小安诧异，这蛇钩深处到底指向哪里，好长的路，像是没有尽头。
<br/>　　涂小安沿途不知道看到多少条毒蛇，上万条都有了。
<br/>　　而且最恐怖的是，放眼望去，长沟里面，好像都是罕见毒蛇，密密麻麻，好像普通品种的毒蛇都没资格待在长沟中。
<br/>　　刚开始，看到罕见品种的毒蛇，涂小安还会收一些到系统中，回去交给美林街作为买卖。
<br/>　　可到了后来，他都麻木了，太多太多的毒蛇了，蛇沟不愧是蛇类的大本营，这还没到蛇沟的中枢地带，就已经遍地全是罕见毒蛇。
<br/>　　如果有捕蛇人在这里，一定会激动的大喊自己发财了。
<br/>　　都是好蛇呀。
<br/>　　当然，前提是他们有命来到这里。
<br/>　　渐渐的，一路继续顺着长沟方向飞行，涂小安已经开始看到一些大蛇，甚至是大蟒。
<br/>　　蛇有蛇的品种，蟒有蟒的品种，但在这里，全都乱套了。
<br/>　　很多蛇品种的毒蛇，都成长到了蟒那么大。
<br/>　　最为让涂小安吃惊的是，前方有一条黑色大蟒在游曳赶路，足足快达到十米了，起码涂小安还以为自己又碰到了一条黑斑蟒。
<br/>　　可飞近一看，好家伙，居然是一条黑曼巴。
<br/>　　黑曼巴可是罕见致命的毒蛇，成年的基本三米长度，可这条黑曼巴足足有十米。
<br/>　　这里的蛇类好像都突破了身体血脉的桎梏，成长无上限。
<br/>　　这是违背大自然常态的。
<br/>　　莫非这些毒蛇也是吃了那红彤彤的小果实吗？
<br/>　　此时空气中弥漫一股异香，这些毒蛇都是朝着蛇沟深处争相恐后的游曳，不知道的，还以为它们去赶集呢。
<br/>　　不过也好在这些大蛇都是待在蛇沟里面，要是游曳出来，都成了吃人的怪兽。
<br/>　　涂小安看着一条又一条的大蛇有点尴尬，因为他就算变回原形，变成十五米的大蟒，在这里也平平无奇，并不算什么稀奇的事，也没有蛇类会为他震惊。
<br/>　　要是蛇博士陈默在，更是颠倒三观，他费尽心思的将毒蛇变成大蟒，拥有庞大的身躯跟致命的毒液。
<br/>　　在这里更不算什么，随便抓出一条，都能达要求。
<br/>　　很多都是蛇类变成大蟒。
