　　第273章
<br/>　　“酒，酒。”
<br/>　　程大雷向身后招手，一坛酒放到他掌心。程大雷敲开酒坛，用极快速的身法潜进院内，又飞速撤回来。
<br/>　　昨日那场战斗，给他留下不小的阴影，程大雷心底已经对这白蛇王有了畏惧。
<br/>　　“喝了么，喝了么？”程大雷逃回来后就问。
<br/>　　大家站在很远的地方瞧，酒香在院内散开，那白蛇王睁开眼睛，蛇目寸光，可程大雷感觉到，它的眼神是居高临下的鄙夷。
<br/>　　当真不能将这蛇王当一个畜生看，事出反常必有妖，长成这样庞大身躯，它不知已经活了多少年，怕已经成妖了。
<br/>　　“当家的，它不喝，你的计划失败了。”徐神机。
<br/>　　“我的计划失败，你需要敲锣打鼓庆祝么？”程大雷白了他一眼，道：“干粮，把干粮用酒泡透了丢进去。”
<br/>　　诸人随身都携带着豆面，路上没有食物，就用水把豆面浸湿了填饱肚子充饥。程大雷命令下来，诸人都开始行动，将身上的干粮泡酒浸透，用树叶裹了丢进院子里。
<br/>　　这次当真存了破釜沉舟的心思，假若今天办不倒这白蛇王，大家下顿饭吃什么都会成问题。
<br/>　　数百斤的干粮丢进院内，散落一片，酒香也冒了出来。程大雷一颗心提着，如果这巨蛇再不上当，他只能漫山遍野去抓活鸡活鸭了。
<br/>　　白蛇王的状态其实非常不好，它并不是不想离开这院子，只是周围都是雄黄，充斥着一种令它讨厌的气味。酒香散开的刹那，冲散了周围雄黄的味道，它其实已经有几分蠢蠢欲动。
<br/>　　程大雷高估了这畜生，畜生就是畜生，永远不可能永远人类的智商，就算一个人类，把他丢在一个无人烟的孤岛上成长，最终他也只是一个蠢物。
<br/>　　它盘在一起的蛇躯开始蠕动，蛇信卷起地上的酒粮，几乎是直接吸如腹中。
<br/>　　“动了，动了。”
<br/>　　程大雷的心脏砰砰跳动起来，像刮开彩票最好一个号码一样忐忑。只见巨蛇吞尽地上的酒粮后，慢慢移动身躯，向那坛被砸开的酒。
<br/>　　“酒，酒，酒。”
<br/>　　程大雷招呼着身后，将一坛坛酒送入院内，放下后再迅速撤离。
<br/>　　最终，程大雷也不知道自己究竟走了多少趟，直到累得已经有些虚脱。
<br/>　　“酒，酒，酒。”
<br/>　　程大雷冲身后招手。
<br/>　　“大当家，没酒了。”
<br/>　　程大雷一愣，看到身后诸人都两手空空。
<br/>　　“把岸边的酒搬过来。”
<br/>　　“已经都搬过来了，所有酒都在这儿，或者说……”徐神机指指院子：“都在那儿。”
<br/>　　程大雷目光看过去，见巨蛇脑袋向这边探来，竖起的蛇首已经有些摇晃。
<br/>　　“好像没晕呐。”徐神机道：“会不会是酒劲儿不够？”
<br/>　　“这至少有上千斤酒了吧，它这么大量？”
<br/>　　程大雷说着，忽然见巨蛇竟竖起脑袋向这边过来，蛇尾仍旧缠在一棵树上，只听咔嚓咔嚓，大树倒塌。
<br/>　　“不好，它开始耍酒疯了！”
<br/>　　程大雷大叫一声，诸人一起往外撤，直到退到一个自以为安全的场地，方才敢探头往这边看。
<br/>　　然后，大家就看到一条巨蛇是如何耍酒疯的。它在地上横冲直撞，路过的大树被它碾塌，山石被其扫飞，身躯跃起，又狠狠砸下，方圆五百米根本容不下它。
<br/>　　程大雷躲在一棵大树上，看到这一幕露出恐惧的神色。程大雷为自己有三百斤、五百斤力气沾沾自喜，可这巨蛇的力气怕上万斤都不止。
<br/>　　还真是天地间的洪荒怪兽呐。
<br/>　　“大当家，它该不会一直这样耍酒疯，直到把酒劲散去吧。”徐神机躲在另外一棵树上。
<br/>　　“也不是没有可能，当撒完酒疯，总该睡一会吧。”程大雷。
<br/>　　大家都提心吊胆的看着这一幕，面对这一口能将牛吞下的巨兽，又有那个可能不怕。
<br/>　　轰！
<br/>　　真不知过了多久，狼烟动地的声势渐渐消失，等烟尘渐渐散去，大家就看到巨蛇卧在那里，不再移动。
<br/>　　“这是睡着了？”
<br/>　　“也可能是酒精中毒死亡。”
<br/>　　“大当家，你下去看看？”
<br/>　　“你怎么不下去？”
<br/>　　“我一把老骨头，爬树不容易，还是大当家去看看吧。”
<br/>　　最后还得是程大雷去打探情况，在场诸多人中，只有程大雷拥有在巨蛇攻击下逃脱的机动力。
<br/>　　他拎着大斧潜了过去，面对大蛇这样体量的怪物，匹夫剑相当于给它挠痒痒。巨蛇的气息很平稳，倒在地上一动不动，可程大雷的心情却十分紧张。
<br/>　　他靠近巨蛇的头颅，双手握斧，没想好冲那里下手。此时其他人渐渐都围了上来，谁都没有说话，谁也不敢说话。
<br/>　　大家用眼神询问程大雷，程大雷示意他们来到自己身边，指了指大蛇的脑袋。
<br/>　　程大雷，秦蛮，赵子龙，张肥，关鱼，高飞豹等等诸人都举起了兵器，程大雷点点头，双手紧紧握斧，热血技能触发，程大雷体内若有火涌。
<br/>　　斧头砸了下来。
<br/>　　秦蛮的大枪，赵子龙的铁枪，张肥的丈八蛇矛，关鱼的大刀……一瞬间说不好有多少兵器砸在巨蛇头颅上。
<br/>　　巨蛇身躯猛地弹起，在空中打了一张弓，然后是重重落下，一颗头颅已经被砸成肉酱。
<br/>　　如今是死透了。
<br/>　　大家同一时间瘫倒，斩杀大蛇，倒是没有多大体力上的消耗，可精神上的压力是巨大的。这大蛇只要稍微动一动，被它碰着了，任你多大本事，也只有死路一条。
<br/>　　良久，程大雷才从地上站起来，看到面前一条庞大的蛇尸，周围大树倒塌，岛上的建筑有一多半都变成废墟。
<br/>　　“大当家，这蛇的尸体怎么办？”徐神机道：“我可听说蛇肉很补的。”
<br/>　　“送给你了，你下辈子就吃蛇肉吧。”
<br/>　　“呃……我可不敢吃。”
<br/>　　不仅徐神机不敢吃，其他也没有谁敢对这巨蛇染指的，这样庞大的怪物，说不好是兽还是妖怪。
<br/>　　程大雷累得筋疲力尽，从怀里把蛤蟆旗抛出来。
<br/>　　“来，挂上。”
<br/>　　有人把蛤蟆旗挂上旗杆，天蟾吞月图在风中展开。
