　　收回目光，陆雪琪视线落在慕清霄身上，其中柔情似水，迈步走上前来，玉体几乎挨着他的身子。
<br/>　　强忍着羞涩，轻声细语，道：“灵药的用途，已经研究清楚了吗？”
<br/>　　感受到压在手臂上的柔软，嗅着幽幽体香，慕清霄倒吸一口凉气，这妮子简直是要他的老命啊。
<br/>　　这柔软圆润的触感，极其惊人，要知道她现在穿的可是泳衣啊。
<br/>　　蕾丝泳衣，虽然能够避免走光，但那触感，就跟没穿似得。
<br/>　　而且，在他的努力挖掘下，雪琪终于摆脱了36B，升级到了32C的程度，提高了不止一个层次。
<br/>　　简直就是赤果果的勾引！
<br/>　　深吸了口气，抛开心中杂念，慕清霄轻轻一推，数十朵鲜花就飘落在泳池中。
<br/>　　转过身来，慕清霄伸手搂住她纤细柔软的腰肢，两人就如此紧紧贴在一起，都能感受到彼此身体的温度。
<br/>　　“嗯，当初摘下的这些花朵，都不是普通灵药，能增强灵魂，珍贵无比，用掉一朵，手中就少一朵。”
<br/>　　“增强灵魂？”
<br/>　　陆雪琪美眸中闪过思索之色，显然不懂增强灵魂是什么意思，不仅只有她，身后众女也是不明所以。
<br/>　　她们接触修炼的时间已经不短，虽然实力已经是高级金丹修士，但论起修仙知识，却连入门都算不上。
<br/>　　对于许多修仙的知识，唯有陆雪琪勉强算是入门，自然也不懂灵魂对于修士的重要性。
<br/>　　别说是陆雪琪她们，哪怕是诸天万界中，没有达到元婴级别的修士，别说知晓灵魂力量，连接触的资格都没有。
<br/>　　唯有修炼至金丹，灵魂力量才会悄然觉醒，直到元婴，灵魂力量随着实力的提升而增强。
<br/>　　看到妻子们困惑的目光，慕清霄微微一笑，极有耐心的解释起来。
<br/>　　“灵魂力量对于修士而言，极其重要，但灵魂与修士的实力却是成正比的，强大的修士，并不是都有着强大的灵魂。”
<br/>　　见她们秒懂的模样，慕清霄微微点头，经过凤血，龙元的改造，她们的资质悟性本就变态，一点就通实属正常。
<br/>　　“灵魂的强大，分先天与后天，灵魂强大的修士，往往是世界中最顶尖的修士，灵魂较弱的可想而知。”
<br/>　　说着，慕清霄视线落在泳池中的花朵上，道：“但这些灵药却能增强灵魂，而且是永久性。”
<br/>　　说到这里，话语戛然而止。
<br/>　　欧阳紫嫣她们，在他面前温柔至极，没有接触过真正的修仙世界，却不代表，她们的智商低。
<br/>　　思维一转，就完全理解慕清霄话语中的意思，同时也知道，这些灵药的珍贵程度。
<br/>　　恐怕放在修仙世界中，哪怕拿出去一朵，都会掀起一阵腥风血雨。
<br/>　　见她们诱人的模样，慕清霄微微一笑，道：“你们都下去吧，按照平时的修炼，吸收其中的药力。”
<br/>　　其实，慕清霄还有许多关于灵魂的信息没有说明。
<br/>　　比如灵魂也能杀敌，强大的灵魂，能够不依靠灵魂而存在，甚至能够完全复活。
<br/>　　至于为何现在不说，那是因为，她们的境界都没到，知道的多了，对她们的心境反倒会有影响。
<br/>　　回过神来，陆雪琪微微点头，第一个踏进其中，欧阳紫嫣等人紧随其后，最后分开盘坐而下。
<br/>　　踏进泳池后，其中蕴含的药力，便顺着毛孔，钻进她们的四肢百骸中......
<br/>　　仅仅数十秒，她们的俏脸就变得通红无比。
<br/>　　伴随着心法的运转，泳池中蕴含的能量，顺着引力流淌，使得游泳池内都泛起小小的水花。
<br/>　　见她们都开始修炼，不知何时，慕清霄的手中就出现一块玉石，玉石碧绿通透，除了美感十足，没有任何的特殊之处。
<br/>　　“这究竟是什么东西呢？系统奖励的东西，绝对不可能普通。”
<br/>　　研究了半天，慕清霄都没有研究出个所以然，只好将玉石丢进储存空间中，随后又取出一个黑黝黝的盒子。
<br/>　　其上透着一股古朴沧桑的气息，正是他在迦南学院，藏书阁中获得的神秘黑盒。
<br/>　　感受到其上的封印，慕清霄微微摇头，又将黑盒丢进储存空间。
<br/>　　以他现在的实力，根本抹除不掉上面的封印，除非等他修至合体期，才有强行开启黑盒的可能。
<br/>　　闲来无事，慕清霄就站起身来，走到泳池前面，种植着绿竹花草的土地。
<br/>　　随后，取出容器，将一些低级的灵药，全都种植了进去。
<br/>　　别墅内任何一块土地，甚至别墅附近的土地，当初都经过了生命之力的滋润。
<br/>　　泥土中蕴含着一股浓郁的生命力，种植一些低级灵药，绰绰有余，甚至都有些浪费了。
<br/>　　地球上所谓的风水宝地，与金帝别苑中任何一寸土地比起来，简直是弱爆了。
<br/>　　将低级灵草种植在别墅两边的花丛中后，慕清霄拍了拍灰尘，脸上挂起一丝笑意。
<br/>　　虽然是低级灵草，派不上什么用场，但用来改造别墅中的环境，是再好不过了。
<br/>　　想到此处，慕清霄看了看周围，捏了捏下巴，陷入沉思之中。
<br/>　　反正周围的别墅都属于自己，倒不如将墙壁推倒，将院落扩大，以后专门用来种植灵药，改造这里的修炼环境。
<br/>　　此时此刻，慕清霄脑海中的想法，越发的不可收拾。
<br/>　　只是，推倒周围几栋别墅，范围依旧是太小了，不如将整个金帝别苑都圈下来，改造一番岂不是更好。
<br/>　　若是另寻他处，可没有如此土地，至于别墅中的住户，有句话叫做，有钱能使鬼推磨。
<br/>　　赔偿他们，给他们当初购买价格的两倍，不行就三倍，再不行就五倍，使劲的用钱砸。
<br/>　　而且，住在这里的客户，都知道欧阳紫嫣住在里面，辉煌集团的总裁，身份与这些住户有着天壤之别。
<br/>　　哪怕有点脑子，他们都会答应搬走。
<br/>　　地球上没有适合她们修炼的环境，他自己来造就一个不就行了，而金帝别苑正好合适。
<br/>　　哪怕是有，她们恐怕也不舍得搬走，因为早就在这里住习惯了，慕清霄也是如此。
<br/>　　“方法可行，但改造还需要慢慢来，这件事情，待会跟紫嫣商量一下吧。”
