　　红色小圆点所示的冷藏运输车，是一辆载重二十吨重厢式卡车，卡车中没有人，冷藏箱也是被上了锁。
<br/>　　很普通的插芯门锁，这并不能难倒齐凡。
<br/>　　可就在齐凡准备动手开锁的时候，他的耳麦中传来布谷的提醒声，“冰刃，有两个守卫正在向你们的方向靠近，注意隐蔽！”
<br/>　　齐凡立刻收起手中的铁丝，矮身钻到了冷藏运输车底下。
<br/>　　作战靴落在水泥地面上发出的脚步声传来，先是一道手电筒光柱扫了过来，紧接着四只穿着黑色作战靴的脚出现在齐凡视野中。
<br/>　　两个人朝着冷藏运输车的方向走来，最后站定在冷藏运输车前，对着冷藏运输车进行仔细地检查。
<br/>　　汽车底下，齐凡已经是把安装了消音器的uSP.45手枪握在了手中，如果这两个守卫尽职的要弯腰对着汽车底下再检查一遍，齐凡会毫不客气的将一颗.45口径的ACP弹送进他的脑袋。
<br/>　　冷藏公司后门街道对面的楼房房顶上，布谷也已经是用狙击镜锁定了一名身上挂着AK-74S短突击步枪的守卫，如果冷藏公司中的三个人有暴露的危险，布谷会毫不犹豫的开枪。
<br/>　　美国人已经盯上了这个冷藏公司，所以这个时候冷藏公司中不能出现枪声，不然就等于是告诉外面盯梢的美国人这个冷藏公司中有猫腻，估计用不了十分钟，特种部队的直升机就会抵达。
<br/>　　冷藏运输车边的两个守卫把冷藏运输车检查了一遍，并没有打开车厢进去检查，转身离开了。
<br/>　　得到布谷确认安全的提示后，齐凡从冷藏运输车底下钻了出来，迅速起身，继续用他手中的铁丝开锁。齐凡并不是专业的开锁专家，可是开一把普通的插芯门锁还不至于花费他很多的时间。
<br/>　　齐凡打开冷藏货箱，一股子冷气扑面而来，让人感觉非常舒服。齐凡翻身爬进冷藏货箱，把货箱门虚掩起来，打开了自己随身携带的战术手电，在冷藏货箱中搜索起来。
<br/>　　冷藏货箱中的冷冻肉已经全都搬空了，空荡荡的冷藏货箱一眼就能看到底，货箱中残留着一些冷冻肉融化后形成的血水，肉腥味很刺鼻很难闻。
<br/>　　一眼看过去，看似这个冷藏货箱中什么也没有，但是在找到这辆冷藏运输车后，齐凡就记下了这辆冷藏货箱车货箱的长宽高，心中对货箱的容积做了计算；而现在在货箱内部重新经过计算，齐凡发现两次计算的结果出入明显，即便是加上货箱钢板的厚度占去的容积，现在在内部计算得到的容积还是要小于这辆冷藏运输车应该有的容易。
<br/>　　而这就只能说明一个问题，那就是这辆冷藏运输车的冷藏货箱中有夹层，这是唯一可以解释为什么冷藏货箱内部容积会小这么多的原因！
<br/>　　齐凡用手电筒一寸一寸的打量着冷藏货箱的侧壁，期间那两个守卫又是有两次巡逻经过齐凡所在的冷藏运输车，每次经过的时候，因为有布谷的提醒，齐凡都是提前关掉了手电，防止自己的暴露。而那两个守卫每次经过冷藏运输车都没有产生什么怀疑，显然他们并不知道藏在这辆冷藏运输车中的秘密，其实这也很容易理解，P病毒非同小可自然是越少人知道越好，如果被别人得知，大概率会被人偷走，并且拿去黑市出售。
<br/>　　要知道，目前黑市上对P病毒的开价已经超过了一千万美金，其中有些是恐怖组织，有些则是某些国家的人，他们不惜砸下高价就是为了得到一支P病毒的样本进行研究。
<br/>　　冷藏货箱中的氧气越来越少，齐凡明显感觉到了呼吸困难，但功夫不负有心人，齐凡终于是在驾驶室背后的那块侧壁上找到了一条缝隙。
<br/>　　因为这辆冷藏运输车平时都是用来运输冷冻肉类的关系，冷藏货箱中侧壁上残留着冷冻肉的残渣，尤其是停止制冷后，这些冷冻肉的残渣都开始融化糊在了侧壁上，再加上侧壁上的缝隙很细小，使得齐凡一开始没有发现这条缝隙的存在。
<br/>　　“总算是找到了！”
<br/>　　齐凡抬手擦掉额头上的汗水，拔出龙牙军刀，沿着壁板上的缝隙切了进去，剔除掉糊在缝隙中的冷冻肉残渣，将这块壁板给撬了下来。
<br/>　　壁板背后被塞满了泡沫塑料，泡沫塑料中间是一个液氮低温瓶。看到液氮低温瓶，齐凡脸上露出欣喜之色，把液氮低温瓶拿了出来，拧开瓶子把存放在里面的P病毒拿出来确认了一下，把P病毒重新塞回到液氮低温瓶中，把液氮低温瓶的盖子重新拧紧。
<br/>　　“我找到布克尔特藏起来那支P病毒了。”齐凡按住耳麦，把自己的发现告诉了其他三个人。
<br/>　　这支P病毒的找到对于战术中队而言可以说是至关重要，找到这支P病毒就等于保住了B组所有队员的命！
<br/>　　“冰刃，你们最好立刻撤退。”布谷盯着狙击镜，看到冷藏公司正门街道上出现了一辆装甲步兵车正朝着冷藏公司的方向而来，说道：“看来美国人是忍不住要对冷藏公司动手了，你们立刻撤出来！”
<br/>　　布谷话音刚落，装甲步兵车上的火炮就轰开了冷藏公司的大门，装甲步兵车庞大的车身碾压进冷藏公司，冷藏公司中的守卫立刻采取了反击措施，可是AK-74S短突击步枪的子弹打在装甲步兵车根本对它造成不了有效威胁，子弹落在装甲步兵车上就像给他挠痒痒一样。
<br/>　　数名机动部队队员端着M16A3自动步枪冲下装甲运兵车，冷藏公司外也是跟着冲进十几个机动部队队员，数名force-recon队员。
<br/>　　冷藏公司中的防御很快就被摧毁，在装甲运兵车的掩护下，机动部队队员和force-recon队员冲进了冷藏公司的那些冷冻仓库中。
<br/>　　他们首先选择的是冷冻仓库，而不是停在停车场中的那些冷藏运输车，这对于齐凡、孙涛、陈应天三人而言，是个很好的撤退机会，如果不抓住这个机会，等到对方回过头来，就是想走也走不了了。
