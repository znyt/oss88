<br/>　　那些药王谷弟子，瞳孔猛然一缩，惊骇无比，见了鬼一样。

<br/>　　楚风眠话音说出的同时，一抹剑光，已经是从楚风眠的面前释放出来，他的祖龙至尊剑，猛然出窍。

<br/>　　这一道剑光的威力，就是当初药海四人联手，都无法抵挡住。

<br/>　　一声声惨叫响起，那些药王谷弟子，都是在剑光的席卷之下，身躯都是破碎，一个个当场泯灭。
<br/>　　上一刻还在耀武扬威的那些药王谷弟子，现在在楚风眠的面前，却是如此的不堪一击，全部斩杀。

<br/>　　楚风眠大手一挥，这些药王谷弟子的空戒，还有他们的一身精血，都被楚风眠所吞噬。

<br/>　　下方的无数武者，在场众人，都没有任何一人敢在开口。
<br/>　　二十三位药王谷弟子，都不是楚风眠的一招之敌。
<br/>　　以生死境五重的境界，能够做到这一切。
<br/>　　九域历史之中的那些超级妖孽，只怕都无人能够做到这一切。

<br/>　　在场无数武者的心中，都是冒出了这样一个想法。
