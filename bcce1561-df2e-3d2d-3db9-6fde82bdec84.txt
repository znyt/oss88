　　慕寒说完，几个长老互相看看，一点激情都没有，慕寒道：“怎么，你们觉得做不到吗？”
<br/>　　“星月奴而今斩尸证道，我们如何斗得过，更不要说抢夺宝剑。”一个长老说道。
<br/>　　慕寒眯眼看他，“照你这意思，压根不用打了，仇也不用报了？”
<br/>　　“我不是这意思。”
<br/>　　“那依你之见呢？”
<br/>　　这位长老左右看看，道：“我觉得，咱们应该与茅山联合……”
<br/>　　“砰！”
<br/>　　慕寒用力拍了一下桌子，“叶少阳，又是叶少阳！你们都是龙虎山的人，不是茅山的人，龙虎山想要崛起，必然要压住茅山才行，不然我干什么要当这个掌门！”
<br/>　　几大长老面面相觑，被他抢白的那位有些不爽地说道：“前任张掌教可从来没对我们拍过桌子！”
<br/>　　“他当年几十年，又做成过什么大事，风头都被茅山那两个家伙抢了。”话一出口，自己也觉得有点过，环视众人，道：“他是我亲叔叔，他有不足，我说出来，理所当然吧？”
<br/>　　这一句话把大家嘴巴都堵上了。
<br/>　　龙阳真人道：“掌教，大家都想振兴龙虎山，想要报仇，但法术公会如今太过强大，光靠我们自己的力量，实在螳臂当车，无谓牺牲……掌教有什么好主意？”
<br/>　　“以我盟主之名，召集大伙，速来龙虎山，一起商量办法！”
<br/>　　龙阳真人皱眉道：“之前那个消息……”
<br/>　　“若法术公会要攻打茅山，那正是机会，叶少阳他们不是很厉害吗，让他们去抵抗，去消耗法术公会的力量，我们来个渔翁得利，你们觉得？”
<br/>　　几大长老目瞪口呆。龙阳真人有些不爽地提醒道：“我们是名门正派……”
<br/>　　“谁说不是了？”
<br/>　　慕寒冷笑一声，“为了亲手给两位祖师报仇，为了重整龙虎山，我愿意当这个小人，诸位但随我去做便是。”
<br/>　　几大长老之中，也有一两个同意的，其余都不说话。
<br/>　　慕寒便让大伙去联络各大门派，先守口如瓶，就说是得到情报，法术公会不日就要进犯人间，让大伙前来开会，共同商讨对策。
<br/>　　也只能这么说了。
<br/>　　大伙离开之后，慕寒也离开了偏殿，从回廊绕到宫殿后面，一个人来到了后山禁地，默默地走在山路上，直到前方就能看到万妖塔，再不往前走了。
<br/>　　他不敢过去。
<br/>　　玲珑塔的封印，自从道渊真人死了之后就解除了，里头那数不清的强大邪物，却一个都没逃走。
<br/>　　它们都是道渊真人当初一个一个给关在玲珑塔下的，每天讲经超度，多年下来，戾气早就消磨了，跟道渊真人也成了极好的朋友，道渊真人死后，这些邪物几乎一个都没走，而是留在塔中，等着给他报仇。
<br/>　　不久之前，它们听说了杀害道渊真人的凶手是星月奴——影魅已经死了，纷纷要去报仇，还是慕寒苦口婆心地劝说，才把它们留下，让它们听从指挥，集中起来，等待机会。
<br/>　　它们等得早就不耐烦了，慕寒怕过去遇到它们，又被质问什么时候才能展开报仇行动。以他的实力和威望，根本镇不住它们。
<br/>　　慕寒于是转身往回走，但没有回山门，而是选择了相反的方向，翻过了两座山，爬上了一座山的山顶。
<br/>　　这是一座荒山，植被茂密，不光旅游没开发到这里，连龙虎山的弟子平时都不会来。
<br/>　　山顶有一片天然的枫林，一到秋天，红叶似火，是一处极美的风光。
<br/>　　慕寒小时候就喜欢一个人来这里，坐在枫林里吐纳修炼，四周只有鸟语虫鸣，有一种万法自然的感觉。十四岁那年，他就是在这里开的悟。
<br/>　　那一刻起，他就觉得自己不是一般人，不是一般法师，有一种天降大任于斯世人的神圣感，始终在他心头萦绕。
<br/>　　踩着厚厚的一层落叶，他走到了枫林的尽头，来到靠近悬崖的那一块山石上，伸手抚摸着巨石的某一处，这是他十几年来经常坐的地方，石头都被磨平了。
<br/>　　他坐了上去，朝山的那一面望去。
<br/>　　在一片松涛竹林的掩映中，能隐约看到一片建筑的影子。一座小的寺庙。他就是在这里度过了自己的童年和少年。
<br/>　　那里还有他的启蒙师父，还有几个师兄弟。他从小到大跟他们交往都不深，他看不上他们。
<br/>　　真正对他授业传道的，是张无生本人。
<br/>　　张无生是他亲叔叔，也是他唯一的亲人。
<br/>　　他知道张无生并不喜欢自己，反之，他也不喜欢这个叔叔。
<br/>　　他不是张家唯一的后人，但却是嫡传血脉、又是天赋最高的那个。
<br/>　　天选之子。
<br/>　　那是在他除此开悟、晋升天师牌的当天，张无生告诉他的。
<br/>　　“你不是普通的法师，你将来是要干大事的。”
<br/>　　从那天起，慕寒便牢记了自己的使命。
<br/>　　他在山石上爬到高处，回头望着远处的龙虎山。
<br/>　　宫殿之上，弥漫着一道经久不散的紫气，象征着这个古老的宗派，在法术界的地位。
<br/>　　小的时候，他多少次站在这块山石上，就这样眺望龙虎山的大殿，他知道，自己终有一天会到那里去，接管那里的一切。
<br/>　　如今，他做到了。用实力证明了自己是最有资格坐在这个宫殿的最上方的。
<br/>　　这也是张无生不喜欢他的地方，他自尊心太强，自视甚高，恃才傲物，张无生试图改变他的性格，但失败了。
<br/>　　因为他自己觉得没什么不对。
<br/>　　他从小孤独，一个人长大，他比同龄人都要成熟和冷静得多。
<br/>　　如果张无生不是突然殒命，他未必会把掌教之位传给自己，这一点，慕寒很清楚，他死得太仓促，而且除了自己，再没有任何掌教的人选，更不要说他是天选之子，是注定要干一番大事的。
<br/>　　他也看不上张无生的中庸内敛之道，因此，他从小就梦想着有一天，能够带领龙虎山恢复昔日荣光，真正的统领法术界，而不是像现在这样，靠着过去的名声在吃老本。
<br/>　　本来一切应该很顺利的。
