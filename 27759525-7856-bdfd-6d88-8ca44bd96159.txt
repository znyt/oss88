　　元始离开洪荒星后，并没有去星空之中的帝族或者圣地，而是来到星空深处的一颗死星上。
<br/>　　这颗星辰上有他感兴趣的人，或者物。
<br/>　　这颗星辰是完全死寂的，上面没有任何生灵，整颗星辰也非常特别，整体看上去是黑色的，星辰中的大地像是全部由黑铁组成一般，一样望去，起伏不定的大山都是漆黑一片。在这颗星辰的最中央，则是一座高达千万丈的巨大黑色山峰，宛若擎天之柱，耸入天际，在巨峰的顶部，则是一个巨大的漆黑洞口，像是火山口一般，下方漆黑一片，宛若深不可测的深渊。
<br/>　　元始的身影降落在这黑色的巨大黑洞旁，没有丝毫引起别人的注意。元始双眼突然泛光，发出璀璨无比的光芒，他看向深渊深处。
<br/>　　深渊内部，一片死寂，入眼所见的，是一片片骸骨，骸骨随意摆放，堆积如山。这些骸骨有大有小，大的如同巨山，小的如人类一般，有些骨骼之上，还隐隐有金色的光泽，显然，骸骨生前修为非常逆天，即便死去了无数年，骸骨依然不朽！
<br/>　　四周弥漫着黑色诡异的迷雾，容易让人迷失自己，更重要的是，这迷雾之中，带着浓浓地怨念以及死亡之力，轻易触碰不得，就算是至尊，若一个不小心，也会丧命。
<br/>　　在众多骸骨的中央，是一个方圆数十丈的黑色潭子，黑色的水潭，幽静而深远，充满了死寂，没有丝毫的波涛，水潭仿佛是一面黑色的镜子一般，黑得发亮。
<br/>　　在水潭的正中央，升起一个数十丈大小的小岛，小岛依然是漆黑一片，似乎在这里，除了黑色，就没有别的颜色了。
<br/>　　小岛之上，有一个祭坛，祭坛之上，摆放着一口青铜棺材。
<br/>　　元始此行的目的，正是那青铜小棺。
<br/>　　此棺名为轮回棺，传说是天地未开劈之前便存在了，能够令人起死回生。
<br/>　　轮回棺一直只存在传说中，谁也没有见过，如今它却出现在元始面前。
<br/>　　更何况，轮回棺，轮回二字便足以表明一切。
<br/>　　起死回生，可能也只是轮回棺的一部分能力，若不然，只有起死回生的能力，又怎么能够与造化青莲媲美呢。
<br/>　　元始一纵跃进深渊，丝毫没有顾忌周围向他涌来的无数诡异死亡迷雾，以及那堆积如山的尸骸。
<br/>　　“当——”
<br/>　　一道悠扬而深远的钟声响起，元始头顶出现一个混沌色大钟，大钟将他罩住，钟声一响，群邪辟易，那笼罩着整片空间的诡异迷雾，瞬间便如潮水般退去。
<br/>　　元始走到水潭边，水潭静得可怕，元始更是在其中感受到一股强烈的死亡之意。
<br/>　　“咦！竟然是死亡之力凝结而成，如此多死亡之力，那该死去多少人？”
<br/>　　元始深感意外，他刚开始还以为这只是某种他不知道的冥水，但仔细观察，才知道这水潭之中全部是死亡之力凝聚而成。要知道，死亡之力可不是那么容易凝聚的，数十个人死亡，才可以凝聚一滴，而且还要在特殊环境之中，如若不然，是不可能成功的，刚刚凝聚便会消散，而此处却又如此一大潭……
<br/>　　这么算来，死去的人真不知道有多少？
<br/>　　“这死亡之力，应该是冥界的特产吧，听说死神便是死亡之源凝聚而成的先天神邸。”元始有些不确定道。
<br/>　　元始打量了一会儿，便将目光投向中央小岛上的祭坛以及其上的青铜棺。看了一会儿，元始好似明白了什么，喃喃道：“原来如此，轮回棺起死回生的秘密竟是如此，死之极致便是生，轮回的奥妙，便是生与死之间的奥妙吗？不，恐怕没那么简单。”
<br/>　　“轮回棺，越来越有意思了。”元始伸手一抓，便将那祭坛上的青铜棺抓到手中，同时他也看到了棺内躺着的人，或者说是死人。
<br/>　　“人皇吗？”
<br/>　　元始眼睛一转，便明白了许多。他进入到此界之时，已经是到了中古年代，上古三皇已经证道，五帝开始争锋。那个时候，三皇已经消失了。而随后的五帝随之也一个个消失不见，他们恐怕就是来到了星空之中，为报当年之仇。
<br/>　　只是帝族与圣地的底蕴丰富，更有至尊高手坐镇，报仇不是那么容易的。现在看来，他们失败了，失败的代价，便是死亡。只不过轩辕运气好，得到了一个能够起死回生的轮回棺。伏羲是因为他背后之人不简单，他得到一个不知哪个世界流落下来的七绝琴，将他带到了洪荒。
<br/>　　元始摇了摇头，感叹此界的洪荒真的很惨，神话时代的称霸天地的天庭被灭了，至尊都死了，三皇五帝也都死了，真是……好弱啊。
<br/>　　“咦，死亡之力在消散？”元始轻咦一声，他发现那“水潭”正在被蒸发，他微微思索，就明白了。“原来是轮回棺在汇聚死亡之力啊，如今轮回棺被我取了，便没有东西镇压，这些死亡之力便会消散一空。”
<br/>　　“浪费是可耻的。”元始不知从何处取来一个黑色瓶子，手捏印诀，滔滔死亡之力便被收进黑瓶之中，能够装住死亡之力的瓶子，也非同小可。
<br/>　　收完死亡之力，元始将目光再次投向手中的青铜棺。
<br/>　　“嗯？用轮回棺起死回生原来是这样的道理，在尸体之中重新孕育出一个神魂，取代死去的神魂，新的神魂会吞噬尸体的记忆，这样也算是重活一世了。”元始看见人皇的状态，便差不多看清了起死回生的本质。
<br/>　　“只是，这样的人，还算是原来的他吗？”元始摇了摇头，叹息道。他自是知道这样做是有后遗症的，轻则记忆丧失，重则沦为只会杀戮的魔头。
<br/>　　“咦，不对，这青铜棺有蹊跷，远不止表面那么简单。”元始开始睁开道眼，仔细打量着着青铜棺，良久，他才开口道：
<br/>　　“好算计，真是好算计！竟然想要用此棺培养出一位真正的仙道尸傀！此棺真正作用不是起死回生，而是……如今看来，这棺恐怕都不叫轮回棺。”
<br/>　　“此棺也并非无主，而真正的主人竟然不在这个宇宙，那便是异域了。而且这棺的主人至少也是一名真仙。”
<br/>　　无敌至尊之后是真仙。真仙，是这个世界的称呼，这个世界的真仙意味着跳出命运长河，斩断过去未来，也就相当于洪荒的大罗金仙。
<br/>　　在这个世界，只有道祖与异域的真主达到过此境界，相传当年大战，道祖引开真主，之后双双失踪，恐怕此事没那么简单。如此看来，此棺极有可能是真主的。
<br/>　　至于真主留此棺在星空的目的也很明确，为将来攻战星空做准备。
<br/>　　这一刻，元始看向棺材内的人皇目光不同了，真是个悲催的孩子，死了都要被人算计。
<br/>　　“罢了，看在你也叫轩辕的份上，我便帮你一把，作为报酬，这青铜棺便是我的了。”
<br/>　　元始摇了摇头，伸手将青铜棺内的人皇移送了出来，让他悬于空中，在伸手一招，手中出现一玉净瓶，轻轻倒出一滴碧绿色的液体，这液体出现后，便静静地漂浮在空中，光华流转，就如早晨的露滴。数息时间，光华内敛，没有丝毫的气息流露出来，就好似一滴普通的绿色液滴。
<br/>　　这是生命晨露，生死人，肉白骨不再话下，仅仅一滴，便蕴含了无数的生命力，简直堪比生命树的母体。
<br/>　　这是元始在混沌之中的收获，斩杀了一混沌中的混元大罗金仙级别的木系生灵所得，他也仅仅提炼出十滴而已，每一滴都价值连城。
<br/>　　混元大罗金仙已经跳到另一个生命层次，他的一身精华所化，对于还没有成为大罗金仙的人皇来说，简直是大补。
<br/>　　绿色水滴从人皇眉心没入，仅仅瞬息时间，便扩散到他全身，滋养着他的身体，这可比死亡之力转化而来的生之力强多了。
<br/>　　“还有魂魄。终究是死的时间太长了，麻烦！”
<br/>　　元始十指连动，在人皇的眉心，太阳穴，胸口，心脏，丹田，膻中等地方不断点动这着，并做出一连串玄妙而复杂的动作。
<br/>　　“聚魂！”
<br/>　　元始低喝一声，他十指突然大放光彩，五光十色的流光随着他的十指在动，一股神秘的力量汇聚而来，顺着元始的十指，流入到人皇体内。
<br/>　　最后，元始一指点在人皇眉心处，突然毫光大放，最后又迅速归于平静。
<br/>　　“好了，魂魄新生，需要蕴养，此处是个好地方，你就在这里好好修复吧，百年时间，你应该醒来了，而且实力会随之突破。”
<br/>　　最后，元始将沉睡中的人皇移进了祭坛上，在此处布下禁制，防止外人进来捣乱。
<br/>　　元始抬头看了看远处，一步走出，消失不见。
<br/>　　……
