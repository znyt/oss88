　　九重浮屠塔第九层空间之中，苏莫双眸紧闭，静静的参悟吞噬奥义。
<br/>　　在这里他心神平和，心境空灵，不仅悟性大增，神魂和受损的精神力，也加快了恢复的速度。
<br/>　　苏莫受损的神魂和精神力，一直在缓慢的恢复，但想要彻底恢复，还需要一段时日。
<br/>　　不过，目前而言，这对他的实力影响，已经很小了。
<br/>　　吞噬战魂微微催动，感受着那玄妙的吞噬之力，苏莫不断的参悟，对于吞噬奥义，他领悟的越发精深。
<br/>　　时间在修炼中过的飞快，一日又一日，约莫半月之后，苏莫的吞噬奥义，达到了二阶的后期。
<br/>　　奥义的提升，对于他好处是巨大的，每一次的提升，都能让他在施展吞噬战魂之时，吞噬之力大增。
<br/>　　吞噬战魂是他的根本，是他强大的源泉，只要吞噬之力够强，他的战力就能飙升。
<br/>　　这一次会武，他根本都没有动用吞噬战魂，不然的话，六大势力所谓的妖孽天才，根本不是他一合之敌。
<br/>　　苏莫继续参悟吞噬奥义，不过，奥义的参悟同样是越到后期越困难，而且参悟的速度，几乎是呈直线下降。
<br/>　　一个月的时间眨眼即过，苏莫缓缓的睁开了眼眸，他的吞噬奥义快要达到二阶圆满的地步了，不过，还是稍差一些火候，但是时间已经到了。
<br/>　　这时，不远处的炎澜天、扬天成等人，也纷纷结束了修炼，他们的时间也全部到了。
<br/>　　几人纷纷长身而起，离开了此地，前往族务府领取奖励。
<br/>　　“哼！”路过苏莫身边，扬天成冷哼一声，嘴角露出一丝冷笑。
<br/>　　苏莫摸了摸鼻子，这个扬天成看来还是没有吃够苦头啊！
<br/>　　随即，他便也离开了此地，向族务府走去。
<br/>　　族务府，是太阴族外族之中，处理族务的地方，位于太阴城的中心部位。
<br/>　　苏莫跟着扬天成、炎澜天等人，没过多久便来到了族务府。
<br/>　　族务府很大，是一座古朴大气的府邸，熙熙攘攘的太阴族之人，让得族务府非常的热闹。
<br/>　　在族务府的一座大殿之中，苏莫等人如愿的领取了会武的奖励。
<br/>　　而苏莫作为会武的第一人，领取了一门尊级上品武技，和一门尊级上品武技。
<br/>　　领取了奖励之后，炎澜天、扬天成等人均是离开了太阴城，各自返回，
<br/>　　而苏莫却是没有离开，回到了太阴城之中属于道宗的府邸，因为他还有奖励，那就是虚神境大能的一次指点。
<br/>　　临走之前，族务府的执事交代，让他回去等候，过不了多久，自然会有人传唤他，前去聆听虚神境大能的指点。
<br/>　　苏莫无奈，只能回来等候。
<br/>　　如今，属于道宗的府邸之中，已经人去楼空了，所有道宗的弟子和长老，已经全部返回了道宗，整个府邸中显得冷冷清清。
<br/>　　苏莫回到住处之后，便将族务府奖励的武技和宝器取了出来。
<br/>　　之前他没有仔细查探，现在倒是要好好看看，尊级上品的武技和宝器有多么的强大。
<br/>　　宝器是自由选择的，苏莫选择的宝器自然便是长剑。
<br/>　　这是一柄灰白色的长剑，长约四尺，外表非常的古朴，朴实无华，锋芒内敛，仿若就是一柄石剑。
<br/>　　若不是知道这是一柄尊级上品的宝器，苏莫甚至以为这就是一柄普通的长剑，没有任何的独特之处。
<br/>　　手握长剑，苏莫微微向其内注入了一丝玄力。
<br/>　　霎时之间，仿若是石质的长剑之上，暴冲起凌厉的的剑光，瞬间洞穿了房间的屋顶，冲上了天穹之上。
<br/>　　这剑光凝练无比，锋锐无匹，充满让人心悸的气息。
<br/>　　“好剑！”苏莫眼眸一亮，忍不住的赞扬了一声，这柄剑可比他之前用的剑强大太多了。
<br/>　　两者，根本不是一个级别的存在。
<br/>　　可以说，用了这柄就剑之后，他的实力能增加两到三成。
<br/>　　可不要小看这两三成，对于他现在的修为而言，两三成已经非常的恐怖了。
<br/>　　可以这么说，如果有和苏莫势均力敌的对手，苏莫的实力能突然增加两三成，不敢说秒杀对手，轻易击败对方是轻而易举的事情。
<br/>　　随后，苏莫又取出了一枚玉简，正是族务府奖励的尊级上品武技的玉简。
<br/>　　苏莫是剑客，奖励于他的武技，自然不可能与他修炼之道相佐，所以奖励与他的武技，正是一门尊级上品的剑法。
<br/>　　意念探入玉简之中，苏莫静心查看了起来，这一看便是几个时辰的时间。
<br/>　　几个时辰之后，苏莫的意念退了出来，面带一丝笑容。
<br/>　　玉简内的剑法，名为：七劫剑法，尊级上品的品阶，非常的玄妙强大。
<br/>　　此剑法共有七式，将攻击化为劫难，仿若一剑便是一劫，强大异常。
<br/>　　“很好！”苏莫暗暗的点了点头，他对于这门剑法非常的满意，正适合他现阶段修炼。
<br/>　　他已经很久没有修炼武技了，以前修炼的武技也很少动用，因为威力实在不高。
<br/>　　现在这门武技，应该够他使用很长一段时间了，当然，此武技的修炼难度也是不小，想要练成需要消耗他大量的时间。
<br/>　　少倾之后，苏莫将宝剑和玉简全部都收了起来，现在也不适合修炼，还是等回到道宗之后，再静心修炼吧！
<br/>　　很快他就要去接受虚神境大能的指点，对于此事，他也是极为重视。
<br/>　　虚神境，这是无尽星空中最巅峰的强者，据闻虚神境大能可以遨游太虚，畅游宇宙，寿元悠远无尽，堪称半个神灵。
<br/>　　苏莫从未见过虚神境的大能，对于虚神境大能也是充满了好奇，这次也是他大开眼界的机会。
<br/>　　随后，苏莫便开始静心打坐了起来，凝练玄力，淬炼根基。
<br/>　　时间飞逝，三日后，一名褐袍老者来到了府邸中，找到了苏莫。
<br/>　　这名褐袍老者，正是上次主持会武的褐袍老者。
<br/>　　“苏莫，你现在跟我去内族，接受神长老的指点！”褐袍老者说道。
<br/>　　“是！”苏莫点了点头。
<br/>　　“嗯！随我来吧！”褐袍老者招呼一声，便带着苏莫离开了府邸，向太阴内族而去。
