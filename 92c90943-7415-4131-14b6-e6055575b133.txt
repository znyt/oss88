        童筱颖记得唐倩兮说过，在他们会所里工作的女人，都有艺名，而且还是《聊斋志异》里面各色女主角的名字。
<br/>
<br/>        晚霞，这个名字，在聊斋志异里面的女主角，好像是一只会跳舞的女鬼。
<br/>
<br/>        莫非，这个晚霞姐善舞？！
<br/>
<br/>        童筱颖心里揣测着，并未去询问晚霞的职业。
<br/>
<br/>        晚霞一边走，一边向童筱颖详细地介绍着“夜惑”里的一切。
<br/>
<br/>        “夜惑”里，暂时共有十个持股人，他们管那十个持股人叫“爷”，在这里执事的是费尔南，费尔南打理“夜惑”里所有大小事务，而其他那十位爷则利用各自的人脉关系招揽生意进来以及给这里一个安全保障。
<br/>
<br/>        晚霞说着说着，带着童筱颖穿过舞池后，又去了“千金会”。
<br/>
<br/>        何为“千金会”，即指“春宵一刻值千金”，说得通俗点，就是，买卖纯洁还是处的女人的地方。
<br/>
<br/>        “你还是处吗？”晚霞接着问。
<br/>
<br/>        童筱颖怔了一下，回答道：“我已经结婚了……”
<br/>
<br/>        “那就可惜了，如果你还是处的话。你就能来这里，如果遇到好的买主。娶回去当老婆，也是很幸运的一件事情。这里的买主，全都是有钱人。”晚霞意味深长地说道，“能来‘千金会’里卖的女孩，当然也都是干干净净的女孩。我所指的是身体干净，至于心里干不干净。我们就不敢保证了！”
<br/>
<br/>        “那……那些卖不出去的女孩，会怎么处置？”童筱颖遥望着前方台上，角落里那个刚刚无人问津的女孩子。
<br/>
<br/>        “看她能力喽！如果没什么才艺，要么就会被派去当最底层的服务员。如果，长相过得去，就会被派去学习云雨之事，以后当出台小姐。”晚霞云淡风轻地说。
<br/>
<br/>        童筱颖听晚霞说完，简直大跌眼镜。
<br/>
<br/>        她从未想过，原来这会所竟然还有不为人知的一面。
<br/>
<br/>        其实，晚霞并未告知童筱颖另一件事情。千金会里的女孩，虽然身体上都干净，但是，心里的城府深着了。
<br/>
<br/>        有些女孩，是经过费尔南找专人培养出来的商业秘密师，被那些富豪们买回去后，专门刺探他们公司内部秘密的。
<br/>
<br/>        所以，也有女孩，从地下情人转正，成为了公司的老板娘的例子。
<br/>
<br/>        “夜惑”，作为那些女孩们的“娘家”，她们自然会介绍一些达官贵人来“夜惑”消费。
<br/>
<br/>        “夜惑”为什么会热闹这么多年，还有个原因，也在这里。
<br/>
<br/>        “在‘夜惑’里工作的每个人，无论男女，都是有头儿的。等级高点儿的人，是由费老板直接管的。比如我和小倩，我们的上司，除了那十位爷，剩下的就只有费老板了。不过，你很幸运，虽然你不跟我同级，但是费老板把你交给了小倩，有小倩罩着，‘夜惑’里的人，不会对你怎样。当小倩不在的时候，我就会替小倩关照你。”晚霞接着说道，顿了顿后，她又邪肆地补充了一句，“当然，也有发生意外的时候，我和小倩是不可能百分百保证你的人身安危的。”
<br/>
<br/>        童筱颖微微欠身，礼貌性地问候：“以后，若是给晚霞姐添麻烦，还请晚霞姐见谅。”
<br/>
<br/>        “你最好，不给我添麻烦。走吧！带你去其他地方逛逛。”晚霞丝毫不领情地说。
<br/>
<br/>        童筱颖也不在乎，继续跟在了晚霞的身后。
<br/>
<br/>        当他们逛到KTV会部时，晚霞发话了：“这里的KTV包厢都是VIP包厢，私密得狠，我就没法带你去参观了。”
<br/>
<br/>        “应该和外边的一样吧！”童筱颖下意识地回答。
<br/>
<br/>        就在这时，走道那边传来了吵闹声。
<br/>
<br/>        “季少，对不起！对不起！我不是故意的！对不起！对不起！”
<br/>
<br/>        是一个女人的哭声。
<br/>
<br/>        “把我鞋子舔干净！”
<br/>
<br/>        紧接着，是一个男人的呵斥。
<br/>
<br/>        童筱颖循声望去，那边有几个人围着，但从人缝间，可以看到一个穿着服务员制服的女人正跪在地上抹眼泪。
<br/>
<br/>        “快舔！”男人接着喝斥。
<br/>
<br/>        女人抽泣着，身子微微往下匍匐。
<br/>
<br/>        童筱颖皱了皱眉头，刚想走过去替那女人解围，却被晚霞一手给拦住。
<br/>
<br/>        “在‘夜惑’，不要当‘圣母’！这里，‘圣母’可是活不下来的！”晚霞早就见怪不怪了。
<br/>
<br/>        童筱颖看了晚霞一眼，再次回过头去看向那个女人时，那个女人匍匐在地上，真的在舔一只高档的暗红色皮鞋。
<br/>
<br/>        “别看了，走吧！以后，你也得给我小心点。”晚霞抓住童筱颖的手腕，拉着她往走道的另一边走去。
<br/>
<br/>        “以后，这里是你的专属化妆间了。前面是工作区，柜子后面有张床，是你休息的地方。”不知何时，晚霞拉着童筱颖来到了一张敞开的房门口。
<br/>
<br/>        童筱颖缓过神来，跟着晚霞走进了房间。
<br/>
<br/>        房间里，有梳妆台，同时还有两排不同风格的服装。
<br/>
<br/>        “这里的衣服，是免费提供给你上台驻唱时穿的，但是你要好好保管着，别弄坏了。知道了吗？”晚霞说着，又把一张卡递给了童筱颖，“这张是餐卡兼房卡以及进会所的门禁卡，下班后，可以去负一层的员工餐厅吃点宵夜。”
<br/>
<br/>        “好的，谢谢。”童筱颖接过房卡后，微微颔首。
<br/>
<br/>        晚霞又说道：“你隔壁的那几间，是几个管事的男职员居住。所以说，这里并不安全。晚上你若想在这里过夜，你最好记得把门反锁了。”
<br/>
<br/>        “谢谢晚霞姐的提醒。”童筱颖欠身道。
<br/>
<br/>        晚霞瞥了童筱颖一眼，再次警告道：“给我记住了！在这里，遇到别人的事情，不要多管闲事，不要去插手，不要试图当圣母去拯救别人！”
<br/>
<br/>        “明白。”童筱颖点了点头。
<br/>
<br/>        晚霞转身便离开了。
<br/>
<br/>        而这叫晚霞的女人前脚刚走，隔壁房间里又闹出了事端。
<br/>
<br/>        童筱颖刚准备进屋，只听到隔壁房间里传来一个熟悉的声音：“主管，我求求你。不要扣我工资好不好？”
<br/>
<br/>        “不扣你工资，你就不长记性！谁叫你得罪的人是季少！”主管疾言厉色地呵斥。
<br/>
<br/>        “是他故意的！呜呜——”女人哭泣地反驳。
