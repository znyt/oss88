　　“逸儿，你太冲动了。”三长老满脸有忧愁之色。
<br/>　　各个族人已经离去，但三长老和萧逸仍旧站在原地。
<br/>　　三长老道，“五长老摆明了就是想趁今天各个族人都在此，趁机宣布这个提议，好对付你，废除你的少家主之位。”
<br/>　　“只要你今天不答应，拖下去，等族人们都离开，他就拿你没办法了。”
<br/>　　三长老怪责着萧逸太过冲动。
<br/>　　萧逸摇了摇头，道，“这对父子太阴险了，拖得一次还会有下次，倒不如一次断绝了他们所有恶毒念头。”
<br/>　　“至于三日后的挑战，说白了，只不过是我与萧若寒一战。对付萧若寒，我有十足把握。”
<br/>　　“十足把握？”三长老皱了皱眉头，随即呵斥道，“逸儿，虽然你这些天大放异彩，但可千万不能有骄傲自负的念头。萧若寒踏入凡境七重已久，如果我没有猜错的话，他在紫云洞修炼十天，现在必然已是凡境九重。”
<br/>　　“而你呢，十天前不过凡境五重，如今顶天了就是七、八重左右，如何能是他的对手。”
<br/>　　“三长老请放宽心。”萧逸自信地说道。
<br/>　　“你让我如何放宽心，哎。”三长老唉声叹气起来。
<br/>　　“萧重叔叔。”萧逸忽然叫了一句。
<br/>　　“额。”三长老忽然愣了愣，印象中，自从家主失踪，萧逸被所有族人瞧不起后，已经很久没有这样亲昵地称呼自己了。
<br/>　　萧逸认真地说道，“逸儿已经让你失望了很多年，从今往后，再也不会让你失望了。”
<br/>　　那声萧重叔叔，萧逸叫得很认真。或许从灵魂角度来说，他和萧重接触的时间不足一月，但这一月内，这个萧重叔叔为自己所做的一切，和那份浓郁的关心，都值得自己尊称他一声叔叔。
<br/>　　萧重表面上似乎没为自己做什么，但萧逸心里明镜般，这些天，他暗地里为自己做了很多很多，只不过他没说罢了。
<br/>　　最明显的，萧若寒父子，胆敢请杀手潜入萧家杀自己，可想而知他们肯定会有很多后续的毒计。
<br/>　　只不过，这些毒计，必然都被萧重在暗地里全部解决了。
<br/>　　三长老听着萧逸的话，忽然虎躯一震，有些情绪莫名。
<br/>　　“哎，好吧，既然你心意已决，我也随你。”三长老叹了口气。
<br/>　　“那逸儿就先行告辞了。”萧逸笑了笑，转身离开。
<br/>　　三长老萧重看着萧逸离开的背影，沉声道，“放心吧逸儿，我不会让那些家伙得逞的。”
<br/>　　说罢，他也转身离开，不过却不是回家，而是往炼药堂而去，他要找大长老。他决定了，不惜一切代价，那些试图对付萧逸的阴谋，一个都别想成功。
<br/>　　.....
<br/>　　萧逸回到家中，迎来的是依依，以及满桌诱人的饭菜。
<br/>　　“我不在的这些天，想我没？”萧逸打趣道。
<br/>　　“想，当然想，依依很想少爷。”依依眨着水灵灵的大眼睛，高兴地说道。
<br/>　　“呵呵。”萧逸笑了笑，看到如今依依没有像以前那么拘束，那么怕自己，自己心中一阵喜悦。
<br/>　　不知从何时起，萧逸发现，一向习惯独来独往的自己，竟慢慢爱上了那种有人陪在身边一起吃饭的感觉。
<br/>　　当然了，这个人仅限于依依。
<br/>　　虽然不是什么山珍佳肴，但每次和依依一起吃饭，都会自己的心格外的宁静与安然。
<br/>　　吃过饭，依依收拾碗碟，萧逸则开始了修炼。
<br/>　　对于萧逸来说，修炼是一件有趣的事，当然，也是为了应付三天后的战斗。
<br/>　　从怀中拿出乾坤袋，那里有着五长老换萧若寒修炼名额所给的银两和丹药。
<br/>　　五长老老谋深算，一心想要让萧若寒在紫云洞内修为大涨。
<br/>　　而萧逸一番阻挠，让萧若寒无法凝结气丹，算起来，萧若寒仅仅提升了一重修为罢了，五长老这算是陪了夫人又折兵。
<br/>　　有这些银两，萧逸可以买更多的材料回来炼制丹药，后天丹也可以现在就吞服。
<br/>　　至于那些先天丹，却暂时不会服用。毕竟这是先天境武者服用的丹药，炼制的材料都是些药力极猛的药材，蕴含庞大的力量，相应的，那些杂质也很难应付。
<br/>　　起码对于凡境武者的肉体来说，这些杂质很难排出去。
<br/>　　别到时候杂质堵塞了丹田，再也无法吸纳天地灵气，那就更麻烦了。
<br/>　　所以，先天丹会等到他突破到后天境之后再服用。
<br/>　　另外，说起钱这事，萧杰和萧石可还欠他的钱呢。
<br/>　　不过，自从大半个月前，他威胁了这两人后，这两个家伙就彻底躲了起来，比老鼠见到猫还要更怕，萧逸也找不到他们。
<br/>　　而后来，家族比武刚刚结束，也就是十天前，那些无缘进入紫云洞修炼的子弟，已经全部被派出去历练了。
<br/>　　萧家乃紫云城三大家族之一，本身有很多产业，还有各种赚取家族收入的来源。家族子弟们都被提前派到这些地方了，起码要历练一年半载才会回家族。
<br/>　　这时，萧逸看了眼刚刚收拾好碗筷的依依，笑道，“依依，我教你修炼好不好？”
<br/>　　“哦。”依依边收拾着碗筷，边一如既往的点头答应。只要是萧逸说的事，无论对错，无论如何，她都会习惯性的点头。
<br/>　　下一秒，她才反应过来，愕然道，“阿？少爷您说什么？”
<br/>　　那天然呆的样子，让萧逸一阵好笑。
<br/>　　“我说教你修炼呢。”萧逸重复了一遍。
<br/>　　“修炼？”依依连忙摇头，道，“依依不想，我听说武者修炼是很麻烦的事呢，而且要浪费大量时间，依依想好好照顾少爷。”
<br/>　　“照顾个屁，我又不是没手没脚的，好了，就这么定了。”萧逸故作严肃，敲定了事情。
<br/>　　......
<br/>　　按照萧逸的指点，依依盘膝坐到了床上，踌躇道，“少爷，依依很笨的呢，我...”
<br/>　　“闭嘴，照做就行。”萧逸呵斥了一声，他知道依依不是笨，就是不想修炼，想将时间都放到照顾他身上。
<br/>　　“首先，武者修炼要吸纳天地灵气入体内，而后转入丹田...”萧逸耐心地指导着。
<br/>　　依依如今没有觉醒武魂，但这并不代表她不能修炼，只不过吸纳天地灵气的速度会非常慢罢了。
<br/>　　觉醒武魂，主要靠天赋。
<br/>　　萧逸也想不明白，按他的观察，依依算得上冰雪聪明，绝不可能没有天赋，但她就是一直都没有觉醒，也不知怎么回事。
<br/>　　“丹田是哪里阿？”依依故作不懂，硬是不想学习修炼知识。
<br/>　　萧逸翻了个白眼，没好气道，“腹下一寸，肚脐边缘。”
<br/>　　“不懂。”依依摇摇头。
<br/>　　“少给我装模作样。”萧逸来气了，直接将手按到依依的丹田位置，道，“就在这里，记清楚了。”
<br/>　　“天地灵气入体，会在四肢百骸中游走一个大周天，最后直达丹田。”
<br/>　　“游走过程，灵气会自动消耗，用于淬炼肉体，强化肉身。”
<br/>　　“而最后到达丹田的剩余灵气，才会被丹田吸收，用于提升修为。”
<br/>　　萧逸一边说着，一边用手指在依依身上游动，模拟出灵气的大概游走方式。
<br/>　　“额。”依依愣了愣。
<br/>　　“额什么额。”萧逸刚要呵斥依依不专心，却发现依依脸颊通红，羞涩地低下了头。
<br/>　　萧逸在专心指导，殊不知，自己的手指刚才几乎把依依的身体游走了个遍。
<br/>　　依依不过是个15岁的少女，当然会不好意思。
<br/>　　“那个...我...”萧逸反应了过来，摸了摸鼻子掩饰自己的尴尬。
<br/>　　“没事的少爷。”依依抢先说道，“我知道少爷只是为了依依好，在教依依修炼呢，少爷继续吧。”
<br/>　　依依本身就是个美人胚子，身体变好后，更是变得国色天香。
<br/>　　加上她那乖巧可人的性格，还有天真无邪的面孔，简直就如祗落凡尘的仙女，让人不忍心去亵渎。
<br/>　　而自己‘非礼’了她一遍，她还善解人意地来一句‘没事的，请继续’，这对男人来说简直是一种无法抵抗的诱惑。
<br/>　　哪怕是以自己一代形意拳宗师的定力，杀手之王的冷酷，此刻也是一阵心猿意马。
<br/>　　“咕”萧逸不禁咽了口口水。
<br/>　　特别是如今自己靠得那么近，闻着依依身上那淡淡的处子体香，更是让自己的内心蠢蠢欲动。
<br/>　　“不行。”萧逸猛地心头一震，暗骂一声，“萧逸你个禽兽，依依还是未成年少女呢，你乱想个什么龌蹉事情，哼。”
<br/>　　心里骂完自己后，萧逸立马跳下床，远离依依。
<br/>　　“少爷，怎么了？”依依吓了一跳。
<br/>　　“没事。”萧逸故意板着脸，掩饰自己的尴尬脸色，喝道，“继续修炼，别来烦我。”
<br/>　　说罢，萧逸立马转过身，盘腿坐下修炼。
