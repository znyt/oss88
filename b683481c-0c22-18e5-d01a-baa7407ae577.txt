        “当然是救人的法子啊！你又不是不知道，玄灵从小就不能淋太久的雨，淋久了，人会……”
<br/>
<br/>        “行了行了！我照做就是！”裴金澈一脸不耐烦地打断了连华生的话，挂了电话后，他又急躁地命令司机，将后座的隔板升起来。
<br/>
<br/>        夏玄灵的脸色惨白，就连昔日的红唇也失去了那诱人的色彩。
<br/>
<br/>        裴金澈的喉咙滚动了一下，修长的指尖，正将夏玄灵的衣摆往上推。
<br/>
<br/>        为了不让自己胡思乱想，他选择了闭上了眼睛，替她脱去衣服之后，将她紧紧地搂在了自己的怀中，并用自己的西装外套将她孱弱的身子包住。
<br/>
<br/>        到了连华生的私人休息室里，连华生给夏玄灵打了针，并让裴金澈好生照顾着。
<br/>
<br/>        “我说你也真是的，傲娇也要有个度。玄灵心心念念地就是她的咖啡店，你有这个能力让她的咖啡店起死回生，可你偏偏不帮她。”连华生一边收拾着医药箱，一边碎碎念叨。
<br/>
<br/>        裴金澈坐在了床边，看了一眼躺在病床上的夏玄灵后，转开了话题：“傅氏集团里，有一个自称是老大的女人，她说她看到了我的自愈能力，你帮我去消除她的记忆。”
<br/>
<br/>        “阿沉的女人？童筱颖？”这是连华生的第一反应。
<br/>
<br/>        裴金澈诧异地看着连华生：“你认识？”
<br/>
<br/>        “当然认识。这件事情，你就放心吧！她肯定不会说出去。”连华生会心一笑。
<br/>
<br/>        裴金澈剑眉微蹙，若有所思地说道：“我怀疑……她也是实验室里出去的女人。”
<br/>
<br/>        “她不是。”连华生笃定道。
<br/>
<br/>        裴金澈不以为然地问：“你验证过？”
<br/>
<br/>        “那到没有。只不过，她是阿沉的女人。连根头发都碰不得，你要我怎么验？再说了，阿沉说她不是，她就得不是。”连华生意味深长地笑了笑，收拾完东西后，又叮嘱道，“你还是好好照顾玄灵吧！”
<br/>
<br/>        裴金澈凝眉，目送连华生的离开。
<br/>
<br/>        再回头时，夏玄灵柳眉微蹙，像是在做噩梦。
<br/>
<br/>        裴金澈俯过身去，抚平了夏玄灵的眉心。
<br/>
<br/>        他一直陪护在她的身边，连公司都未回。
<br/>
<br/>        直到傍晚，夏玄灵终于醒了过来。
<br/>
<br/>        “终于醒呢？”裴金澈就坐在一旁的单人沙发上，双手抱臂，翘着二郎腿，目光不带任何感情地凝望着她。
<br/>
<br/>        夏玄灵支起身子坐起来，眸光黯然，淡淡地说：“我决定不再求你了。比起求你，我还不如去求夜沉哥。”
<br/>
<br/>        “你夜沉哥已经有女人了，你还不死心吗？”裴金澈讥笑道。
<br/>
<br/>        夏玄灵抬眸瞥了裴金澈一眼，三年过去了，他倒是一点都没变。
<br/>
<br/>        除了对傅夜沉唯命是从以外，这男人从骨子里透着一丝生人勿进的冷冽，甚至浑身带着荆棘，谁靠近就刺谁。
<br/>
<br/>        夏玄灵背靠着床头板，淡然地笑了笑：“是啊！夜沉哥都有女人了，我怎么还不死心呢？你说我为什么还不死心？你倒是说说，我跟夜沉哥无法走到一起，是被谁害的呢？”
<br/>
<br/>        裴金澈笑容渐敛，双手撑着扶手，从单人沙发上起身，踱步走到了夏玄灵的跟前，一把捏住了她的下巴，将她的脸靠近自己的面庞前。
<br/>
<br/>        “老大现在在法国忙他的海外市场拓展，压根就没空管你这芝麻点大的破事。你想让我帮你把咖啡店起死回生，也不是不可以。你给我当姘头，我就免费拯救你那咖啡店。”裴金澈手下的力度重了点。
<br/>
<br/>        夏玄灵生痛地皱了下眉头。
<br/>
<br/>        姘头是什么？
<br/>
<br/>        呵！
<br/>
<br/>        给这男人免费发泄生理需求的工具。
<br/>
<br/>        反正已经被这男人折磨过一次了，也不在乎再被他折磨了。
<br/>
<br/>        “我需要一个时间限度。”夏玄灵淡定道。
<br/>
<br/>        裴金澈勾了勾唇：“一年。与此同时，我会让你的咖啡店达到巅峰的业绩。”
<br/>
<br/>        “好。一年之后，我们桥归桥，路归路。”夏玄灵同意了。
<br/>
<br/>        “那么，就从今晚开始。”裴金澈似笑非笑，冷冷地说道，紧接着从西装裤兜里掏出了一把钥匙，拉起夏玄灵的手，放在了她的手心上，“这是我公寓的钥匙，第一晚，可别让我失望。”
<br/>
<br/>        夏玄灵握住钥匙，淡然地笑了笑。
<br/>
<br/>        窗外的雨，还在下……
