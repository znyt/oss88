　　自己将来吐纳的时候，倒是可以尝试一下，如果真的能将体内的戾气完全炼化……叶少阳相信，自己的实力绝对能提升一大截，甚至突破上仙牌位也不是没可能。
<br/>　　他自己清楚，自己体内蕴含了多少戾气……曾经几次被逼上绝路，戾气爆发，叶少阳靠这个战胜过许多强大的对手。
<br/>　　虽然，他根本不知道自己体内的戾气是从哪里来的。直到现在，这都是一个谜。不过，这并不妨碍他将他收为己用。
<br/>　　叶少阳爬起来，发现毛小方坐在地上正在吐纳，于是不打扰他，等他结束之后，睁开眼睛，看到叶少阳，笑了笑说道：“多亏你，我现在提升得很快。”
<br/>　　叶少阳犹豫了一下，并没有把有关大周天吐纳心法能巩固丹田的事情告诉他，他生怕这不是一条修行的正道，或者说，自己是因为先天灵体的缘故，才能够承受这种方式的修行，他生怕毛小方万一出什么意外，后悔都来不及。反正他现在这样，已经提升得很快了，早晚能触及瓶颈，到那个时候，因为有较长时间的沉淀，对于应劫和突破瓶颈反而有好处。
<br/>　　叶少阳讲了自己被碧清抓住之后所经历的事情，但是并未提到自己恢复了实力……他临时做了一个决定，要扮猪吃老虎，倒不是故意想这样，而是自己既然被人认为是弱者，那也有弱者的好处，至少不会被重点关注。
<br/>　　毕竟这不是属于自己的时代，被关注过多，自己在这个时代留下的痕迹也会越多，叶少阳总觉得这样做有些不妥，更不要说他只想当一个小透明，不被人打扰，能够专心调查徐福的下落，争取早一天回去。
<br/>　　对于毛小方，其实是不用保密的，叶少阳只是怕他提前知道，到时候面对别的人，装的不像，被人看出破绽。
<br/>　　“照这么说，你回到古墓也没看到碧清。不知道她去了哪里？”
<br/>　　“不知道，反正这件事到此为此，也不用再调查下去了。”
<br/>　　毛小方皱眉说道：“不管她了么，这邪物好生了得，万一到处害人怎么办？”
<br/>　　叶少阳苦笑：“第一，她并不是被古墓封印起来的，至少蛰伏起来修炼，如果她想随便杀人，早就不知道杀了多少。”
<br/>　　像碧清这种邪物，叶少阳见过也有不少，他们留在人间就是为了修炼，或者完成某个目的，一般不会滥杀无辜，倒不是有什么怜悯之心，而是不屑，多杀人对他们只有坏处没有好处，还容易被法术界盯上。
<br/>　　靠杀人之类的方法来修炼，虽然修为提升很快，不过对于邪物来说也是饮鸩止渴，除非是没有办法的，或者苦大仇深，无所顾忌，否则一般不会这么干，有很多高端的邪物，也不屑于这么干。
<br/>　　“第二，就算我们找到她……除了去送人头，我想不到还有什么动机。”
<br/>　　毛小方怔了一下，喃喃道：“说的也是啊。我们不是她对手。”
<br/>　　“所以啊，这件事我们管不着，陈晓宇他们回到法术公会，一定会把这件事上报，至于法术公会怎么处理这件事，是他们的事了，跟我们无关。”
<br/>　　毛小方点点头。“那我们现在要做什么？”
<br/>　　叶少阳想了一下，道：“之前跟道渊约好，在这里等他，那就继续等，我估计他也快回来了。”
<br/>　　“好吧。”毛小方站起来伸了个懒腰，“你睡好了，那我去睡了。”
<br/>　　走到门口又回来，找叶少阳要点钱去支付房费。毛小方身上是没什么钱的，知道叶少阳身上有，也不跟他客气。
<br/>　　叶少阳现在确实也是土豪，摸了几块大洋给他。
<br/>　　毛小方把银元在手里颠了颠，突然想到什么，眼睛放出光来，对叶少阳说道：“话说，那座古墓下面的藏宝库……咱们要不要去取点上来？别的就算了，那些黄金倒是可以拿上一点。”
<br/>　　叶少阳道：“你一个法师，捉鬼降妖就好了，要那么多钱干什么。”
<br/>　　“捉鬼降妖，跟这个不耽误吧，身上有些钱，路上行走也方便一点，你以为如何？”
<br/>　　叶少阳不是不贪财，相反，他比一般法师都贪财，只是……自己根本就不是这个时代的人，就算得到再多的财富也带不走，所以对毛小方的提议不是很有热情，不过他说的也没错，多弄点钱在身上，将来或许会有用处，于是答应明天准备一下，一起下墓。
<br/>　　毛小方走之后，叶少阳坐在床上，把这些事情又想了一遍，脑海中最后只剩下梧桐的影子，梧桐……跟冷玉到底有什么关系？
<br/>　　叶少阳想了又想，觉得自己在回到现代之前，一定要把这个问题弄清楚。
<br/>　　猛然间想到什么。叶少阳一拍脑门，差点忘了！
<br/>　　叶少阳把阴阳镜拿出来，刚把上面的符印抹去，就看见一道人影飞了出来，正是美华，看到叶少阳好端端的，又看了看周围，发现是在客栈房间，这才松了一口气。“老大，你没事了？之前为什么不让我出去啊，害我胆战心惊等到现在！”
<br/>　　叶少阳笑了笑，跟她聊了几句，道：“你之前跟我说什么来着，那个小家伙，在哪里？”
<br/>　　“哦，还在里头，被我绑起来了，一直叫嚷，太烦人了。”
<br/>　　“行你把它弄出来，我看看什么情况？”
<br/>　　美华钻进阴阳镜，过了一会出来，手里提着一个小男孩。胖乎乎的，才几岁的样子，全身上下被一道白色的气息像绳子一样捆住，有点委屈地看着叶少阳。
<br/>　　叶少阳双手叉腰，细细打量，说道：“你是谁？为什么要溜进我的阴阳镜？”
<br/>　　“呜呜……”小男孩口中发出奇怪的声音。
<br/>　　“什么情况，你是哑巴？”
<br/>　　“我给他下了禁制，他不能说话。”美华抢着回答道。
<br/>　　“哦哦，给他解开。”
<br/>　　美华上前解开禁制，小男孩能说话了，立刻叫了一声：“主人，我错啦！”
<br/>　　叶少阳吓的差点摔倒。
<br/>　　“我靠，你这倒是不含糊，上来就认亲，我说，你到底是谁，不对，应该说，你是什么？”
