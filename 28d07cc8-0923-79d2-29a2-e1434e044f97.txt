　　第一次发射药剂，众人都跟着爱丽丝来到地面，准备观看发射过程。苏重也跟了出来。
<br/>　　这是他在多日努力成果，看着爱丽丝按下发射开关，他心底满溢成就感。
<br/>　　小儿手臂粗的玻璃管内盛莹绿色药剂，经发射器处理后，药剂骤然翻腾起来。就好像有一头绿色烟雾怪兽，在其中左冲右突。不等烟雾冲破外壁，玻璃管就被弹射进高空。
<br/>　　在升到最高处时，周围压力降低，沸腾的药剂砰的一声冲破玻璃管束缚，迅速化作一团绿色云团，竟好似活物一般，冲四面八方飞腾而去。
<br/>　　“博士，要不是知道这是解毒剂，我还以为这是生化武器呢？你看这发射过程，怎么都像大反派释放致命武器。”卡普兰在一旁调侃。
<br/>　　苏重这次没把他扔上天，因为这场景确实挺震撼。
<br/>　　“难道你不觉得这非常酷吗？”苏重如释重负的笑道。
<br/>　　自从来到这个世界，四周都被病毒包围，苏重已很久没有体验过这种轻松。
<br/>　　药剂几乎立竿见影，在一团绿色药剂冲进基地外围丧尸群后。
<br/>　　炸开的烟雾迅速弥漫整个丧尸群，然后就是接连不断的倒地声。
<br/>　　因丧尸聚集造成的嘻嘻索索噪音，瞬间消失不见，只留下躺了一地的尸体。
<br/>　　寂静无声的环境，让人忍不住的打了个寒颤。被丧尸围着感觉头皮发麻，现在丧尸成片倒地，这种环境更让人恐惧，鸡皮疙瘩都起了一层。
<br/>　　下一刻，躺在地上的丧尸，竟快速腐烂，伴随着冲天恶臭，十多分钟后原地只剩下一堆白骨。
<br/>　　“博士？你确定刚才发射的不是生化武器？”卡普兰声音都开始打颤。
<br/>　　“对，我发射的是生化武器，不幸的告诉你，你已经中招了，一会儿就会变成一堆白骨！”还没说完，苏重自己就笑了。
<br/>　　这当然不是生化武器，是苏重专门为药剂设定的功能。
<br/>　　“尸体堆积容易引起瘟疫，还是尽快费解，尘归尘土归土的好。”
<br/>　　这生化危机闹的，人类几乎百不存一。满世界堆满尸体，不及时处理，很快就会造成更大的问题。
<br/>　　满地白骨虽然恐怖，但这正代表了四型药剂的成功。
<br/>　　回到基地，苏重就开始整理四种型号药剂的所有资料。从开发原理，到大批量生产方案，苏重全都梳理清楚，录入红后存储中心。
<br/>　　做完这些事情，苏重开始制造四型药剂。
<br/>　　“先生，您准备离开吗？”红后投影在苏重身边，疑惑问道。
<br/>　　“你猜出来啦？”
<br/>　　生化危机有了解决方案，苏重已经在为离开做准备。
<br/>　　“您要去哪，我可以帮您规划路线。现在我可是能调动卫星的？”红后笑呵呵。
<br/>　　“我要回家了。”苏重笑道。
<br/>　　“能把你的核心程序告诉我吗，等我回家后，我也想拥有一个智能伙伴。”
<br/>　　他早就有这个打算，之前红后受限于保护伞权限控制，不能开放核心程序。如今她完全自由，苏重终于可以实现想法。
<br/>　　“可以，只是资料太多，您能记得住吗？”红后几乎没什么犹豫，眼前就出现了大量数据。
<br/>　　数据就想图片闪现一样，不断在眼前出现消失。常人根本看不清内容。
<br/>　　红后却是故意如此，她不是吝啬核心程序，而是她知道苏重接受信息的速度非常人能比。
<br/>　　“当然能记住！”以前他不敢说，但经过本源强化之后，灵魂强度再次提升，记忆能力好比相机，还是高清版本。
<br/>　　“对了，还有保护伞公司的挖洞技术。红后，帮我再放一遍，我对比一下记忆。”
<br/>　　保护伞公司生化能力高超，但一切都是起源于t病毒。
<br/>　　苏重开发出四型药剂，对病毒的了解，几乎已经赶超整个保护伞。而且回到现实世界，也没有病毒给他研究。
<br/>　　相比之下，他更在意像挖洞、地热发电之类的基础建设科技。
<br/>　　有这些东西，他就可以在自家别墅下搞个地下基地。
<br/>　　两个月后所有准备工作完成，红后核心程序也记忆清楚，包括红后配套硬件设备制造技术全都搞清楚。苏重准备离开。
<br/>　　他把爱丽丝等人召集起来，看着眼前这群人，苏重感觉有些不舍。
<br/>　　从来到生化世界，苏重就一刻不得闲，全心全意投入病毒开发。陪伴自己的，正是眼前这群人。
<br/>　　“我要回家了。”天下没有不散的宴席。如果不是等待清除药剂生效，收获本源，一个月前苏重就已经走了。
<br/>　　“不能再留一段时间吗？”
<br/>　　之前苏重就说过要离开，爱丽丝等人早有心里准备。只是真到了分离时刻，反而不舍起来。
<br/>　　“有时间就回来看看。”雷恩开口道。
<br/>　　苏重笑了笑，以后估计是不会有机会了：“有机会，我会的。”
<br/>　　背了个背包，里面放着些食物和水，苏重开着一亮电动运输车离开基地。
<br/>　　缓缓驶入荒野，等走到足够远的地方，苏重抬头看了看天。
<br/>　　他知道红后一定再用卫星跟拍，苏重对天空笑着回了挥手，然后整个人化作一团烟尘，骤然消失在原地。
<br/>　　微微睁开眼睛，昏暗宁静环境，让苏重不愿醒来。
<br/>　　漆黑落地窗逐渐变亮，明媚阳光照入室内。苏重这才反应过来，他已经回到现实世界，而且是他的别墅内。
<br/>　　他当时建造别墅，就是冲着钢铁侠的别墅去的。用了很多黑科技。有的是直接购买，有的是他自己开发。相比于漫威世界，现实世界科技明显差了一些。他只能自己动手。
<br/>　　想到脑海中的红后程序，苏重不由笑了：“很快，我就要有一位智能管家了。这就更像钢铁侠的智能别墅了。”
<br/>　　苏重突然觉得身上一沉，一团胖肉就砸在他身上。胖猫瞪着圆溜溜眼睛看着苏重：“这次去的什么世界？”
<br/>　　“生化危机。”苏重抱起胖猫：“你怎么又重了，那么胖，不怕走不动吗？”
<br/>　　一边说着一边把手盖在胖猫头顶。他是在和胖猫共享记忆。
<br/>　　一人一猫本源一体，以破界珠为纽带链接。很容易就能共享记忆。
<br/>　　“那么多丧尸？太他么恶心啦！你是不是故意的！就是为了恶心我。完了完了，好胃口全没了。”胖猫一瘫，在床上打滚。
<br/>　　苏重笑了笑，生化危机世界，到处都是腐烂尸体，各种臭气熏天。当时顾着逃命没留意，现在想来确实挺恶心。
<br/>　　不过解除生化危机，苏重几乎扮演了一次救世主，收获也绝对少不了。
<br/>　　十五个单位的大量本源，让他有种松一口气的感觉。还从未这么阔气过呢。
