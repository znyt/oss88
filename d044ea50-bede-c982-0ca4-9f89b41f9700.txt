　　地下室没有气窗，也没有灯光，唯一的光源来自于挂在墙壁上那两排液晶屏幕。
<br/>　　长时间处于昏暗环境下，在突然面对强光时，人的眼睛会在短时间内难以适应，这适应的时间可能只有几秒钟的时间。
<br/>　　生与死的决断往往只需要不到一秒的时间！
<br/>　　布克尔特从一开始就藏在佟千雪背后没有露头，所以他并不知道罗昊手中有强光战术手电，当白色强光照来的时候，布克尔特的眼睛眯了眯，罗昊抓住了布克尔特眯眼的瞬间，果断的扣动扳机。
<br/>　　砰！
<br/>　　uSP.45手枪套筒后退，抛壳窗打开，一颗金黄色的弹壳飞出，翻滚着飞向一旁，橘红的膛口焰中，一颗.45口径的ACP弹旋转着飞出枪膛，击向布克尔特。
<br/>　　噗嗤！
<br/>　　子弹击中了布克尔特右侧眉骨的位置，直接穿过了他的脑袋，从他后脑勺穿出，打在他背后的墙壁上，墙壁上留下一个带有点点血迹的弹孔。
<br/>　　血从布克尔特右侧眉骨上的那个弹孔中涌出，顺着他的脸颊滴淌下来，就连佟千雪的衣服上也是被沾上了一些血迹。
<br/>　　当啷！
<br/>　　布克尔特握在手中的P226手枪掉落在地上，紧接着他自己也是倒在了地上。
<br/>　　“啊——！”
<br/>　　佟千雪尖叫着想要跑开，可是发软的双脚却是让她差点跌倒在地。慕容飞眼疾手快直接冲上去，一把抱住了佟千雪。
<br/>　　砰！砰！
<br/>　　齐凡对着倒在地上的布克尔特又补了两枪，才上去把布克尔特掉在地上的P226手枪踢开。
<br/>　　“确认死亡！”齐凡对着罗昊说道。
<br/>　　慕容飞抱着佟千雪，轻声安慰着她。感觉到熟悉的气息，佟千雪的情绪也逐渐平稳下来，脸埋在慕容飞胸口，突然低声抽泣起来，“我不应该不听你的话，执意要出来旅游的；如果我听你的话，乖乖的待在家里这一切都不会发生，我们的孩子……也不会……”
<br/>　　提起失去的孩子，佟千雪哽咽着再也说不下去。
<br/>　　慕容飞抱着佟千雪的手不自觉的收紧，但他不会去责怪佟千雪太多，就算这次佟千雪不去T国，没有遭遇到这次绑架，布克尔特这个威胁还是存在的，他还是会因为西伦科特的事情而策划一系列报复行动。
<br/>　　“没事了，没事了。”慕容飞轻抚着佟千雪的后背安慰道：“一切都结束了，我们还会有孩子的。”
<br/>　　罗昊在地下室中找了一遍，并没有找到可以存放P病毒的地方，低声说道：“奇怪，布克尔特居然没有把P病毒放在这里。”
<br/>　　“会不会在那个村子里？”林晓阳问道。
<br/>　　“应该不会。”罗昊摇头，回答道：“P病毒是布克尔特最后的底牌，如果他真的放在了那个村子里，那么他在离开村子的时候，就一定会带走，甚至他就算不带走佟医生也会拿走P病毒。”
<br/>　　“那就是他存放在了别的地方。”齐凡看了眼地上布克尔特的尸体，说道：“可惜他死了，不能从他嘴里问出点什么东西来。”
<br/>　　“就算他没死，要想从他嘴里撬出P病毒的下落也几乎为零。”
<br/>　　罗昊不认为像布克尔特这种人，随便给他用点刑就能从他嘴里逼问出P病毒的下落。罗昊再次走到布克尔特身边，在他身上摸索着，最后找到了他放在迷彩服上衣口袋中的手机。
<br/>　　“这家伙脑子没问题吧？”林晓阳看着罗昊拿在手里的手机，说道：“有了卫星电话，还要手机干什么？用来打游戏？”
<br/>　　“这部手机被他很妥善的收藏起来，应该有特别的用处，等出去以后再想办法研究清楚。”
<br/>　　罗昊没有立刻就研究从布克尔特身上搜出来的这部手机，把手机放进迷彩裤侧边口袋中，现在他们所需要解决的是怎么从这个地下室突围出去，看情况，外面的人应该很快就在地下室的脱氧钢板门上完成炸药的安装，实施爆破，然后对地下室发起突袭。
<br/>　　“大不了跟他们拼了！”周立拽了拽他挂在身上的轻机枪，说道。
<br/>　　“你别傻了。”
<br/>　　陈应天对周立说道：“真是头脑简单四肢发达，如果外面的人知道布克尔特已经死了的话，只要丢几颗手雷下来，我们就是团灭，一个都跑不了，我们是来营救人质的，不是来送死的！”
<br/>　　“狂龙，你说应该怎么办？”梅庆伟对罗昊说道：“我们都听你的，如果你也同意跟他们硬碰硬，我们绝对没有二话，眉头都不会皱一下。”
<br/>　　“跟外面的人硬碰硬只是下策，而且就算是硬碰硬也会造成很大的伤亡。”
<br/>　　罗昊摇了摇头，显然是不主张跟外面的CiA行动队硬碰硬，盯着布克尔特的尸体看了一会儿，罗昊突然眼睛一亮，有了想法：“我有办法了。外面那些人是为了布克尔特手中的P病毒来的，现在布克尔特死了，P病毒很自然就落在了我们手中……”
<br/>　　齐凡是第一个反应过来的，兴奋的说道：“战术讹诈！”
<br/>　　“什么战术讹诈，你们两个在说什么？能不能不要在这个时候打哑谜？”
<br/>　　有些急性子的孙涛忍受不了罗昊和齐凡之间的哑谜，这两个家伙从小一起长大，两个人之间的默契他们这些人比不了，自然是不能像齐凡那样立刻明白所谓的“办法”是什么！
<br/>　　“布克尔特手中所掌握的，西伦科特交给他的，经过‘暗月’改造后的P病毒。这种P病毒，除了我们这些接触过的，没有其他人知道它是什么样子，就是拿个塑料瓶里面灌上一些自来水，把它当做P病毒，外面那些人也不能说它不是。”齐凡看了眼周围的战术中队队员，继续解释道：“虽然西伦科特手中的改进型P病毒被蛰龙拿走，但是美国人一直都认为落在了我们手中，那我们拥有改进型P病毒的抗病毒疫苗是不是也顺理成章？”
<br/>　　“我明白你们两个人的意思了！”梅庆伟这个时候也是反应了过来，说道：“我们‘得到’了布克尔特手中的改进型P病毒，而且我们手中还掌握着改进型P病毒的抗病毒疫苗，我们就能讹诈外面那些家伙，让他们投鼠忌器！”
