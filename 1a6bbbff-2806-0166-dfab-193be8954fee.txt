
    王惊蛰和小草的押宝还真是押对了，这小娘们的心里可能最终是没抱着玉石俱焚的态度，既然是以交换为目的的话，总归还是有些底线的，他俩一步步的试探下来，还真是慢慢的把凉绣的底线给套了出来。<br/>　　<br/>　　    王惊蛰故作为难的说道“真要是让你先得了连山和归藏，事后你在反悔我可就亏大了，谁知道你是不是真的想要那一页生死薄啊，万一收了这两书后在给人灭口了咋办”<br/>　　<br/>　　    凉绣犹豫半天，神情颇为有些哀怨和惆怅的说道“你要这生死薄和八字借命是想为人续命，而我又何尝不是如此呢”<br/>　　<br/>　　    王惊蛰顿时了然，原来如此，这凉绣和韩观山等人凑在一起去墓崖村，为的居然也是想要给人续命，并且多续年已经明显不是她想要的了，她要的自然是十年八年甚至更多，所以这才苦巴巴的把茅小草给掠走了，然后一直研究着自己。<br/>　　<br/>　　    “好，那我们就约定如此，我要你们的墨家机关术用连山和归藏来换，第一笔交易完事了后，咱们在分头去往八拐里坡，我在为你谋取那一页生死薄”<br/>　　<br/>　　    凉绣皱眉说道“生死簿不在你身上”<br/>　　<br/>　　    “要是在我身上，我可能早就给用了，这生死薄是带不走的，但却可以在八拐里坡中用，你要是不信大可以在派个死士进去，提我的名走一趟也好使，到时你就看看这生死薄是不是能带的出来就行了”王惊蛰看了下天色，然后说道“你要是同意了呢，明日午时，咱们山顶再见，我为你带来那两书”<br/>　　<br/>　　    凉绣冷冷的看着他，说道“你该不会像糊弄余生婆那样，用两本假的连山和归藏来骗我吧王惊蛰，那到时候我说不得就是跟你拼个玉石俱焚了”<br/>　　<br/>　　    王惊蛰摆手说道“都是道上的人，我拿出来的东西一你看就知真假了，我还不至于在你面前做这种没水准的事，同样我也提醒你一句，墨家机关术我虽然不懂，但奇门八卦我还略知三四，你的机关术是真是假，我看个眼就能知道了，所以啊交易得诚心才是，咱们谁也瞒不住谁，少一点套路多一点真诚吧”<br/>　　<br/>　　    “行，那就明天午时再见”<br/>　　<br/>　　    王惊蛰瞥了眼神情落寞和哀伤的小草一眼，两人眼神一触即逝，随即他转身下了唐梁山。<br/>　　<br/>　　    王惊蛰走后，凉绣围着小草转了一圈，然后在她耳边嘲讽着说道“你这男人也不过如此么，书中自有颜如玉天下女人何其多呵呵，小草啊，这两句话说的是不是像刀子一样插在你的心口上了”<br/>　　<br/>　　    “有取有舍，当机立断，历来中枭雄似的人物才有这样的心境，她要是优柔寡断，我还真觉得他跟个娘们似的，成不了大气呢”<br/>　　<br/>　　    “哈哈”凉绣仰着脑袋鄙夷的笑了好几声，直摇头的说道“你嘴真硬，我猜你的心头可能现在都在滴血了”<br/>　　<br/>　　    茅小草余光瞥了她一眼，哀哀凄凄的叹了口气，甚是可怜<br/>　　<br/>　　    当天夜里，王惊蛰回到镇上后直接就蒙头大睡了一觉，直到第二天旅馆外有熙攘的人声传来，他才从床上爬起，然后找到一家商店买了纸笔，又回到了旅馆里，然后趴在桌上奋笔直书。<br/>　　<br/>　　    连山和归藏早就成为了孤本，根本没有传世，都是从每一代的赊刀人嘴中口口相传下来的，人在书在，王家人要是都死了的话，那这两书可就真的成为绝版了。<br/>　　<br/>　　    王惊蛰写的自然是真的连山和归藏，这里面他一个字都没有掺假，他根本就用不着用假的来糊弄凉绣，因为反正到最后她也带不走。<br/>　　<br/>　　    连山，归葬虽然是两书，但加起来却不过万字左右，又不是什么长篇巨论，一上午的时间就足够王惊蛰从头到尾都给默写一遍了。<br/>　　<br/>　　    临近午时，王惊蛰揣着一个笔记本出了旅馆，直奔唐梁山再去。<br/>　　<br/>　　    于此同时，山上的凉绣也拿出了墨家的机关术数，等待着王惊蛰前来交换。<br/>　　<br/>　　    一切看看似挺和平的，本来要撕逼的节奏，居然演变成了桌上谈判的路子<br/>　　<br/>　　    正午时分，王惊蛰如约来到山顶。<br/>　　<br/>　　    “唰”王惊蛰一上来就没二话，直接把手里的本子就给甩了过去，动作干脆利索。<br/>　　<br/>　　    凉绣先是愕然一愣，诧异的接到手中之后，就急促的翻看起来，看了寥寥几页，凉绣看的是一片心惊，这里面记载的东西以她的经验和资历，自然能看出并不是胡编乱造写出来的，哪怕其中若是有几句掺假或者少了一些字，凉绣都自问，她能有很大的把握给复原出来。<br/>　　<br/>　　    墨家三千年的历史，除了机关术数以外，其他多少也是懂一些的。<br/>　　<br/>　　    凉绣翻看完之后，挺惊异的说道“你还真的就这么交给我了”<br/>　　<br/>　　    王惊蛰淡淡的说道“别用你那小女子之心来衡量我，我说与你交换，就肯定会与你换，男人说话就是一个唾沫一个坑，我犯不上跟你扯皮”<br/>　　<br/>　　    凉绣晃了晃手里一本发旧的册子也甩手扔给了王惊蛰“墨家机关术数也在这里面，你看看吧”<br/>　　<br/>　　    王惊蛰接到手中，随意的看了几眼，然后点头说道“行，也算你守信了”<br/>　　<br/>　　    “那剩下的呢”<br/>　　<br/>　　    “明天启程，各自往八拐里坡走，你先到了就给我打个电话在哪里等着我，我过去找你就是了，然后赶在天黑的时候进入八拐里坡”<br/>　　<br/>　　    “你说的倒是干脆，到时可别反悔了”<br/>　　<br/>　　    “把心放在肚子里吧”
