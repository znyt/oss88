　　掌控五行之力的五大先天神邸形成的同时，五大道宫开辟完成，元始的《玄黄不灭》第二重也圆满了。
<br/>　　五行之力圆满的同时，在元始体内形成了一个巨大的五色天地烘炉，能够炼尽一切五行之物。同时，元始所创之五行印在这一刻也达到了大成。五行印化为五印，分别飞到五尊神邸手中，就好像是每一尊帝皇的帝玺。
<br/>　　与此同时，元始的《玄黄不灭》正式进入第三重四象化四极，无上玉身暴涨，玄黄不灭真身更进一步。
<br/>　　而且，因为之前四象之力地水火风的积累，《玄黄不灭》第三重也达到了小成。之后想要大成，则需汇聚象征四象的青龙、白虎、朱雀、玄武四大神兽之力。
<br/>　　不过，这一切距离元始还比较远，先天三大族还未开启争霸，四大神兽就在那里，镇压着这广大的洪荒天地四极，此事等有时间再去，也为时不晚。
<br/>　　同时，在此四层之中，元始一样遇到一些神材，他自然是不会嫌神材多，全部收了，以后炼制宝物，自己不用，赏赐给手下也好啊。
<br/>　　当然，因为他在罡火层中找到了火灵珠，元始路过其余层时，也仔细找了找，还这让他凑齐了金木水火土五行灵珠。
<br/>　　《玄黄不灭》第二重圆满与五灵珠的凑齐，固然值得高兴，但元始并没有忘记这一趟不周山之行的最终目的，他还没有攀上不周之巅，目的还没有达成，他还得继续前进。
<br/>　　跨过罡水层，元始遇到的是罡雷层。雷层之中，雷闪电鸣，狂暴的雷电随处可见，要是普通大罗金仙修为被这雷电击中，也将飞灰湮灭。
<br/>　　不过，元始并不是这普通的大罗金仙，先不说他五行圆满，《玄黄不灭》第二重圆满后真身强大无比，这些罡雷未必能够给他带来威胁，就说他那凶险的化形之劫，他都都过了，更不用说这些与化形之劫相仿的罡雷了。
<br/>　　元始在罡雷层中借助雷电之力淬炼刚刚突破的玄黄不灭真身，无上玉身随着五行的圆满，其身上已经开始散发着白青黑赤黄五彩之色，五行轮转，使得玉身变得更加玄妙高贵。
<br/>　　身处雷电之中，是领悟雷之一道最好的地方，正好元始渡劫之时有所领悟，只是当时天劫来得快，去得也快，领悟不多，现如今有现成的雷电在，元始哪有不趁机参悟的道理。
<br/>　　身为盘古元神所化，元始本身就有大量功德气运加身，福泽运道深厚，他就是天道之子，怎会有领悟不快的道理。
<br/>　　狂暴的罡雷轰在身上，毁灭之中带有新生，极致毁灭便是创造。雷电的刚烈、正阳、浩大、诛邪、破灭种种玄妙涌上心头，元始很快就沉浸在感悟法则的美妙之中，不可自拔！
<br/>　　……
<br/>　　五百年之后，元始起身往更高层走去，罡雷一层对他的帮助已经十分有限了，雷电打在身上，只能留下一道浅浅的雷痕，并不能让他受伤，雷电的存在已经没有了意义，除非是更加高级的雷电。
<br/>　　而且，对于雷电一道的领悟，也已经到了瓶颈，不是闭关苦修就能领悟的。
<br/>　　既然没有帮助，那留在此处又有何用？既然没用，又怎会留在此地？
<br/>　　故而，元始再次向上攀去。
<br/>　　随着元始的继续攀爬，威压也越来越重了，但，这些对于元始来说，都不算什么大事，经过时间的磨合适应，他终究会顶住。
<br/>　　这一次，元始再次进入下一层天。
<br/>　　冷！
<br/>　　非常冷！
<br/>　　冰冷到极致！
<br/>　　寒冷到骨子里！
<br/>　　天地一片死寂，四周无穷无尽的森白，没有任何声音，也没有任何光芒，完全是一片森白而死寂的空间。
<br/>　　元始一踏入这不周山上的第八层天，就感受到冰冷到极致的寒气，就连运功抵御中的元始，都被这寒冰之气完全冰冻住，无法动弹，甚至连灵魂都被短暂冻结。
<br/>　　入目之处，全是一片寒冰，白森森一片，仿佛一片冰冢，要葬了这苍天一般。
<br/>　　天空随处可见冰锥横飞，冰棱倒挂，时而形成一片寒冰风暴，寂灭了这天地。
<br/>　　至寒至冷，冰寒之极，冻结虚空。在此地，天地万物尽归寂灭，除了寒冰，不剩任何事物。
<br/>　　元始运转《神元九转》，元神恢复正常思维。但，之后，元始并没有驱除冻结全身的寒冰，而是在闭目思索，或者是在闭目参悟什么。
<br/>　　元始身处寒冰之中，忘记了天，忘记了地，忘了光明，忘了黑暗，忘了自己，忘了一切的一切，任由极致之寒冰冻结他的肉身，冻结他的元神，甚至冻结他的灵魂，只留下微弱的意识，在这寒冰之中挣扎。
<br/>　　识海深处，鸿蒙元始印散发出微弱的光芒，助他悟道！
<br/>　　随着时间的流逝，不知过了多久，元始的那一丝丝微弱的意识，也渐渐模糊了，渐渐失去了灵动思维。
<br/>　　渐渐地，失去了一切……
<br/>　　“嘭！”
<br/>　　随着元始最后一丝意识归于寂灭之时，元始的灵魂深处，猛地响起一声爆响！
<br/>　　一颗晶莹剔透，仿佛水晶一般的神魂晶体，显化在元始识海之中。
<br/>　　《神元九转》第四转融术圆满！随时可进第五转。不过，元始并不想如此快的进入第五转，因为第四转融术越多，第五转所生神元界也就越广。第四转为第五转之基础，基础越牢固，对自身越好！
<br/>　　而元神深处的神魂晶体，则为魂核，元始所融之术全部在这魂核之中，乃第五转之基础。
<br/>　　渐渐地，元始识海之中，意识再度复苏。而且，随着他意识的苏醒，元始被冻结的灵魂、元神、甚至肉身也渐渐开始了复苏。
<br/>　　“轰隆隆！”
<br/>　　伴随着元始开始的全面复苏，元始身体之中的寒冰之力轰然冲起，之后，狠狠地撞入元始识海之中的神魂晶体。与此同时，一股莫名的寂灭之意从元始身上散发而出。
<br/>　　这股寂灭之意所过之处，寒冰迅速消失，归于虚无，归于寂灭。
<br/>　　元始缓缓睁开双眼，眼中闪过一丝不带任何感情的冰冷之色，全身气质如冰，生人莫近。
<br/>　　随后，冰冷迅速消失，元始双眼之中再度恢复神采。
<br/>　　“呼——！”
<br/>　　元始轻轻呼出一口气，白色的长气如同一条长长的白色匹练，冲出数丈之远，之后归于虚无。
<br/>　　“这就是寂灭吗？”
<br/>　　元始看着自己一指点出，天地开始冻结，之后又再次归于黑暗，归于虚无，自言自语道。
<br/>　　“这，就是寂灭！”
<br/>　　元始眼中闪过莫名神采，之后，肯定道。
<br/>　　领悟寂灭法则，这一层的至寒至冷，至冰至冻，对于元始来说，已经没有威胁了，他不再耽搁，再次顺着不周山脉，向上而去。
<br/>　　至始至终，元始的目的都是这直入云霄苍穹的不周之巅！没有任何困难能够阻拦他，而这些艰险困苦只会成就更加强大的他。
<br/>　　“金、木、水、火、土、风、雷、冰，之后又会是什么呢？黑暗？光明？本座非常期待！”元始莫名一笑，轻声淡言道。
<br/>　　不再停留，元始漫步在这极致寒冰之中，向上而去！
