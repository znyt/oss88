夏妍妍一直开着车跟着陆寒霆，她也不知道自己想要干什么，她只知道她是不会放手的，陆寒霆是她的！
<br/>
<br/>        但是很快前面的劳斯莱斯幻影就变了车道，他变得太迅速，杀的她措手不及，她想要紧跟着变换车道，但是后面一道刺耳的汽车鸣笛声响起了，有人破口大骂，“喂，你是不是找死啊？”
<br/>
<br/>        夏妍妍出了一手的冷汗，她开车技术没有陆寒霆好，刚才想变道连方向灯都没有打，差一点就跟后面的车撞上了。
<br/>
<br/>        等夏妍妍再抬头看的时候，前面的劳斯莱斯幻影已经消失不见了。
<br/>
<br/>        夏妍妍气的路灯停了车，陆寒霆太警觉了，轻而易举的就将她给甩了。
<br/>
<br/>        她不甘心！
<br/>
<br/>        她怎么能甘心？
<br/>
<br/>        夏妍妍拿出了手机，给陆寒霆打电话…
<br/>
<br/>        ……
<br/>
<br/>        劳斯莱斯幻影豪车里，陆寒霆突然变了方向，夏夕绾因为惯性直接倒进了他的怀里。
<br/>
<br/>        陆寒霆抬手搂住了她莹润的香肩将她给稳住，嗓音低沉磁性的笑道，“陆太太，你是在投怀送抱？”
<br/>
<br/>        夏夕绾稳住了自己，迅速坐回了自己的座椅上，“陆先生，你怎么突然打方向盘，还开的这么快，怎么，后面有什么妖精在追你？”
<br/>
<br/>        陆寒霆挑起英气的剑眉，“后面的妖精没看到，我身边倒坐了一个。”
<br/>
<br/>        “…”夏夕绾哼了哼，这时她突然发现这条路不是回幽兰苑的路，“陆先生，你是不是走错了啊？”
<br/>
<br/>        “没有走错，打个电话给奶奶，就说今晚我们不回去了。”
<br/>
<br/>        夏夕绾纤长的羽捷一颤，黑漉漉的翦瞳落在他的身上，“陆先生，你…你该不会带我去…酒店开房吧？”
<br/>
<br/>        “你很期待？”
<br/>
<br/>        “…我没有！”
<br/>
<br/>        “不是去酒店开房，而是去我名下的别墅，有时候我会在那里住。”
<br/>
<br/>        像他这样的男人，名下的房产是很多的，虽然他大部分时间都住在幽兰苑里陪奶奶，但有时他也会去自己的别墅，今天晚上他要带她去别墅里住。
<br/>
<br/>        “可以不去么？”
<br/>
<br/>        “你再这副扭扭捏捏的样子好像今晚一定有事情要发生，那我晚上就对你不客气了。”
<br/>
<br/>        夏夕绾迅速拿出了手机，给奶奶打电话。
<br/>
<br/>        那端悠扬的铃声响了一遍就被接起了，奶奶慈祥和蔼的声音传递了过来，“喂，绾绾，生日快乐呀，我家绾绾又长大一岁了，今天玩的开心吗？”
<br/>
<br/>        夏夕绾点头，“恩，谢谢奶奶，我今天玩的很开心哦，就是…今天晚上太晚了，怕打扰奶奶的休息，所以…所以我和陆先生就…不回去了…”
<br/>
<br/>        陆寒霆看着女孩羞赧的模样，雪白的小耳垂红红的，眸子躲躲闪闪，很像是好学生第一次说了谎话。
<br/>
<br/>        这时他的手机传来了震动声，他抬头一看，是夏妍妍打来的。
<br/>
<br/>        陆寒霆的俊脸上并没有什么情绪波澜，夜晚璀璨的霓虹灯透过蹭亮的玻璃车窗折射在了他精致的五官上，成熟英俊的一塌糊涂。
<br/>
<br/>        薄唇抿了一下，溢出几分森寒，他伸手直接将手机关机了。
<br/>
<br/>        奶奶，“好的绾绾，我已经千万别惦记奶奶，跟寒霆好好的二人世界一下，顺便把我的重孙再提上日程哈。”
