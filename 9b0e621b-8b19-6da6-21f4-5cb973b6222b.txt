　　“这圣种的力量爆发而出，据说可以提升两重的境界，罗辰的实力本来就在楚风眠之上，如今在加上这圣种的力量，楚风眠是必败无疑了。”
<br/>　　“罗辰能在北邙学院的内门之中，混成地榜第一，也绝对不是弱者。”
<br/>　　“看看吧，这小子马上就要败了，这一招混乱神拳本就强大无比，再加上圣种的力量，击败这小子，轻而易举。”
<br/>　　在场的众人，看着如今罗辰出手，尤其是这一招混乱神拳，都是不由的皱起了眉头。
<br/>　　就让他们跟罗辰交手，在面对着混乱神拳的时候，想要抵挡下来，也并不容易。
<br/>　　“混乱神拳？真正混乱神体的混乱神拳的确是强大无比，但是就你这旁枝末节？给我破！”
<br/>　　楚风眠不屑的看了一眼那罗辰。
<br/>　　这罗辰一身的实力，全部都是来自与身体之中的圣种，这虽然是一种捷径，可以短时间的提升自己的实力。
<br/>　　但是实际上在楚风眠看来，却是一种愚蠢到极限的做法。
<br/>　　不是自己的实力，就算是在强大，也无法发挥到最好。
<br/>　　要是夜天君来，楚风眠当然只能是逃，但是就罗辰，楚风眠还不会将他放在眼中。
<br/>　　“战龙之舞！龙腾世纪！”
<br/>　　“吼！”
<br/>　　在楚风眠的身边，响彻着一声声的龙吟，在楚风眠的身边，无数灵力顿时化为了一条条真龙，在空中翱翔。
<br/>　　天地之间，尽是龙腾。
<br/>　　这混乱神拳的力量，在这无数真龙的龙腾之下，居然都是瑟瑟发抖，似乎是有些无法对抗，被轻易化解。
<br/>　　“什么？混乱神拳就这样被化解了？”
<br/>　　“此人真的是神力境四重的武者？”
<br/>　　“外门弟子，一个北邙学院的新人，居然能够做到这一步，怎么可能！”
<br/>　　白萱的眼神中，也都是震惊无比。
<br/>　　她自然猜想过，关于现在楚风眠的实力，到底是达到了哪一步，却是无论如何也想不到。
<br/>　　楚风眠的实力，居然是还要超越了这罗辰。
<br/>　　罗辰催动圣种，凝聚出的混乱神拳，已经是他的最强手段的，但是在如今楚风眠的面前，却是轻而易举的被化解了。
<br/>　　似乎是根本没有受到多少影响。
<br/>　　“想要来对付我，还是让夜天君自己来吧，一条夜天君的走狗，今日我就抽了你一半圣者精血，给你一个教训！”
<br/>　　楚风眠猛然凝聚灵力，猛然一掌，便是冲着那罗辰抓了过去。
<br/>　　“两位还请住手！”
<br/>　　两道身影，突然飞了出去，白萱跟麟木，分别是拦下了楚风眠跟罗辰二人。
<br/>　　“大家都是北邙学院的弟子，不要因为这小小的口舌之争，伤了和气，更何况还是颜古龙帝的洞府重要，我们现在可不要内斗。”
<br/>　　白萱一边说着，一边悄然传音给楚风眠，劝解道。
<br/>　　“颜古龙帝的洞府要紧，如今不是撕破脸的时候。”
<br/>　　“恩。”
<br/>　　听这白萱的话，楚风眠的脸色逐渐的平息了下来。
<br/>　　不过只要楚风眠心中起了杀心，这罗辰，他必须要杀，颜古龙帝的洞府要紧，等探索完了，楚风眠一样会杀了他。
<br/>　　“就给你几日时间的活头。”
<br/>　　楚风眠瞥了那罗辰一样，一双眼睛的杀意，令人不寒而栗。
<br/>　　现在在场众人，可没有任何人还敢来轻视楚风眠的了，罗辰都败在了楚风眠的手中。
<br/>　　他们几人跟楚风眠交手，虽然胜负大多都是五五之数，但是这样一尊敌人，可没有什么人愿意去主动招惹。
<br/>　　“时间不早了，诸位还是准备一下离开吧，颜古龙帝的洞府，可不要被其他人抢占先机了。”
<br/>　　那尊云公子笑了笑开口道。
<br/>　　“好。”
<br/>　　其他众人，都是点了点头，同意了云公子的话。
<br/>　　如今就算是有着天大的恩怨，也抵不过颜古龙帝的洞府，颜古龙帝可是五万年前的无敌人物。
<br/>　　他的洞府中，到底是会有着多少的机缘，只怕是许多人想都不敢想。
<br/>　　不过唯一可以确定的是，在其中，只要是得到一些收获，拿出来都是天大的机遇。
<br/>　　在这等机遇面前，只能是停手，楚风眠也是。
<br/>　　比起杀了罗辰，当然是颜古龙帝洞府的事更加重要，等到结束了这一切之后，楚风眠随意找个机会，就可以杀了罗辰。
<br/>　　“走！”
<br/>　　六道身影，同时化为遁光，一下子便是从这茶楼之中离开，飞遁到了虚空海中。
<br/>　　“我们这次要去的区域，是虚空海中三大禁区之一的乱葬海，其中虽然没有极为危险的空间风暴，但是也极为危险，存在这许多危险的虚空兽，各位还请小心。”
<br/>　　一边飞着，一边司空轩出声提醒道。
<br/>　　众人都是点了点头，虚空海的危险，他们还是都清楚的。
<br/>　　没有人敢大意。
<br/>　　楚风眠一边跟随人群飞行，一边也是从白萱那边，了解到了一些关于虚空海的种种。
<br/>　　其实第一便是虚空海中的危险。
<br/>　　虚空海是一处极为奇异的空间，似乎是无尽虚空的尽头。

<br/>　　这便是虚空海中，最大的危险。
<br/>　　只是好在楚风眠一行人即将前往的乱葬海比较稳定，并不会出现太大的空间风暴。
<br/>　　但是身为虚空海三大禁区之一，乱葬海中，也有着一种极为特殊的危险。
<br/>　　那便是虚空兽。
<br/>　　这虚空兽，许多都是从虚空深处中诞生而来的，乃是极为精粹的灵力凝聚而成的生物。
<br/>　　每一尊虚空兽的力量，都丝毫不比寻常圣者要差，强大一些的虚空兽，足以是毁灭生死境高阶的强者。
