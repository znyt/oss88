　　这种东西在民国时期，还算是非常新潮的玩意，翠云看着很是新奇，想碰一下又不敢，叶少阳看她这个样子，把钟表拿起来，放在她手上。
<br/>　　翠云连忙往外推，但看到附近没人，还是欢喜地把玩了一会。叶少阳道：“你喜欢吗？”
<br/>　　翠云不好意思地笑了笑，说道：“喜欢是喜欢的，不过我一个乡下妇女，对这么新式的玩意可是不敢想。”
<br/>　　叶少阳道：“你喜欢，赶明儿我送你一个。”
<br/>　　翠云冲他咧嘴一笑，“你有这份心，姐我就满足了。”
<br/>　　过了一会，镇长带着一个珠光宝气的胖女人进来，两人知道就是镇长夫人了，急忙站起来打招呼。
<br/>　　镇长夫人上下打量了叶少阳一会，明显有点不信任，不过既然镇长都把人请到家里来了，也就让试试了。
<br/>　　夫人走后，镇长亲自领着叶少阳和翠云离开偏厅，顺着走廊一路穿过了宅院，来到宅子里角，这里有个小花园，对面就是围墙。靠围墙下面有一排房屋，镇长指着最靠墙的一间上锁的房门，表示他的姨太太就被关在这里。
<br/>　　没等叶少阳开口，屋子里突然传来了一阵唱戏的声音，是个尖细嗓子的女声。镇长一听，脸色顿时发白，后退了几步，对叶少阳说道：“这根本就不是我那姨太太的声音，我姨太太是小户人家出身，也不会唱戏……”
<br/>　　翠云听他这么一说，再听唱戏的声音，虽然是大白天，也有些头皮发麻，又不自主地往后缩了缩，有些担忧地朝叶少阳看过去。
<br/>　　叶少阳并不害怕，凑上去看，见门上有个洞，方方正正，有人头差不多大，镇长告诉叶少阳，自从姨太太关在这里之后，每天就通过这个门洞往里面送吃的。
<br/>　　叶少阳挑开门洞的挡板，往里面看去，房间没窗户，光线很暗，能依稀对面靠墙摆着一张床，床上坐着个女子，头发很长，身穿素裙，在用梳子一下一下地梳头。嘴里咿咿呀呀地唱着听不懂的戏词。
<br/>　　这画面，很有点恐怖片里的感觉。
<br/>　　叶少阳盯着姨太太看了一会，姨太太慢慢把脸抬起来，与叶少阳对视，叶少阳这才看到了一张恐怖的脸：脸上五颜六色，看半天才知道像是涂的颜色。
<br/>　　叶少阳盯着她看了一会，退了出去，用力喘了几口气，问镇长：“屋里怎么这么臭？”
<br/>　　镇长一听，有些为难地说道：“那个，因为不敢不她放出去，她便溺也都在屋里，有便桶。”
<br/>　　翠云一听，也忍不住说道：“多少天没清理了，那得多臭啊。”
<br/>　　“三五天清理一次，她神智还是有的，知道吃，也知道如厕，不过见人就挠，各种辱骂，我只能三五天才派几个人进去按住她，清理一次房间。”
<br/>　　叶少阳一听，说道：“镇长大人对她还挺在乎的。”
<br/>　　镇长听了这话，有点不好意思，叹了口气说道：“实不相瞒，这姨太太是我一个远房表妹，从小青梅竹马，后来我出去闯了十几年，总算混到一官半职，心里想着回来娶她，但当时我已娶妻，只要纳做姨太太，也就是一两年的事。我本打算哄了夫人开心，纳她做妾，怎料生出这种怪事，我心中也是很对不住这个表妹啊。”
<br/>　　叶少阳听他这么说，心中也有了几分判断，开始盘算起来，在门前坐下来，打开背包，拿出朱砂笔，在舌头上抹了唾沫，然后蘸了点朱砂，在左手心画了个掌心雷。
<br/>　　镇长在旁边看着，注意到他手上五颜六色的魂印，说道：“这是什么？”
<br/>　　“我的帮手，每一条代表一个人，呼之则来。”
<br/>　　镇长听不懂，随口问道：“那你怎么不找他们来帮忙？”
<br/>　　“来不了啊。”叶少阳看着掌心的数道魂印，叹了口气，喃喃道：“他们要是在，哪怕随便来一个，我还用画什么掌心雷。”
<br/>　　叶少阳摇了摇头，继续将掌心雷画好，也不着急起来，现在日头还好，他摊开左手，让掌心雷吸收日精，一边对镇长说道：“是谁告诉你，她是狐狸精附身？”
<br/>　　镇长一怔，道：“前面几个先生都这么说，难道不是？”
<br/>　　叶少阳道：“放弃，一屋子的鬼气，明显是女鬼附身，跟狐狸精毛线关系。”
<br/>　　现在的他，虽然只有以往的一成左右的法力，不过神识并没有受到损伤，还是一样强大，以他灵仙牌位的神识，刚才把脑袋贴在门洞上，立刻就能感知到房间里弥漫的鬼气，其中但这阴沉之气，说明是女鬼，而且阴气虽然浓郁，但并不强烈，叶少阳以自己丰富的经验判断，这个女鬼最多是普通的幽魂，离怨灵都还差一截。
<br/>　　镇长怕他是故作惊人之语，说道：“不会吧，五个先生，难道都看错了？”
<br/>　　叶少阳道：“这很简单，第一个草包自己解决不了，甚至都不知道是鬼是妖，是好忽悠你说是狐狸精，消息传出去，后面几个骗子对付不了，自然也跟着说是狐狸精，实话说吧，镇长大人，你被骗了，这也不奇怪，你人傻钱多，没骗子找上门那才是怪了。”
<br/>　　镇长听他说自己人傻钱多，脸上有点挂不住。翠云用胳膊怼了叶少阳一下，瞪了他一眼，让他不要乱说话。
<br/>　　“只希望这第六个，不是骗子。”镇长盯着叶少阳，不冷不热地说道。
<br/>　　叶少阳嘿嘿一笑，“我办事你放心，只要出的起钱，什么都搞得定。”
<br/>　　镇长道：“这你放心，只要能救了我家阿兰，钱不是问题。”
<br/>　　叶少阳本想先跟他谈条件，后来一想，还是先把鬼抓住，让他眼见为实，于是让他打开门。
<br/>　　镇长有些犹豫，迟疑地说道：“她力大无穷，我们几个肯定不是对手，要是让她跑了就麻烦了。”
<br/>　　叶少阳道：“我既然让你开门，肯定没事，你听我的就是了。”
<br/>　　镇长只好叫了几个家丁过来，把锁打开，然后在们两边站好，以防万一。
<br/>　　叶少阳一脚踹开了门。
