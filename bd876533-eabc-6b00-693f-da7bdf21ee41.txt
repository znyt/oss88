　　“萧逸副统领，你做什么？”夜修脸色一冷。
<br/>　　但他的脸色再冷，也比不过周遭弥漫的森然寒气。
<br/>　　那种仿佛要冻入骨髓的冰冷，让在场所有人不自觉地打了个冷颤。
<br/>　　“将事情交待清楚，否则别怪我不客气。”萧逸冷冷地说着。
<br/>　　同时，大手一挥，霸道剑势，轰然降临。
<br/>　　剑势，以及寒冰剑罡所施展的滔天寒气，形成了一股强悍的禁锢之力。
<br/>　　也就是说，从现在起。
<br/>　　若无他萧逸的允许，任何人都休想离开。
<br/>　　当然，夜修除外。
<br/>　　他这个天元六重的剑道强者，可不是单凭这两种手段能禁锢的。
<br/>　　这时，仅仅剑势，就已经被压制得动弹不得的东竹、东月、东来三位郡王，脸色极其难看。
<br/>　　他们三人不过地元九重，自然连剑势都抵挡不住。
<br/>　　“萧逸副统领，你可知自己在做什么？”东月郡王冷声质问道。
<br/>　　“又可知，囚禁一位郡王，后果何其严重，你可承担得起？”
<br/>　　“就是。”东来郡王亦冷声道。
<br/>　　“更别说这里足足五位郡王。”
<br/>　　“我们随便一人，若修书一封到王都炎武卫总部问罪。”
<br/>　　“你这区区一个炎武卫副统领，第二天就得被撤职，关入大牢。”
<br/>　　“去，尽管去。”萧逸脸色冰冷。
<br/>　　下一秒，身影一动，一把揪住了东月郡王的衣领。
<br/>　　东月郡王一惊，连声道，“萧…萧逸副统领，你想做什么？”
<br/>　　“你说呢？”萧逸冷笑一声。
<br/>　　“东月郡王，你可还记得那天你率东月卫拦我时，我说过什么？”
<br/>　　“最好别让我发现你与那些失踪之事有关，否则我饶不了你。”
<br/>　　“现在你与我谈后果？”
<br/>　　“这十数年来，你的不作为，纵容千雄教在你东月郡掳人，炼制血意丹。”
<br/>　　“因此而惨死的武者，何止数十万。”
<br/>　　“就凭这一点，我杀你一百遍都是轻的。”
<br/>　　萧逸的话语，一句比一句冰冷。
<br/>　　说到最后，更是大手一挥，直接将东月郡王震得跌坐在地。
<br/>　　“还有你。”萧逸看向东来郡王。
<br/>　　“修书一封到王都炎武卫总部问罪？”
<br/>　　萧逸指了指远处高山内的血意丹祭坛。
<br/>　　“你看看那里头大量被折磨的武者，数以万计瑟瑟发抖的平民。”
<br/>　　“那些都是你东来郡治理下的子民。”
<br/>　　“若要问罪，他们更应该问你这个郡王的包庇之罪。”
<br/>　　说着，萧逸大手一挥，隔空一掌将东来郡王扇得吐血击飞。
<br/>　　一旁东竹郡王，顿时大怒。
<br/>　　“萧逸副统领，你不要太过分了。”
<br/>　　“我们乃一郡之王，如何行事，还轮不到你一个炎武卫副统领来过问。”
<br/>　　“怎么？现在不仅囚禁我们，还想杀我们不成？”
<br/>　　东竹郡王，直接将一个‘意图击杀郡王’的罪名，扣在萧逸头上。
<br/>　　三十六郡王，先不说修为实力如何，单论其地位，是非常高的。
<br/>　　击杀郡王这个罪名，极其严重。
<br/>　　东竹郡王的想法很简单，直接凭此震慑萧逸。
<br/>　　让萧逸忌惮下，撤去剑势和禁锢。
<br/>　　但，他的震慑，在萧逸眼中，只是个笑话。
<br/>　　“杀你们？”萧逸冷笑一声，“你以为我不敢？”
<br/>　　说着，萧逸手中寒霜剑一挥，一道剑气重重劈过。
<br/>　　直接将东竹郡王劈成重伤。
<br/>　　“你…你敢？”东竹郡王满脸不可置信之色，以及恐慌。
<br/>　　“我有何不敢？”萧逸冷声一笑。
<br/>　　“我乃裂天剑宗最强剑主，全炎武王国的剑道象征之一。”
<br/>　　“除非公然造反。”
<br/>　　“否则，就是杀了你们，也无人能奈何我。”
<br/>　　“放肆。”万剑一冷喝一声，“萧逸，你也知道你是裂天剑宗弟子？”
<br/>　　“开口闭口就是杀人，宗门何时教过你这些？”
<br/>　　“先不说在宗门内我是你的长辈，单单在炎武卫，我是你的上级。”
<br/>　　“你便无权禁锢我。”
<br/>　　“还不快快撤去禁制，放我等离去？”
<br/>　　萧逸冷笑一声，“你也知道你是宗门长辈？”
<br/>　　“你可做好你应有的表率？”
<br/>　　“宗门又何时教过你黑白不分，包庇千雄教这等恶毒教派了？”
<br/>　　“至于上级，说实话，你这个炎武卫正统领，还不放在我眼中。”
<br/>　　说着，萧逸眼中杀意暴增。
<br/>　　“好了，都少给我废话。”
<br/>　　“要么给我交待清楚，要么，休怪我不客气。”
<br/>　　话音落下，空气中弥漫的寒气，更为浓郁。
<br/>　　实力稍弱的五位郡王，甚至渐渐被寒霜所覆盖。
<br/>　　不过片刻，连万剑一与另外一位统领，也有呈冻结成冰坨之势。
<br/>　　毫无疑问，若他们再不老实交代，等待他们的，只有被冻结成冰。
<br/>　　时间再长，必死无疑。
<br/>　　恰在此时，一道无奈的沉稳声音响起。
<br/>　　“好了。”说话的，是夜修。
<br/>　　“萧逸副统领，停下吧。”
<br/>　　“嗯？夜修统领肯如实交待了吗？”萧逸冷笑一声。
<br/>　　“我还以为会是实力最弱的三位郡王承受不住，率先开口呢。”
<br/>　　夜修点点头，道，“我会说，但不是现在。”
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　“你想何时说？”
<br/>　　夜修沉声道，“回王都。”
<br/>　　“千雄教主，以及相关人等，炼制血意丹，证据确凿。”
<br/>　　“我会亲自押送他们回炎武卫总部受审。”
<br/>　　“届时，我必定给你一个满意的交代。”
<br/>　　“我如何能信你？”萧逸皱眉道。
<br/>　　“我保证。”夜修认真地说道，“而且，我没必要骗你。”
<br/>　　“好，我姑且再信你一次。”萧逸冷冷说着，大手一挥，收回了剑势和滔天寒气。
<br/>　　万剑一等人，顿时松了口气。
<br/>　　对于夜修，萧逸还是信得过的。
<br/>　　再者，就算今日他是骗自己，自己日后也大可重新逼问。
<br/>　　夜修没理由做如此白费功夫之事。
<br/>　　另外，萧逸这次的主要目的，是捉住千雄教主审问。
<br/>　　而炎武卫和各郡王包庇之事，只是顺带查出来的。
<br/>　　个中缘由，暂时不急于弄清楚，回王都再得到答案也行。
<br/>　　而且，萧逸还有其它事情要处理。
<br/>　　恰好，处理完后，他便打算回王都了。
<br/>　　时间刚刚好。
<br/>　　……
<br/>　　猛虎郡，炎武卫据点大牢。
<br/>　　此刻，千雄教主已成阶下囚。
<br/>　　在他面前，一冷酷年轻人，正冷冷地盯着他。
<br/>　　年轻人，自是萧逸。
<br/>　　“在夜修统领押你回王都受审前，还是我先审问一番吧。”萧逸冷笑一声。
<br/>　　千雄教主脸色强硬，闭嘴不言。
<br/>　　“哼，我知道你嘴硬。”萧逸冷笑一声。
<br/>　　“你给不少人都服下了烙火丹吧。”
<br/>　　“那种滋味，估计你还未尝过。”
<br/>　　“切。”千雄教主嗤笑一声，“小子，你不用唬我。”
<br/>　　“烙火丹，是火焰圣教特有的丹药，你不可能会炼制，更不可能拥有。”
<br/>　　“再者，就算你有，烙火丹也不会立刻发作。”
<br/>　　“等它发作时，我已经在王都了，还轮不到你这个毛头小子来审我。”
<br/>　　“是吗？”萧逸的笑容，忽然变得狰狞。
<br/>　　“我不仅会炼制烙火丹，还会炼制噬元丹。”
<br/>　　“什么…不可能。”千雄教主脸色大变，“噬元丹，那是圣子那个层次的炼药师才有本事炼制的丹药。”
