　　不仅是苏莫疑惑，广场上的所有人都心有疑惑，巫人王居然不进塔，而是直接飞到了塔顶上！
<br/>　　难道不通过塔内的三层空间，就这样就算过关了？这也太扯了吧！
<br/>　　这时，只见巫人王向大祭司等人抱拳，沉声道：“大祭司、四祭司，各位核心长老，我想我没有必要进塔了吧？”
<br/>　　众核心长老闻言，均是看向两位祭司，有两位祭司在，他们可不敢做决定。
<br/>　　“这一关，你无需去闯了！算你过关！”大祭司微微颌首道。
<br/>　　巫人王点头，面上露出了笑容，他目光环视整个广场，面上带着霸绝环宇的自信，此次少司选拔，无人能阻挡他的脚步！
<br/>　　族中无数的天才，不过是见证他辉煌的观众而已。
<br/>　　众人见此，均是羡慕不已，不用闯关就算过关了，这真是人比人气死人啊！
<br/>　　这样就有些坑了啊！
<br/>　　不过，众人心中也没有什么不平，因为这第一关对于巫人王来说，是轻而易举之事，闯与不闯都没有什么区别。
<br/>　　五年前的上一次少司选拔，巫人王就有不弱于巫凌的实力，两人大战了十几个时辰，最终巫凌略胜半招，最后成为了少司。
<br/>　　所以，这第一关对于巫人王来说，根本就是摆设而已，不闯也没有什么！
<br/>　　苏莫目光淡淡，此人或许无比的强大，无比的自信，但他却是同样有必胜的信心。
<br/>　　嗖！嗖！嗖！！
<br/>　　少倾之后，大量的巫族天才不再迟疑，纷纷身形一动，向三层宝塔中冲去。
<br/>　　无数的巫族年轻天才，如同一道长河，流向三层宝塔之中。
<br/>　　宝塔虽然不是太大，却是自成空间，足以容纳亿万武者。
<br/>　　嗖！嗖！嗖！！
<br/>　　大量天才鱼贯而入，片刻之后，便有不下百万人进入了三层宝塔之中。
<br/>　　巫天御、巫玛、巫幽兰等超级天才，亦是纷纷进入了塔中。
<br/>　　不过，进入三层宝塔的人，只有广场上的人十分之一不到，因为很多巫族年轻人都放弃了进去闯关的打算。
<br/>　　闯过三层空间，可是要与同阶的少司巫凌一战，虽然不是巫凌的全部实力，但七成实力也是无数人可望而不可及的地步了。
<br/>　　所以，之前很多打算参加选拔的普通天才，直接就放弃了进塔闯关的打算。
<br/>　　他们也是有自知之明，知道自己根本不可能成为少司，所以也没有什么失望。
<br/>　　嗖！
<br/>　　苏莫见已经没有人再进塔中，他便抬起脚步，最后一个掠进了塔门之中。
<br/>　　嗡嗡~~
<br/>　　进入塔中之后，苏莫眼前场景变换，出现在了一片略显虚幻的空间之中。
<br/>　　转头看向四方，并没有发现其他人，看来所有人都被自行隔离了。
<br/>　　而在他的前方空间之中，正静立着一名白衣青年，静静的望着他。
<br/>　　白衣青年面目较为普通，身躯好像融入了周围的空间之中，给人一种缥缈虚幻之感。
<br/>　　此人身上的气息并没有多强，修为和苏莫一样，同样是武帝境三重境界。
<br/>　　苏莫知道，这就是上一届少司选拔的胜出者，巫凌的投影。
<br/>　　不过，这第一层的巫凌投影，只有对方三成的实力，他丝毫不在意。
<br/>　　就在这时，巫凌的投影缓缓的向苏莫走了过来，每一步都让得虚空震动，一股强大的气势从其身上升腾而起。
<br/>　　随即二话不说，巫凌的投影猛然一指点向苏莫。
<br/>　　刹那之间，一道浩大而又犀利的指芒，仿若是一根巨大的光柱，向苏莫暴刺而来。
<br/>　　这根光柱仿若是一个巨大的钻头，还未到达苏莫身前，便开始急速旋转了起来，绞碎一切，发出震天的轰鸣声。
<br/>　　轰隆隆！
<br/>　　指芒速度快到极致，瞬息见便绞碎了空间，来到了苏莫的身前。
<br/>　　“碎！”
<br/>　　剑光一闪，苏莫厉喝一声，一剑狠狠的斩向指芒。
<br/>　　犀利的剑光，裹挟着冲天的魔气，瞬间便斩中了浩大的指芒。
<br/>　　轰！
<br/>　　一声爆响，指芒立刻崩碎，化为漫天气流涌动四方。
<br/>　　“死吧！”苏莫动作不停，再此出剑，瞬间连斩三剑，三道浩大无比的黑色剑气成品字型向巫凌的投影杀了过去。
<br/>　　三道剑气威势惊天，魔威浩荡，杀机滚滚。
<br/>　　面对苏莫的强力攻击，巫凌的投影即便立刻出招抵挡，但也丝毫无法挡住。
<br/>　　三道剑气袭杀而过，巫凌的投影立刻被斩成了碎片，而后消散在了空气之中。
<br/>　　“的确不错！”苏莫低声自喃，这巫凌的投影，和他同境界的情况下，三成的实力能有这种战力，已经算是非常的强了。
<br/>　　按照这种战力计算，第三层之时，对方七成的实力，估计他若是不用战魂的情况下，胜负还真的不好说。
<br/>　　少倾之后，这片略显虚幻的空间渐渐清晰了起来，前方出现了一道楼梯，通向上一层。
<br/>　　苏莫没有丝毫犹豫，抬脚便走了过去，向第二层走去。
<br/>　　外界，圣巫广场之上。
<br/>　　在苏莫闯关之时，外界已经一片纷乱，因为大量的巫族天才，纷纷口喷鲜血，从宝塔的大门处倒飞了出来，摔的七零八落。
<br/>　　嗖！嗖！嗖！！
<br/>　　简直是络绎不绝，仿佛是争先恐后一般，不停的有人从大门中倒飞而回。
<br/>　　有的人甚至进入塔中还不到半个呼吸，便被淘汰而出，身受重伤。
<br/>　　即便是和巫凌的投影同阶一战，即便第一层的投影只有三成的实力，依旧有大多数的人被淘汰了出来。
<br/>　　“幸亏我们没有进去！”
<br/>　　“是啊！不然第一层都闯不过去，真是够丢人的！”
<br/>　　“估计能闯过三层，最终登上塔顶的人，不会超过五百个！”
<br/>　　很多没有进塔的人议论纷纷，不禁心有唏嘘之意，暗赞自己有先见之明。
<br/>　　不少人望向塔顶，只见巫人王盘膝坐的塔顶，双眸紧闭，对于众人闯塔的情况漠不关心。
<br/>　　其端坐在塔顶，虽然没有任何气势爆发，但却有登临绝顶，独领风骚的意境。
<br/>　　不用战斗，便已经过关，这种待遇还有谁配拥有？
<br/>　　众人心中羡慕，这一次少司选拔，估计是巫人王获得最终的胜利了，此事八九不离十。
<br/>　　或者，巫玛和巫幽兰也有一些机会，但机会不是很大，两人很难撼动巫人王。
<br/>　　至于其他超级天才，机会则是非常渺茫，甚至可以忽略不计了。
