        而他的母亲，是部队的军医。
<br/>
<br/>        因为常年在外为国家漂泊，他的母亲跟随在傅夜沉父亲的身侧，对傅夜沉的父亲产生了不该有的爱慕之情。
<br/>
<br/>        两人的结合，也是一次意外，才有了他。
<br/>
<br/>        当初，傅夜沉的父亲并不知道是他傅昼景的妈妈，还误以为是自己的原配妻子陪他过了一夜。
<br/>
<br/>        后来，傅昼景的母亲偷偷地生下傅昼景后，又回到了傅夜沉的父亲身边并肩作战。
<br/>
<br/>        “我妈妈殉情后，是大哥的生母将我从孤儿院里接回傅家的，爷爷拿我的血和爸爸的血做了亲子鉴定。确定我是傅家的骨肉后，才肯让我进这傅家的门，并把傅家祖传的另一块凤凰玉坠给了我。”傅昼景黯然神伤地说道，顿了顿后，他声音有些哽咽了，“可终究，我只是个私生子，并不受爷爷的待见。”
<br/>
<br/>        听完傅昼景的讲述，童筱颖沉默了。
<br/>
<br/>        终究是她婆婆的不对，但婆婆为爱殉情，也着实令人感动。
<br/>
<br/>        想必，大哥的生母，也是看在这个情分上，才主动去把昼景从孤儿院里接回来的吧！
<br/>
<br/>        不然，哪个女人会愿意让一个私生子进自家的门？
<br/>
<br/>        傅昼景见童筱颖不吭声，叹了口气，接着说道：“对不起，筱颖。因为我的不受待见，也连累到了你。爷爷因为宠着大哥，所以难免会爱屋及乌地偏爱大嫂。要是爷爷对你很苛刻，请你不要在意。”
<br/>
<br/>        “不会的！我觉得，爷爷人挺好。如果，爷爷真的偏爱苏漫雪。那么爷爷就没必要把我也接回傅家来了。”童筱颖安慰道。
<br/>
<br/>        傅昼景只笑不语，筱颖到底是个单纯的女孩子，很多事情，她都看不透。而他，其实早就看透了。
<br/>
<br/>        他的生母真的是为了他的父亲殉情而死，这么简单吗？
<br/>
<br/>        这种话，他也就说给筱颖听听，也只有单纯的筱颖才会相信他的这种说辞。
<br/>
<br/>        童筱颖在傅昼景的带领下，将傅家大庄园参观了一遍。行走在这大庄园里，看着这古建筑群，童筱颖感觉自己就像穿越回到了古代豪宅了一般新鲜感十足。
<br/>
<br/>        但是，苏漫雪就没她这么好运气了，被傅夜沉禁足在房间里不许出来。
<br/>
<br/>        直到晚饭，傅老太爷的召唤，一家子才又聚在了一起。
<br/>
<br/>        而这一刻，傅夜沉再看到童筱颖的时候，心里百般不是滋味，甚至有些难受。
<br/>
<br/>        “漫雪、筱颖，你们俩既然是我傅家的孙媳妇，那么无论学识和品格都要配得上我的两个孙子。我看过你们两的资料了，都是大专学历，所以，从明日开始，我给你们两请了家庭教师，开始补习考试科目，参加十月份的转升本考试。你两的对口专业学校，我也给安排好了。就是咱们临海城的临海大学附属学院。这个学院虽然是一所私立学院，但是国内数一数二的二本学府。而且，教学资源是共用了临海大学的教学资源。”傅老太爷傅炎廷语重心长地说。
<br/>
<br/>        苏漫雪顿时难以置信地反问：“爷爷真打算要我们专升本？！”
<br/>
<br/>        傅老太爷笃定地回答：“当然，对于你两这大专生来说，已经是很不错的选择了。将来，你两本科毕业，还可以继续深造考一本大学的研究生。至于，你们两对举行婚礼的大事。等你们两这孙媳妇读完本科，毕业了后再一起浓重举办。”
<br/>
<br/>        他倒是为两个孙媳妇做好了长远的打算。
<br/>
<br/>        这两个女孩子的家世背景是配不上他们傅家，但是，学历上总要往上抬一抬吧！
<br/>
<br/>        如果不是两个孙儿分别对这两个女孩情有独钟，他才不会让这两个门不当户不对的女孩进他们傅家的大门。
<br/>
<br/>        “爷爷，我一定会不负您所望的！我不仅要专升本，而且，将来还要考一本大学的研究生，考完研究生再考博士。对了，我还想考公务员了！这样，才能给爷爷的脸面争光！”苏漫雪嘴巴特甜，立即微笑着哄着傅老太爷开心。
<br/>
<br/>        “明天，你们俩该干嘛去就干嘛去。两个孙媳妇从今以后，就留在这庄园里了，直到她们两考起临海大学的附属学院为止！”傅老太爷又放话了。
<br/>
<br/>        傅昼景看了一眼苏漫雪，见她对爷爷无事献殷勤，阿谀奉承的样子，就无语了。
<br/>
<br/>        而傅夜沉虽然坐在傅老太爷身边，但他的心思，全在童筱颖的身上。
<br/>
<br/>        傅夜沉看着童筱颖的一颦一笑，黯然神伤，显得非常地无可奈何。
<br/>
<br/>        童筱颖坐在一旁，沉默了许久，突然插上话，真诚地看向傅老太爷，问道：“爷爷，我可以参加今年六月份的普通高等学校招生全国统一考试吗？”
<br/>
<br/>        从去年的高考失利，一直到现在，童筱颖其实从未放弃过学习，而且一心只想复读重考，只可惜，诺母不愿意出这份钱。
<br/>
<br/>        傅老太爷非常吃惊地看着童筱颖。
<br/>
<br/>        这丫头居然想重新参加高考？！
<br/>
<br/>        要做出这样的决定，那得有很大的勇气和决心才行啊！
<br/>
<br/>        傅夜沉回过神来后，见状，立即给童筱颖帮腔道：“爷爷，筱颖有心想要重新参加高考，就托关系给她选个复读学校拿个学籍去参加高考便是。”
<br/>
<br/>        “好！好！想重新考一个好的大学，这是好事！现在才四月，你还有两个月的时间准备复习。”最宠爱的长孙都开口帮腔了，傅老太爷自然也支持。
<br/>
<br/>        傅夜沉想了想，又接着提议道：“既然爷爷觉得筱颖重新参加高考是好事，那么爷爷就给筱颖挑选六科优质的高考辅导教师，来家里给筱颖辅导功课吧！这样，可以让筱颖考上更好的大学！”
<br/>
<br/>        苏漫雪见童筱颖要重新复读考大学，自然也不甘落后，立即跟傅老太爷撒娇道：“爷爷，我也要重新复读参加高考！我要给爷爷争光！”
<br/>
<br/>        “好好好！你也重新复读，和筱颖一起参加高考！”傅老太爷顿时笑不拢嘴，他这长孙媳也挺有志气的。
<br/>
<br/>        傅夜沉却反驳道：“爷爷，漫雪的事情，就不用您操心了。筱颖重新参加高考倒是不错。漫雪就不用了。”
<br/>
<br/>        “你这是什么话？”傅老太爷不满地皱了皱眉头。
<br/>
<br/>        傅夜沉看都不看苏漫雪一眼，冷淡地回答道：“漫雪对学习不感兴趣。”
<br/>
<br/>        “谁说我对学习不感兴趣啦！就算我以前不感兴趣，我现在可感兴趣了！为了爷爷，为了婆婆，更是为了你，我愿意去学习！”苏漫雪顿时信誓旦旦地说。
<br/>
<br/>        傅老太爷倒是非常欣赏苏漫雪这话。
<br/>
<br/>        一个女孩子能为自己所爱的人而上进，就是一个非常不错的女孩子了。
<br/>
<br/>        所以，傅老太爷便帮着苏漫雪说话：“你听听，漫雪自己都亲口说，为了你，愿意改变自己。”
<br/>
<br/>        他们几个人说话，傅昼景一句话也插不上来，其实，是他在这个家里压根就没有说话做主的权利。
<br/>
<br/>        好在筱颖想要重新参加高考这事，大哥替他帮了腔。
<br/>
<br/>        所以，在这个家里，到底还是大哥真心对他和筱颖好。
<br/>
<br/>        但对于傅老太爷说苏漫雪的好话，傅夜沉只是冷哼。
<br/>
<br/>        若不是因为筱颖已经成为了弟弟昼景的妻子，他如果把实情告知了爷爷，爷爷十有八九会为了他，而逼昼景跟筱颖离婚。
<br/>
<br/>        傅夜沉能从童筱颖的眼中，看出她对傅昼景有爱意。
<br/>
<br/>        所以，他不想毁了她的幸福。
<br/>
<br/>        事如今，弄得他傅夜沉里外不是，对苏漫雪无可奈何，对自己心爱的女人童筱颖更加无可奈何。
<br/>
<br/>        爱而不得，他的心，是多么的痛。
<br/>
<br/>        算了，这段日子，就先这么过着，反正，爷爷也不打算急着给他们办婚礼。
